## Проблемы с аккаунтом и входом

| Сообщение об ошибке | Причина | Решение |
|---------------|-------|----------|
| "Invalid email or password. Please check your credentials and try again." | Неверный email или пароль | Внимательно проверьте email и пароль. Помните, что пароль чувствителен к регистру. |
| "We couldn't find an account with this email." | Email не зарегистрирован | Проверьте адрес электронной почты или создайте новый аккаунт. |
| "Your account has been suspended. Please contact support for assistance." | Аккаунт приостановлен | Свяжитесь с нашей службой поддержки по адресу [hi@s.ee](mailto:hi@s.ee). |
| "This login link has expired or is invalid. Please request a new one." | Ссылка для входа истекла | Запросите новую ссылку на странице входа. |
| "This verification code has expired or is invalid. Please request a new one." | Код подтверждения истек или недействителен | Запросите новый код подтверждения. |
| "This email is already registered." | Email уже используется | Используйте другой адрес или попробуйте войти с этим email. |
| "The current password you entered is incorrect." | При смене введен неверный текущий пароль | Проверьте текущий пароль и попробуйте снова. |

**Примечание:** для входа работает только ваш основной адрес электронной почты. Другие подтвержденные адреса в аккаунте не подходят для аутентификации. Письма для сброса пароля также отправляются только на основной email.

## Проблемы с именем пользователя

| Сообщение об ошибке | Причина | Решение |

## Проблемы с короткими ссылками

| Сообщение об ошибке | Причина | Решение |
|---------------|-------|----------|
| "This slug is already taken. Please choose a different one." | Пользовательский короткий код уже занят | Попробуйте другой пользовательский короткий код для короткой ссылки. |
| "This slug cannot be used. Please choose a different one." | Короткий код зарезервирован системой | Этот короткий код зарезервирован. Выберите другой. |
| "This short link does not exist or is no longer available." | Короткая ссылка не существует или удалена | Проверьте правильность URL. Возможно, ссылка была удалена. |
| "The domain linked to this short URL is unavailable or no longer exists." | Домен недоступен | Обратитесь в поддержку или попробуйте использовать другой домен. |
| "Please enter a valid destination URL starting with http:// or https://" | Неверный формат URL | Убедитесь, что URL назначения начинается с `http://` или `https://`. |
| "This destination URL is not allowed. Please use a different link." | URL назначения заблокирован | Этот URL находится в нашем черном списке. Используйте другой адрес назначения. |
| "Incorrect password. Please try again." | Неверный пароль для защищенной ссылки | Введите правильный пароль для доступа к ссылке. |
| "URL has already been shortened." | URL уже сокращен | Для этого URL уже существует короткая ссылка. |

## Проблемы с подпиской и тарифом

| Сообщение об ошибке | Причина | Решение |
|---------------|-------|----------|
| "You already have an active subscription plan." | У вас уже есть активный тариф | Если хотите сменить тариф, сначала отмените текущую подписку. |
| "You don't have an active subscription to cancel." | Нет активной подписки | Сейчас у вас нет активной подписки для отмены. |
| "This feature requires a premium subscription. Please upgrade your plan to access it." | Возможность требует более высокого тарифа | Повысьте тариф, чтобы получить доступ к этой функции. |
| "Unable to find your subscription plan. Please contact support if this issue persists." | Тариф не найден | Свяжитесь с нашей службой поддержки по адресу [hi@s.ee](mailto:hi@s.ee). |
| "The selected pricing option is no longer available." | Выбранный вариант цены больше недоступен | Обновите страницу и выберите актуальный вариант тарифа. |

## Проблемы с загрузкой файлов

| Сообщение об ошибке | Причина | Решение |
|---------------|-------|----------|
| "File size exceeds limit" | Файл слишком большой | Сожмите файл или перейдите на более высокий тариф с поддержкой больших загрузок. |
| "Storage limit exceeded" | Превышена квота хранения | Удалите часть файлов, чтобы освободить место, или повысьте тариф для увеличения хранилища. |
| "Unsupported file type" / "File type not allowed" | Неподдерживаемый формат файла | Используйте поддерживаемый формат файла. |
| "Maximum concurrent uploads exceeded" | Слишком много одновременных загрузок | Дождитесь завершения текущих загрузок перед запуском новых. |
| "This file format is not supported. Allowed formats: JPG, JPEG, PNG, GIF, WEBP, AVIF." | Неподдерживаемый формат аватара | Загрузите аватар в одном из поддерживаемых форматов: JPG, JPEG, PNG, GIF, WEBP или AVIF. |

## Проблемы с командой

| Сообщение об ошибке | Причина | Решение |
|---------------|-------|----------|
| "This feature is not available on your current plan." | Функция команды не включена в тариф | Перейдите на тариф с поддержкой командных функций. |
| "You have already created a team." | Команда уже создана | Каждый пользователь может создать только одну команду. Управлять существующей командой можно из панели управления. |
| "You don't have permission to manage this team." | Недостаточно прав | Свяжитесь с владельцем вашей команды. |
| "The invitation has already been processed." | Приглашение уже обработано | Это приглашение уже было принято или отклонено. |
| "Please wait at least 24 hours before resending." | Слишком ранняя повторная отправка | Подождите 24 часа перед повторной отправкой приглашения этому пользователю. |

## Проблемы при удалении аккаунта

| Сообщение об ошибке | Причина | Решение |
|---------------|-------|----------|
| "Your team has other members, please delete the team first." | В команде еще есть участники | Удалите всех участников команды или удалите саму команду перед удалением аккаунта. |
| "Too many deletion requests, please try again later." | Слишком много запросов | Подождите некоторое время и попробуйте снова. |
| "Invalid or expired verification code." | Код подтверждения недействителен или истек | Запросите новый код подтверждения. |

## Проблемы с управлением email

| Сообщение об ошибке | Причина | Решение |
|---------------|-------|----------|
| "Email already exists." | Email уже используется | Используйте другой адрес электронной почты. |
| "Main email cannot be deleted." | Нельзя удалить основной email | Назначьте другой подтвержденный email основным, а затем удалите этот. |
| "You need to close two factor verification first." | Сначала нужно отключить 2FA | Отключите двухфакторную аутентификацию перед изменением основного email. |
| "This email has not been verified." | Email не подтвержден | Сначала подтвердите этот адрес электронной почты. |
