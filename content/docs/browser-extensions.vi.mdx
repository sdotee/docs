Tiện ích mở rộng trình duyệt S.EE mang tính năng rút gọn URL, chia sẻ văn bản và tải tệp lên trực tiếp ngay trong trình duyệt. Hỗ trợ Chrome, Firefox và Edge.

Mã nguồn mở trên [GitHub](https://github.com/sdotee/browser-extensions).

## Tính năng

| Tính năng | Mô tả |
|----------|-------|
| Rút gọn URL | Rút gọn bất kỳ URL nào chỉ với một cú nhấp, hỗ trợ slug tùy chỉnh và nhiều lựa chọn tên miền |
| Chia sẻ văn bản | Chia sẻ đoạn mã và văn bản với hỗ trợ tô sáng cú pháp |
| Chia sẻ tệp | Tải nhiều tệp với theo dõi tiến độ |
| Định dạng xuất | Sao chép kết quả ở dạng URL thuần, Markdown, HTML hoặc BBCode |
| Tạo mã QR | Tạo mã QR cho bất kỳ URL rút gọn nào |
| Xuất mã QR | Tải mã QR xuống ở dạng PNG, SVG hoặc PDF (512x512) |
| Menu chuột phải | Rút gọn trang/liên kết, hiển thị QR, chia sẻ văn bản đã chọn, tải ảnh lên |
| Quản lý lịch sử | Quản lý lịch sử theo tab cho URL, văn bản và tệp, hỗ trợ xóa hàng loạt |
| Lưu bản nháp | Tự động lưu nội dung trong ô nhập giữa các phiên |
| Tự động sao chép | Tự động sao chép kết quả vào bảng nhớ tạm |
| Chế độ sáng/tối | Tự động nhận diện giao diện sáng/tối với tùy chọn chuyển thủ công |
| Thông báo trên máy tính | Nhận thông báo khi thao tác hoàn tất qua menu ngữ cảnh |
| Đa trình duyệt | Hoạt động trên Chrome (MV3), Firefox (MV3) và Edge |
