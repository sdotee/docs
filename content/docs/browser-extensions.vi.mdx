Tiện ích mở rộng trình duyệt S.EE mang tính năng rút gọn URL, chia sẻ văn bản và tải tệp lên trực tiếp ngay trong trình duyệt. Hỗ trợ Chrome, Firefox và Edge.

Mã nguồn mở trên [GitHub](https://github.com/sdotee/browser-extensions).

## Tính năng

| Feature | Description |
|---------|-------------|
| URL Shortening | Rút gọn bất kỳ URL nào chỉ với một cú nhấp, hỗ trợ slug tùy chỉnh và nhiều lựa chọn tên miền |
| Text Sharing | Chia sẻ đoạn mã và văn bản với hỗ trợ tô sáng cú pháp |
| File Sharing | Tải nhiều tệp với theo dõi tiến độ |
| Export Formats | Sao chép kết quả ở dạng Plain URL, Markdown, HTML hoặc BBCode |
| QR Code Generation | Tạo mã QR cho bất kỳ URL rút gọn nào |
| QR Code Export | Tải mã QR xuống ở dạng PNG, SVG hoặc PDF (512x512) |
| Right-Click Menu | Rút gọn trang/liên kết, hiển thị QR, chia sẻ văn bản đã chọn, tải ảnh lên |
| History Management | Quản lý lịch sử theo tab cho URL, văn bản và tệp, hỗ trợ xóa hàng loạt |
| Draft Persistence | Tự động lưu nội dung trong ô nhập giữa các phiên |
| Auto-Copy | Tự động sao chép kết quả vào clipboard |
| Dark/Light Mode | Tự động nhận diện giao diện sáng/tối với tùy chọn chuyển thủ công |
| Desktop Notifications | Nhận thông báo khi thao tác hoàn tất qua menu ngữ cảnh |
| Cross-Browser | Hoạt động trên Chrome (MV3), Firefox (MV3) và Edge |
