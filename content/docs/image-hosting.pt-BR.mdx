A [página de envio de imagens](https://s.ee/user/images/) permite envios em lote. Arraste e solte arquivos ou pastas, cole da área de transferência ou clique para escolher arquivos. As imagens coladas são convertidas automaticamente para WebP (ou PNG, se o seu navegador não oferecer suporte a WebP).

Os arquivos enviados são ordenados por nome, para que os lotes mantenham a ordem correta.

---

## Códigos de compartilhamento

Cada imagem recebe um link de compartilhamento. Você pode copiar os links individualmente ou obter um código de compartilhamento para o lote inteiro.

Formatos: URL direta, Markdown, HTML, BBCode.

---

## Gerenciar arquivos

A página de imagens serve apenas para envio. Para excluir imagens ou gerenciar arquivos, acesse [s.ee/user/files/](https://s.ee/user/files/).
