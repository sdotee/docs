## Créer un code QR

1. Accédez à [https://s.ee/user/qr/](https://s.ee/user/qr/)
2. Cliquez sur **New QR Code**
3. Saisissez l’URL de destination
4. Personnalisez l’apparence (facultatif)
5. Cliquez sur **Créer**

---

## Options de personnalisation

| Option | Description |
|--------|-------------|
| Couleurs | Définissez les couleurs d’avant-plan et d’arrière-plan |
| Logo | Ajoutez votre logo au centre du code QR |
| Format de téléchargement | Exportez en PNG, SVG ou PDF |

<Callout type="info">
**Astuce :** Pour les codes QR créés sur cette page, vous pouvez accéder rapidement à l’image du code QR en ajoutant `/qr.png`, `/qr.svg` ou `/qr.pdf` à l’URL courte. Par exemple, si l’URL courte de votre code QR est `https://s.ee/sZvL`, vous pouvez accéder à :
- `https://s.ee/sZvL/qr.png` - image PNG
- `https://s.ee/sZvL/qr.svg` - vecteur SVG
- `https://s.ee/sZvL/qr.pdf` - document PDF

**Remarque :** Cette fonctionnalité ne fonctionne que pour les codes QR dynamiques créés dans la section [QR Codes](https://s.ee/user/qr/), et non pour les liens courts classiques.
</Callout>

---

## Statistiques

Les codes QR incluent les mêmes statistiques que les liens :

- Nombre de scans
- Données de localisation
- Informations sur l’appareil
- Sources de provenance

---

## Bonnes pratiques

- Utilisez des couleurs à fort contraste pour faciliter le scan
- Gardez les logos petits afin de préserver la lisibilité
- Testez votre code QR avant impression
