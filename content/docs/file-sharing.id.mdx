## Ikhtisar

S.EE File Sharing memungkinkan Anda mengunggah dan membagikan file dengan tautan pendek yang mudah dibagikan. Cocok untuk berbagi gambar, dokumen, audio, video, dan file lainnya.

---

## Unggah File

### Unggah Cepat (Drag & Drop)

1. Login ke [https://s.ee/user/dashboard](https://s.ee/user/dashboard)
2. Klik **Sharing > Files** di sidebar kiri atau langsung buka [https://s.ee/user/files/](https://s.ee/user/files/)
3. Tarik dan lepas file atau folder ke jendela **Drag & Drop**
   - Anda juga dapat mengklik jendela untuk mencari dan memilih file
   - Mengunggah folder akan mengunggah semua file di dalamnya
4. File akan dibagikan dengan domain default dan slug acak

### Unggah Lanjutan (Create File)

Untuk kontrol yang lebih besar atas berbagi file Anda:

1. Klik tombol **Create File**
2. Atur pengaturan Anda:
   - **Domain** - Pilih domain yang Anda inginkan untuk halaman berbagi
   - **Custom Slug** - Atur slug URL kustom
   - **Other options** - Tag, kedaluwarsa, dan lainnya
3. Pilih file Anda lalu unggah

---

## Tautan File

Setiap file yang diunggah memiliki dua jenis tautan:

| Link Type | Format | Description |
|-----------|--------|-------------|
| Share Page | `fs.to/[your_slug]` | Domain dan slug dapat dikustomisasi |
| Direct Link | `i.see.you/[yyyy]/[mm]/[dd]/[random_slug]/[filename].[ext]` | Akses langsung ke file dengan nama file aslinya |

**Catatan:** Saat ini direct link tidak mendukung domain kustom.

<Callout type="info">

</Callout>

## Retensi File

- File disimpan **secara permanen** selama langganan Anda aktif
- Jika langganan Anda berakhir, file akan disimpan selama **7 hari** sebelum dihapus

## FAQ

### Siapa yang dapat mengunggah file?

Hanya pelanggan berbayar yang dapat mengunggah file. Langganan aktif diperlukan.

### Apakah ada batas unduhan?

Tidak. Tidak ada batas jumlah unduhan maupun bandwidth untuk file yang dibagikan.

### Apakah saya bisa mengkustomisasi URL halaman berbagi?

Ya, Anda dapat mengatur domain dan slug kustom untuk halaman berbagi saat menggunakan metode unggah **Create File**.

### Apakah saya bisa mengkustomisasi URL direct link?

Saat ini direct link menggunakan format tetap dan tidak mendukung domain kustom.
