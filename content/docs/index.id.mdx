## Apa itu S.EE?

S.EE adalah platform manajemen tautan untuk memendekkan URL, membuat kode QR, dan membagikan konten, dengan analitik bawaan serta dukungan domain kustom. Platform ini mencakup:

- **[S.EE Links](https://s.ee/user/links/)** - Buat tautan pendek dengan domain kustom, perlindungan kata sandi, dan penargetan cerdas
- **[S.EE QR Codes](https://s.ee/user/qr/)** - Buat kode QR yang dapat dikustomisasi dengan analitik
- **[S.EE Text Sharing](https://s.ee/user/text/)** - Bagikan teks biasa, cuplikan kode, dan Markdown dengan tautan pendek
- **[S.EE File Sharing](https://s.ee/user/files/)** - Unggah dan bagikan file dengan tautan pendek
- **[S.EE Image Hosting](https://s.ee/user/images/)** - Unggah dan host gambar dengan tautan pendek

## Fitur Utama

### Manajemen Tautan

Buat tautan pendek dengan:

- Slug dan domain kustom
- Perlindungan kata sandi
- Tanggal kedaluwarsa
- Penargetan berdasarkan lokasi, perangkat, atau bahasa
- Parameter UTM
- Penerusan parameter

### Kode QR

Buat kode QR dengan:

- Warna dan logo kustom
- Unduh dalam format PNG, SVG, atau PDF
- Analitik bawaan

### Berbagi Teks

Bagikan konten teks dengan:

- Dukungan teks biasa, cuplikan kode, dan Markdown
- Sorotan sintaks untuk kode
- Perlindungan kata sandi dan kedaluwarsa
- Endpoint teks mentah untuk unduhan yang mudah

### Berbagi File

Unggah dan bagikan file dengan:

- Drag-and-drop, paste, atau unggah folder
- Domain dan slug kustom
- Perlindungan kata sandi untuk file privat, serta kedaluwarsa
- Tautan unduhan langsung

### Hosting Gambar

Host gambar dengan:

- Drag-and-drop, paste, atau unggah folder
- Optimasi gambar otomatis saat paste dari clipboard
- Tautan gambar langsung untuk embed

### Analitik

Lacak performa dengan:

- Jumlah klik
- Data geografis
- Informasi perangkat dan browser
- Sumber perujuk
