[ShareX](https://getsharex.com/), Windows için açık kaynaklı bir ekran yakalama ve dosya paylaşım aracıdır. S.EE, ShareX için özel yükleyici yapılandırmaları sunar; böylece ekran görüntülerini, dosyaları, metinleri ve kodları doğrudan S.EE'ye yükleyip paylaşılabilir bağlantıyı geri alabilirsiniz.

Ayrıca bkz.: [ShareX özel yükleyici dokümantasyonu](https://getsharex.com/docs/custom-yükleyici)

---

## Özel yükleyiciler

S.EE şu yükleme türleri için yapılandırma sunar:

| Uploader | İşlevi |
|----------|-------------|
| URL kısaltıcı | URL'leri S.EE üzerinden kısaltır |
| Metin yükleyici | Metin parçaları yükler |
| Kod yükleyici | Sözdizimi vurgulamalı kod yükler |
| Markdown yükleyici | Render edilmiş önizlemeyle Markdown yükler |
| Görsel ve dosya yükleyici | Görselleri ve dosyaları yükler |

### İçe aktarma

1. **ShareX** uygulamasını açın
2. **Hedefler** > **Özel yükleyici ayarları** yoluna gidin
3. **İçe aktar** düğmesine tıklayın ve ardından:
   - İndirdiğiniz `.sxcu` dosyasını seçin veya
   - İndirme URL'sini doğrudan yapıştırın

## API anahtarını ayarlama

1. API anahtarınızı **[Araçlar > API Token](https://s.ee/user/developers/)** sayfasından alın
2. ShareX içinde **Özel yükleyici ayarları** bölümünü açın ve içe aktardığınız S.EE yükleyici'ını seçin
3. **Başlıklar** bölümünde `Authorization` değerini API anahtarınız olarak ayarlayın

<Callout type="warn">
API anahtarınızı herkese açık şekilde paylaşmayın ve sürüm kontrolüne eklemeyin.
</Callout>

## Kullanım

### Uploader atama

ShareX içinde **Hedefler** bölümüne gidin ve S.EE yükleyici'larını atayın:

- URL kısaltıcı -> S.EE URL Shortener
- Metin yükleyici -> S.EE Text Uploader, Code Uploader veya Markdown Uploader
- Görsel yükleyici -> S.EE File Uploader
- Dosya yükleyici -> S.EE File Uploader

### Yükleme

ShareX kısayollarını veya sağ tık menüsünü kullanarak URL kısaltabilir, ekran görüntüsü yükleyebilir, metin paylaşabilir ve daha fazlasını yapabilirsiniz. ShareX içeriği S.EE'ye yükler ve bağlantıyı panonuza kopyalar.

## Sorun giderme

| Sorun | Çözüm |
|-------|-----|
| 401 hatası | `Authorization` başlığında API anahtarının doğru ayarlandığını kontrol edin |
| 403 hatası | API anahtarınızın gerekli izinlere sahip olduğundan emin olun |
| Dosya reddedildi | Dosya türü desteklenmiyor olabilir; izin verilen türler için [Dosya paylaşımı](/docs/tr/file-sharing) sayfasına bakın |

Her şeyin çalıştığını doğrulamak için **Özel yükleyici ayarları** içinde **Doğrula** düğmesine tıklayabilirsiniz.
