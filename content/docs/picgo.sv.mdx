[PicGo](https://picgo.app/) är ett verktyg med öppen källkod för bild- och filuppladdning till Windows, macOS och Linux. Nya versioner har stöd för S.EE direkt från början, så du behöver bara en API-token för att komma igång.

- [Ladda ner PicGo](https://github.com/Molunerfinn/PicGo/releases)
- [Källkod](https://github.com/Molunerfinn/PicGo)

---

## Konfiguration

### 1. Hämta din API-token

1. Logga in på ditt S.EE-konto
2. Gå till **[Tools > API Token](https://s.ee/user/developers/)**
3. Generera en ny API-token

<Callout type="warn">
Dela inte din API-token offentligt och checka inte in den i versionshantering.
</Callout>

### 2. Konfigurera PicGo

1. Ladda ner och installera [PicGo](https://github.com/Molunerfinn/PicGo/releases)
2. Öppna PicGo
3. Gå till **Picbeds Settings** > **SM.MS (S.EE)**
4. Klicka på **+** för att lägga till en ny konfiguration
5. Ange ett namn i **Configuration Name**, till exempel `S.EE`
6. Klistra in din API-token i **Set Token**
7. Klicka på **Confirm**

![PicGo S.EE-konfiguration](https://i.see.you/2026/02/10/k6Nr/pasted-image-1770748236942.webp)

## Användning

Det finns flera sätt att ladda upp:

- Dra och släpp filer i uppladdningsytan
- Klicka på uppladdningsytan för att välja filer från datorn
- Klicka på det flytande minifönstret för att ladda upp direkt
- Högerklicka på det flytande minifönstret och välj **Upload by Clipboard**

PicGo kopierar den resulterande länken till urklipp. Du kan välja format som Markdown, HTML och rå URL.
