## Создать ссылку

1. Перейдите на [https://s.ee/user/links/](https://s.ee/user/links/)
2. Нажмите **Create Link**
3. Введите URL назначения
4. Настройте параметры при необходимости
5. Нажмите **Create**

---

## Параметры ссылки

При создании ссылки можно настроить:

| Параметр | Описание |
|----------|----------|
| Пользовательский slug | Задайте собственную конечную часть ссылки, например `s.ee/link` |
| Домен | Выберите один из официальных доменов или свой домен |
| Пароль | Запрос пароля перед переходом по ссылке |
| Срок действия | Укажите дату, когда ссылка перестанет работать |
| Таргетинг | Перенаправляйте пользователей по стране, устройству или языку |
| Метки | Организуйте ссылки с помощью меток |
| UTM-параметры | Добавляйте параметры отслеживания кампаний |
| Передача параметров | Передавайте URL-параметры на целевую страницу |

---

## Аналитика

Нажмите на любую ссылку, чтобы открыть ее аналитику:

- Общее число кликов
- Клики по географии
- Клики по устройствам и браузерам
- Источники переходов

---

## Управление ссылками

В панели ссылок можно:

- **Редактировать** - обновлять URL назначения и настройки
- **Удалять** - безвозвратно удалять ссылку
- **Копировать** - копировать короткую ссылку в буфер обмена
