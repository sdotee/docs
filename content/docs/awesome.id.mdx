## Alat CLI

| Name | Description | Link |
|------|-------------|------|
| **see-cli** | S.EE dari command line. Pendekkan URL, bagikan teks, unggah file. | [sdotee/cli](https://github.com/sdotee/cli) |

---

## Plugin

| Name | Description | Link |
|------|-------------|------|
| **S.EE for WordPress** | Pemendekan URL, berbagi teks, dan hosting file untuk WordPress. Otomatis membuat URL pendek saat publish, mengunggah media ke S.EE, dan melacak semuanya dari dasbor. | [WordPress.org](https://wordpress.org/plugins/sdotee/) · [GitHub](https://github.com/sdotee/see-wordpress) · [Docs](/docs/id/wordpress) |
| **S.EE for Raycast** | Pemendekan URL, berbagi teks, dan unggah file langsung dari Raycast. Mendeteksi isi clipboard secara otomatis, mendukung slug kustom, kata sandi, tanggal kedaluwarsa, dan riwayat yang bisa ditelusuri. | [Raycast Store](https://www.raycast.com/missuo/sdotee) · [GitHub](https://github.com/missuo/raycast-sdotee) |
| **Typecho-Plugin-PicUp** | Plugin Typecho untuk unggah gambar dan lampiran dengan banyak backend. Juga mendukung unggahan gambar ke S.EE, sehingga memudahkan pengelolaan media dari situs Typecho Anda. | [GitHub](https://github.com/lhl77/Typecho-Plugin-PicUp) · [Homepage](https://blog.lhl.one/artical/1026.html) |
