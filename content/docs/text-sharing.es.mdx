## Descripción general

S.EE Text Sharing le permite compartir texto sin formato, fragmentos de código y documentos Markdown con un enlace corto y fácil de compartir. Es ideal para compartir archivos de configuración, ejemplos de código, notas y documentación.

---

## Crear un texto compartido

1. Inicie sesión en [https://s.ee/user/dashboard](https://s.ee/user/dashboard)
2. Haga clic en **Sharing > Text** en la barra lateral izquierda o vaya directamente a [https://s.ee/user/text/](https://s.ee/user/text/)
3. Complete los datos:
   - **Title** (opcional) - Se usa como nombre de archivo al descargar
   - **Short Link** - Seleccione el dominio que prefiera
   - **Text Type** - Elija entre Plain Text, Code o Markdown
4. Introduzca su contenido en el cuadro **Editar**
5. Configure los ajustes opcionales (vea más abajo)
6. Haga clic en **Crear texto**

---

## Tipos de texto

| Tipo | Descripción |
|------|-------------|
| Texto sin formato | Texto sencillo sin formato |
| Código | Fragmentos de código con resaltado de sintaxis |
| Markdown | Markdown renderizado con formato |

---

## Ajustes opcionales

| Opción | Descripción |
|--------|-------------|
| Contraseña | Proteja su texto con una contraseña |
| Etiquetas | Organice sus textos compartidos con etiquetas |
| Vencimiento | Defina cuándo vence el texto compartido |
| Desactivar la indexación en motores de búsqueda | Evite que los motores de búsqueda indexen su texto compartido |

---

## Ver texto compartido

Una vez creado, recibirá un enlace corto como `ba.sh/demo`.

### Modos de visualización

| URL | Descripción |
|-----|-------------|
| `ba.sh/demo` | Ver la página de texto con formato y resaltado de sintaxis |
| `ba.sh/demo/raw` | Ver directamente el texto sin formato |

El endpoint `/raw` es útil para:

- Descargar scripts directamente con `curl` o `wget`
- Incrustar contenido en otras aplicaciones
- Ver contenido sin formato
