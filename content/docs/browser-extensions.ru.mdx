Расширение браузера S.EE переносит сокращение URL, обмен текстом и загрузку файлов прямо в ваш браузер. Доступно для Chrome, Firefox и Edge.

Открытый исходный код на [GitHub](https://github.com/sdotee/browser-extensions).

## Возможности

| Возможность | Описание |
|---------|-------------|
| Сокращение URL | Сокращайте любой URL в один клик, используйте пользовательские короткие коды и разные домены |
| Обмен текстом | Делитесь текстом и фрагментами кода с подсветкой синтаксиса |
| Обмен файлами | Загружайте файлы пакетно с отслеживанием прогресса |
| Форматы экспорта | Копируйте результат как обычную URL, Markdown, HTML или BBCode |
| Генерация QR-кода | Создавайте QR-коды для любой сокращенной ссылки |
| Экспорт QR-кода | Скачивайте QR-коды в PNG, SVG или PDF (512x512) |
| Контекстное меню | Сокращайте страницу / ссылку, показывайте QR, делитесь выделенным текстом, загружайте изображения |
| История | История по вкладкам для URL, текста и файлов с пакетным удалением |
| Сохранение черновиков | Поля ввода автоматически сохраняются между сессиями |
| Автокопирование | Результаты автоматически копируются в буфер обмена |
| Светлая / темная тема | Автоматическое определение темы с ручным переключением |
| Уведомления на рабочем столе | Получайте уведомления после завершения действий из контекстного меню |
| Поддержка разных браузеров | Работает в Chrome (MV3), Firefox (MV3) и Edge |
