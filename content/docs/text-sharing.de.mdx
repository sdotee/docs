## Überblick

Mit S.EE Text Sharing können Sie Klartext, Code-Snippets und Markdown-Dokumente über einen kurzen, leicht teilbaren Link bereitstellen. Ideal zum Teilen von Konfigurationsdateien, Code-Beispielen, Notizen und Dokumentation.

---

## Eine Textfreigabe erstellen

1. Melden Sie sich bei [https://s.ee/user/dashboard](https://s.ee/user/dashboard) an
2. Klicken Sie in der linken Seitenleiste auf **Sharing > Text** oder gehen Sie direkt zu [https://s.ee/user/text/](https://s.ee/user/text/)
3. Füllen Sie die Details aus:
   - **Title** (optional) - Wird beim Download als Dateiname verwendet
   - **Short Link** - Wählen Sie Ihre bevorzugte Domain
   - **Text Type** - Wählen Sie zwischen Plain Text, Code oder Markdown
4. Geben Sie Ihren Inhalt in das Feld **Edit** ein
5. Konfigurieren Sie optionale Einstellungen (siehe unten)
6. Klicken Sie auf **Create Text**

---

## Texttypen

| Type | Description |
|------|-------------|
| Plain Text | Einfacher unformatierter Text |
| Code | Code-Snippets mit Syntaxhervorhebung |
| Markdown | Gerendertes Markdown mit Formatierung |

---

## Optionale Einstellungen

| Option | Description |
|--------|-------------|
| Password | Schützen Sie Ihren Text mit einem Passwort |
| Tags | Organisieren Sie Ihre geteilten Texte mit Tags |
| Expiration | Legen Sie fest, wann die Textfreigabe abläuft |
| Disable search engine indexing | Verhindern Sie, dass Suchmaschinen Ihren geteilten Text indexieren |

---

## Geteilten Text anzeigen

Nach der Erstellung erhalten Sie einen Kurzlink wie `ba.sh/demo`.

### Ansichtsmodi

| URL | Description |
|-----|-------------|
| `ba.sh/demo` | Zeigt die formatierte Textseite mit Syntaxhervorhebung an |
| `ba.sh/demo/raw` | Zeigt direkt den rohen Klartext an |

Der Endpunkt `/raw` ist nützlich für:

- Direktes Herunterladen von Skripten mit `curl` oder `wget`
- Einbetten in andere Anwendungen
- Anzeigen ohne Formatierung
