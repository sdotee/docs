## 帳號和登入問題

| 錯誤訊息 | 原因 | 解決方法 |
|---------------|-------|----------|
| "Invalid email or password. Please check your credentials and try again." | 電子郵件或密碼錯誤 | 請再次確認您的電子郵件和密碼。請注意，密碼區分大小寫。 |
| "We couldn't find an account with this email." | 電子郵件未註冊 | 請確認電子郵件地址是否正確，或重新建立新帳號。 |
| "Your account has been suspended. Please contact support for assistance." | 帳號已被暫停 | 請寄信至 [hi@s.ee](mailto:hi@s.ee) 聯絡支援團隊。 |
| "This login link has expired or is invalid. Please request a new one." | 登入連結已過期 | 請在登入頁面重新申請新的登入連結。 |
| "This verification code has expired or is invalid. Please request a new one." | 驗證碼已過期或無效 | 請重新申請新的驗證碼。 |
| "This email is already registered." | 電子郵件已被使用 | 請使用其他電子郵件地址，或直接用該電子郵件登入。 |
| "The current password you entered is incorrect." | 修改密碼時當前密碼錯誤 | 請確認當前密碼後重試。 |

**注意：** 只有您的主電子郵件可用於登入。帳號中其他已驗證電子郵件不能用於身份驗證。密碼重置郵件也只會傳送到主電子郵件。

## 使用者名稱問題

| 錯誤訊息 | 原因 | 解決方法 |
|---------------|-------|----------|
| "This username is already taken. Please choose a different one." | 使用者名稱已被使用 | 請選擇其他使用者名稱。 |
| "Username can only be changed once." | 已達到使用者名稱修改次數限制 | 很遺憾，每個帳號的使用者名稱只能修改 1 次，且該更改不可撤銷。 |

## 短網址問題

| 錯誤訊息 | 原因 | 解決方法 |
|---------------|-------|----------|
| "This slug is already taken. Please choose a different one." | 自訂短碼已被使用 | 請為短網址嘗試其他自訂短碼。 |
| "This slug cannot be used. Please choose a different one." | 短碼被系統保留 | 該短碼已被保留，請選擇其他短碼。 |
| "This short link does not exist or is no longer available." | 短網址不存在或已刪除 | 請確認連結 URL 是否正確。該短網址可能已被刪除。 |
| "The domain linked to this short URL is unavailable or no longer exists." | 網域不可用 | 請聯絡支援，或嘗試使用其他網域。 |
| "Please enter a valid destination URL starting with http:// or https://" | URL 格式無效 | 請確保目標 URL 以 `http://` 或 `https://` 開頭。 |
| "This destination URL is not allowed. Please use a different link." | 目標 URL 被攔截 | 該 URL 位於我們的封禁列表中，請使用其他目標 URL。 |
| "Incorrect password. Please try again." | 造訪受密碼保護的連結時密碼錯誤 | 請輸入正確密碼以造訪該連結。 |
| "URL has already been shortened." | URL 已被縮短 | 此 URL 已關聯一個現有短網址。 |

## 訂閱和套餐問題

| 錯誤訊息 | 原因 | 解決方法 |
|---------------|-------|----------|
| "You already have an active subscription plan." | 已有有效訂閱 | 如果您想更換套餐，請先取消當前訂閱。 |
| "You don't have an active subscription to cancel." | 沒有有效訂閱 | 您當前沒有可取消的有效訂閱。 |
| "This feature requires a premium subscription. Please upgrade your plan to access it." | 功能需要更高套餐 | 請升級訂閱套餐以使用此功能。 |
| "Unable to find your subscription plan. Please contact support if this issue persists." | 未找到套餐 | 如果問題持續存在，請寄信至 [hi@s.ee](mailto:hi@s.ee) 聯絡支援團隊。 |
| "The selected pricing option is no longer available." | 價格選項已過期 | 請重新整理頁面並重新選擇當前可用的價格選項。 |

## 檔案上傳問題

| 錯誤訊息 | 原因 | 解決方法 |
|---------------|-------|----------|
| "File size exceeds limit" | 檔案過大 | 請壓縮檔案，或升級套餐以支援更大的檔案上傳。 |
| "Storage limit exceeded" | 已達到儲存空間配額 | 請刪除部分檔案以釋放空間，或升級套餐以獲得更多儲存空間。 |
| "Unsupported file type" / "File type not allowed" | 檔案格式不受支援 | 請使用支援的檔案格式。 |
| "Maximum concurrent uploads exceeded" | 同時上傳數量過多 | 請等待當前上傳完成後再開始新的上傳。 |
| "This file format is not supported. Allowed formats: JPG, JPEG, PNG, GIF, WEBP, AVIF." | 頭像格式不受支援 | 請使用 JPG、JPEG、PNG、GIF、WEBP 或 AVIF 之一上傳頭像。 |

## 團隊問題

| 錯誤訊息 | 原因 | 解決方法 |
|---------------|-------|----------|
| "This feature is not available on your current plan." | 當前套餐不包含團隊功能 | 請升級到包含團隊功能的套餐。 |
| "You have already created a team." | 團隊已存在 | 每位使用者只能建立 1 個團隊。您可以在儀表板中管理現有團隊。 |
| "You don't have permission to manage this team." | 權限不足 | 請聯絡團隊擁有者取得協助。 |
| "The invitation has already been processed." | 邀請已處理 | 此邀請已被接受或拒絕。 |
| "Please wait at least 24 hours before resending." | 重發邀請過於頻繁 | 請等待至少 24 小時後再向該使用者傳送新邀請。 |

## 電子郵件管理問題

| 錯誤訊息 | 原因 | 解決方法 |
|---------------|-------|----------|
| "Email already exists." | 電子郵件已被使用 | 請使用其他電子郵件地址。 |
| "Main email cannot be deleted." | 不能刪除主電子郵件 | 請先將其他已驗證電子郵件設為主電子郵件，再刪除該電子郵件。 |
| "You need to close two factor verification first." | 必須先關閉 2FA | 修改主電子郵件前，請先停用雙重驗證。 |
| "This email has not been verified." | 電子郵件未驗證 | 請先驗證該電子郵件地址。 |
