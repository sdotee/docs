## Problemas de cuenta e inicio de sesión

| Mensaje de error | Causa | Solución |
|---------------|-------|----------|
| "Invalid email or password. Please check your credentials and try again." | Correo o contraseña incorrectos | Verifique su correo y contraseña. Tenga en cuenta que las contraseñas distinguen mayúsculas y minúsculas. |
| "We couldn't find an account with this email." | Correo no registrado | Verifique su dirección de correo o cree una cuenta nueva. |
| "Your account has been suspended. Please contact support for assistance." | Cuenta suspendida | Contacte con nuestro equipo de soporte en [hi@s.ee](mailto:hi@s.ee). |
| "This login link has expired or is invalid. Please request a new one." | Enlace de acceso vencido | Solicite un nuevo enlace de acceso desde la página de inicio de sesión. |
| "This verification code has expired or is invalid. Please request a new one." | Código de verificación vencido o inválido | Solicite un nuevo código de verificación. |
| "This email is already registered." | Correo ya en uso | Use otro correo o pruebe a iniciar sesión con ese correo. |
| "The current password you entered is incorrect." | Contraseña actual incorrecta al cambiarla | Verifique su contraseña actual y vuelva a intentarlo. |

**Nota:** solo su correo principal funciona para iniciar sesión. Otros correos verificados de la cuenta no sirven para autenticación. Los correos de restablecimiento de contraseña también se envían solo al correo principal.

## Problemas con el nombre de usuario

| Mensaje de error | Causa | Solución |
|---------------|-------|----------|
| "This username is already taken. Please choose a different one." | Nombre de usuario ya en uso | Elija un nombre de usuario diferente. |
| "Username can only be changed once." | Límite de cambios alcanzado | Lamentablemente, el nombre de usuario solo puede cambiarse una vez por cuenta. Este cambio es permanente. |

## Problemas con enlaces cortos

| Mensaje de error | Causa | Solución |
|---------------|-------|----------|
| "This slug is already taken. Please choose a different one." | Slug personalizado ya en uso | Pruebe otro slug personalizado para su enlace corto. |
| "This slug cannot be used. Please choose a different one." | El sistema reserva el slug | Este slug está reservado. Elija otro diferente. |
| "This short link does not exist or is no longer available." | El enlace corto no existe o se eliminó | Verifique que la URL del enlace sea correcta. Puede que se haya eliminado. |
| "The domain linked to this short URL is unavailable or no longer exists." | Dominio no disponible | Contacte con soporte o pruebe otro dominio. |
| "Please enter a valid destination URL starting with http:// or https://" | Formato de URL no válido | Asegúrese de que su URL de destino empiece por `http://` o `https://`. |
| "This destination URL is not allowed. Please use a different link." | URL de destino bloqueada | Esta URL está en nuestra lista de bloqueo. Use otra URL de destino. |
| "Incorrect password. Please try again." | Contraseña incorrecta para un enlace protegido | Introduzca la contraseña correcta para acceder a este enlace. |
| "URL has already been shortened." | La URL ya se acortó | Esta URL ya tiene asociado un enlace corto existente. |

## Problemas de suscripción y planes

| Mensaje de error | Causa | Solución |
|---------------|-------|----------|
| "You already have an active subscription plan." | Ya está suscrito | Si quiere cambiar de plan, cancele primero su suscripción actual. |
| "You don't have an active subscription to cancel." | No hay suscripción activa | Actualmente no tiene una suscripción activa que cancelar. |
| "This feature requires a premium subscription. Please upgrade your plan to access it." | La función requiere un nivel superior | Actualice su suscripción para acceder a esta función. |
| "Unable to find your subscription plan. Please contact support if this issue persists." | Plan no encontrado | Contacte con nuestro equipo de soporte en [hi@s.ee](mailto:hi@s.ee). |
| "The selected pricing option is no longer available." | Opción de precio vencida | Actualice la página y seleccione una opción de precio vigente. |

## Problemas de subida de archivos

| Mensaje de error | Causa | Solución |
|---------------|-------|----------|
| "File size exceeds limit" | El archivo es demasiado grande | Comprima el archivo o actualice su plan para subir archivos más grandes. |
| "Storage limit exceeded" | Se alcanzó la cuota de almacenamiento | Elimine algunos archivos para liberar espacio o actualice su plan para obtener más almacenamiento. |
| "Unsupported file type" / "File type not allowed" | Formato de archivo no compatible | Use un formato de archivo compatible. |
| "Maximum concurrent uploads exceeded" | Demasiadas subidas simultáneas | Espere a que terminen las subidas actuales antes de iniciar nuevas. |
| "This file format is not supported. Allowed formats: JPG, JPEG, PNG, GIF, WEBP, AVIF." | Formato de avatar no compatible | Suba su avatar en uno de estos formatos compatibles: JPG, JPEG, PNG, GIF, WEBP o AVIF. |

## Problemas de equipo

| Mensaje de error | Causa | Solución |
|---------------|-------|----------|
| "This feature is not available on your current plan." | La función de equipo no está incluida | Actualice a un plan que incluya funciones de equipo. |
| "You have already created a team." | El equipo ya existe | Cada usuario solo puede crear un equipo. Puede gestionar su equipo actual desde el panel. |
| "You don't have permission to manage this team." | Permisos insuficientes | Contacte con el propietario de su equipo. |
| "The invitation has already been processed." | Invitación ya procesada | Esta invitación ya fue aceptada o rechazada. |
| "Please wait at least 24 hours before resending." | Reenvío demasiado pronto | Espere 24 horas antes de enviar otra invitación a este usuario. |

## Problemas al eliminar la cuenta

| Mensaje de error | Causa | Solución |
|---------------|-------|----------|
| "Your team has other members, please delete the team first." | El equipo aún tiene miembros | Quite a todos los miembros del equipo o elimine el equipo antes de borrar su cuenta. |
| "Too many deletion requests, please try again later." | Demasiadas solicitudes | Espere un momento antes de volver a intentarlo. |
| "Invalid or expired verification code." | Código de verificación inválido o vencido | Solicite un nuevo código de verificación. |

## Problemas de gestión de correo electrónico

| Mensaje de error | Causa | Solución |
|---------------|-------|----------|
| "Email already exists." | Correo ya en uso | Use una dirección de correo diferente. |
| "Main email cannot be deleted." | No se puede eliminar el correo principal | Configure otro correo verificado como correo principal antes de eliminar este. |
| "You need to close two factor verification first." | Primero debe desactivar la 2FA | Desactive la autenticación en dos pasos antes de cambiar su correo principal. |
| "This email has not been verified." | Correo no verificado | Verifique primero esa dirección de correo. |
