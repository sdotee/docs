S.EE:s webbläsartillägg ger dig URL-förkortning, textdelning och filuppladdning direkt i webbläsaren. Det finns för Chrome, Firefox och Edge.

Öppen källkod på [GitHub](https://github.com/sdotee/browser-extensions).

## Funktioner

| Funktion | Beskrivning |
|---------|-------------|
| URL-förkortning | Förkorta vilken URL som helst med ett klick, anpassade slugs och flera domänval |
| Textdelning | Dela kodsnuttar och text med stöd för syntaxmarkering |
| Fildelning | Batchuppladdning av filer med framstegsspårning |
| Exportformat | Kopiera resultat som vanlig URL, Markdown, HTML eller BBCode |
| QR-kodsgenerering | Skapa QR-koder för varje förkortad URL |
| QR-kodsexport | Ladda ner QR-koder som PNG, SVG eller PDF (512x512) |
| Högerklicksmeny | Förkorta sida/länk, visa QR, dela markerad text och ladda upp bilder |
| Historikhantering | Flikmedveten historik för URL:er, texter och filer med batchradering |
| Utkast sparas | Inmatningsfält sparas automatiskt mellan sessioner |
| Automatisk kopiering | Kopiera resultat automatiskt till urklipp |
| Mörkt/ljust läge | Automatisk temadetektering med manuell växling |
| Skrivbordsnotiser | Få notiser när åtgärder slutförs via snabbmenyn |
| Fungerar i flera webbläsare | Fungerar i Chrome (MV3), Firefox (MV3) och Edge |
