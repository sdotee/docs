Berikut adalah dokumentasi API untuk developer. Anda dapat menggunakan API untuk membuat, memperbarui, dan menghapus URL pendek, berbagi teks, dan berbagi file, serta mengambil informasi tentang domain yang tersedia.

API ini dirancang agar sederhana dan mudah digunakan sehingga Anda dapat dengan cepat mengintegrasikan fungsi S.EE ke aplikasi Anda.
