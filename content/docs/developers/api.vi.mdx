Đây là tài liệu API dành cho developer. Bạn có thể dùng API để tạo, cập nhật và xóa URL ngắn, chia sẻ văn bản và chia sẻ tệp, cũng như lấy thông tin về các tên miền khả dụng.

API được thiết kế đơn giản và dễ dùng, cho phép bạn nhanh chóng tích hợp chức năng của S.EE vào ứng dụng của mình.
