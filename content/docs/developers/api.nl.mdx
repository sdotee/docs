Hier vindt u de API-documentatie voor ontwikkelaars. U kunt de API gebruiken om korte URL's, tekstdeling en bestandsdeling te maken, bij te werken en te verwijderen, en om informatie over beschikbare domeinen op te halen.

De API is ontworpen om eenvoudig te gebruiken te zijn, zodat u S.EE snel in uw applicaties kunt integreren.
