Siit leiad arendajatele mõeldud API dokumentatsiooni. API abil saad luua, uuendada ja kustutada lühikesi URL-e, teksti jagamisi ja failijagamisi ning pärida teavet saadaolevate domeenide kohta.

API on loodud lihtsaks ja hõlpsasti kasutatavaks, et saaksid S.EE funktsionaalsuse kiiresti oma rakendustesse integreerida.
