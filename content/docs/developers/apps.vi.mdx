| Nền tảng | Yêu cầu | Công nghệ |
|-----------|---------|-----------|

## Tính năng

### Rút gọn URL

- Tạo liên kết ngắn với slug tùy chỉnh, tiêu đề, mật khẩu và ngày hết hạn
- Chọn từ nhiều tên miền
- Xem thống kê liên kết — hôm nay, tháng này và tổng lượt truy cập
- Tìm kiếm theo slug, URL đích, tiêu đề và tên miền
- Chỉnh sửa hoặc xóa liên kết hiện có

### Chia sẻ văn bản

- Chia sẻ văn bản thuần, mã nguồn hoặc Markdown
- Trình soạn thảo chữ mono cho mã
- Slug tùy chỉnh, mật khẩu và ngày hết hạn
- Chọn tên miền và gán tag (tối đa 5 tag)

### Tải tệp lên

- Chọn tệp, chọn từ thư viện ảnh hoặc chụp bằng camera
- Dán từ clipboard với chuyển đổi WebP tự động (iOS)
- Hiển thị tiến độ tải lên theo thời gian thực
- 9 định dạng liên kết — liên kết trực tiếp, trang chia sẻ, BBCode, HTML, Markdown, v.v.
- Sao chép hàng loạt và xóa hàng loạt trên tất cả danh sách

### Bảng điều khiển sử dụng

- Hiển thị mức sử dụng dung lượng bằng progress bar trực quan
- Giới hạn tốc độ cho lệnh gọi API, liên kết, bản chia sẻ văn bản, lượt tải lên và mã QR
- Thống kê sử dụng theo ngày và theo tháng

### Cài đặt

- Cấu hình URL cơ sở cho phiên bản tự lưu trữ
- Quản lý khóa API với xác minh một chạm
- Chọn tên miền mặc định theo từng dịch vụ (liên kết, văn bản, tệp)
- Chủ đề: hệ thống / sáng / tối
- Màu động / Material You (Android)

## iOS

Được xây dựng bằng **SwiftUI** và **SwiftData** để mang lại trải nghiệm iOS native.

- **Bộ chọn ảnh** — chọn từ thư viện Ảnh
- **Máy ảnh** — chụp ảnh (JPG) hoặc quay video (MOV sang MP4)
- Pull-to-refresh cho tag và danh sách
- Lưu trữ API key an toàn bằng Keychain

## Android

Được xây dựng bằng **Jetpack Compose** và **Material 3** (Material You) để mang lại trải nghiệm Android native.

- **Đích chia sẻ** — nhận văn bản, URL và tệp được chia sẻ từ ứng dụng khác
- **Lối tắt ứng dụng** — nhấn giữ biểu tượng ứng dụng để thao tác nhanh
- Hỗ trợ cử chỉ **Predictive Back** (Android 14+)
- **Biểu tượng thích ứng** với hỗ trợ biểu tượng theo chủ đề
- Cơ sở dữ liệu Room cho lịch sử cục bộ
- EncryptedSharedPreferences cho lưu trữ an toàn

## Ngăn xếp công nghệ

### iOS

| Lớp | Công nghệ |
|-----|-----------|
| UI | SwiftUI |
| Lưu trữ | SwiftData |
| Mạng | URLSession (dựa trên actor) |
| Bảo mật | Keychain |
| Ảnh thu nhỏ | QuickLookThumbnailing |
| Xử lý ảnh | Swift-WebP |

### Android

| Lớp | Công nghệ |
|-----|-----------|
| UI | Jetpack Compose + Material 3 |
| Kiến trúc | MVVM + Repository |
| Mạng | Retrofit 3.0 + OkHttp 5 |
| DI | Hilt (Dagger) |
| CSDL cục bộ | Room |
| Bảo mật | EncryptedSharedPreferences |
| Tải ảnh | Coil 3 |

Cả hai ứng dụng đều là mã nguồn mở theo giấy phép MIT:

- **iOS**: [sdotee/app/macos](https://github.com/sdotee/app/tree/main/macos)

## Bắt đầu

1. Tải ứng dụng từ [App Store](https://s.ee/ios) hoặc [Google Play](https://s.ee/android)
2. Tạo tài khoản tại [s.ee](https://s.ee) nếu bạn chưa có
3. [Tạo API key](https://s.ee/user/developers/) từ bảng điều khiển của bạn
4. Mở ứng dụng, vào **Cài đặt**, nhập khóa API và nhấn **Xác minh**
