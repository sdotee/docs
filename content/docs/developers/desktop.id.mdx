Pendekkan tautan, bagikan teks, dan unggah file dari desktop Anda, dengan aplikasi native untuk macOS dan Linux.

- [sdotee/app](https://github.com/sdotee/app)

## Unduh

- **macOS**: [https://s.ee/macos](https://s.ee/macos)
- **Linux**: [https://s.ee/linux](https://s.ee/linux)

## Platform yang Didukung

| Platform | Status | Teknologi |
|----------|--------|-----------|
| macOS | Tersedia | SwiftUI |
| Linux | Tersedia | GTK4 + libadwaita + Rust |
| Windows | Dalam pengembangan | - |

## Fitur

- **Pemendekan URL** — Slug kustom, judul, kata sandi, dan kedaluwarsa
- **Berbagi Teks** — Teks biasa, kode sumber, atau Markdown dengan editor monospasi
- **Unggah File** — Seret dan lepas, tempel dari clipboard, atau pilih file
- **Pembuatan Kode QR** — Ekspor sebagai PNG, SVG, atau PDF
- **Riwayat** — Catatan lokal untuk semua tautan, teks, dan file yang dibuat
- **Dukungan Multi-Domain** — Pilih dari beberapa domain per layanan
- **Dasbor Penggunaan** — Penggunaan penyimpanan dan batas laju dalam sekali lihat

## macOS

Dibangun dengan **SwiftUI** dan **SwiftData**. Mendukung penuh pintasan keyboard dan integrasi bilah menu.

- [sdotee/app/macos](https://github.com/sdotee/app/tree/main/macos)
- Memerlukan macOS 14.0+

### Sorotan

- **Bilah Menu** — Pendekkan URL, tempel & unggah, dan akses item terbaru tanpa membuka jendela utama
- **Seret & Lepas** — Letakkan file di mana saja pada area unggah
- **Tempel dari Clipboard** (Cmd+V) — Otomatis mengonversi gambar ke WebP dengan fallback PNG
- **Keychain** — Penyimpanan kunci API yang aman melalui framework keamanan macOS

### Pintasan Keyboard

| Pintasan | Aksi |
|----------|------|
| Cmd+N | Tautan pendek baru |
| Cmd+Shift+N | Berbagi teks baru |
| Cmd+V | Tempel & unggah dari clipboard |
| Cmd+, | Pengaturan |
| Cmd+Q | Keluar |

### Tumpukan Teknologi

| Lapisan | Teknologi |
|---------|-----------|
| UI | SwiftUI |
| Penyimpanan | SwiftData |
| Jaringan | URLSession (berbasis actor) |
| Keamanan | Keychain |
| Thumbnail | QuickLookThumbnailing |
| Pemrosesan Gambar | Swift-WebP |

## Linux

Dibangun dengan **GTK4**, **libadwaita**, dan **Rust**. Mengikuti GNOME Human Interface Guidelines.

- [sdotee/app/linux](https://github.com/sdotee/app/tree/main/linux)
- Memerlukan GTK 4.14+ dan libadwaita 1.5+

### Sorotan

- **Pengalaman GNOME Native** — Tata letak adaptif yang mengikuti GNOME HIG
- **Ekspor Kode QR** — PNG, SVG, atau PDF
- **Berkas Konfigurasi** — `~/.config/see/config.toml` untuk kunci API, URL dasar, dan domain bawaan
- **Variabel Lingkungan** — `SEE_API_KEY`, `SEE_BASE_URL`, `SEE_TIMEOUT` menimpa berkas konfigurasi

### Pintasan Keyboard

| Pintasan | Aksi |
|----------|------|
| Ctrl+1 | Pindah ke tampilan tautan |
| Ctrl+2 | Pindah ke tampilan teks |
| Ctrl+3 | Pindah ke tampilan file |
| Ctrl+, | Buka preferensi |
| Ctrl+? | Tampilkan pintasan keyboard |
| Ctrl+Q | Keluar |

### Tumpukan Teknologi

| Lapisan | Teknologi |
|---------|-----------|
| UI | GTK4 + libadwaita |
| Bahasa | Rust (edisi 2024) |
| Klien API | see-sdk |
| Kode QR | Bawaan (ekspor PNG, SVG, PDF) |

### Penyimpanan Data

- **Konfigurasi**: `~/.config/see/config.toml`
- **Riwayat**: `~/.local/share/see/history.json`

## Mulai

1. Unduh untuk [macOS](https://s.ee/macos) atau [Linux](https://s.ee/linux)
2. Buat akun di [s.ee](https://s.ee) jika Anda belum memilikinya
3. [Buat kunci API](https://s.ee/user/developers/) dari dasbor Anda
4. Buka aplikasi, masuk ke **Preferensi**, lalu masukkan kunci API Anda
