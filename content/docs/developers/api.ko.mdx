이 문서는 개발자를 위한 API 문서입니다. API 를 사용해 단축 URL, 텍스트 공유, 파일 공유를 생성, 수정, 삭제할 수 있으며, 사용 가능한 도메인 정보도 조회할 수 있습니다.

S.EE API 는 간단하고 사용하기 쉽게 설계되어 있어 애플리케이션에 빠르게 연동할 수 있습니다.
