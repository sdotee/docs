Здесь собрана документация по API для разработчиков. С его помощью можно создавать, обновлять и удалять короткие ссылки, текстовые публикации и файлы, а также получать сведения о доступных доменах.

API устроен просто и позволяет быстро встроить возможности S.EE в ваши приложения.
