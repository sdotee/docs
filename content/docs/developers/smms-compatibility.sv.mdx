S.EE erbjuder ett SM.MS-kompatibelt API-lager så att du kan migrera från SM.MS till S.EE med minimala kodändringar. Byt bara ut API-endpointen så är du igång.

## Snabb migrering

Byt ut SM.MS API-bas-URL mot motsvarande i S.EE:

| SM.MS | S.EE |
|-------|------|
| `https://sm.ms/api/v2` | `https://s.ee/api/v1/file` |

## Endpoint som stöds

| SM.MS-endpoint | S.EE-endpoint | Beskrivning |
|----------------|---------------|-------------|
| `POST /api/v2/upload` | `POST /api/v1/file/upload` | Ladda upp bild |
| `GET /api/v2/upload_history` | `GET /api/v1/files` | Hämta uppladdningshistorik |

## Ladda upp bild

### Förfrågan

```bash
curl -X POST "https://s.ee/api/v1/file/upload" \
  -H "Authorization: your-api-key" \
  -F "smfile=@/path/to/image.png" \
  -F "domain=sm.ms" \
  -F "custom_slug=my-image"
```

### Parametrar

| Parameter | Typ | Krävs | Beskrivning |
|-----------|------|----------|-------------|
| `smfile` | file | Ja | Bildfilen som ska laddas upp, alias: `file` |
| `domain` | string | Nej | Anpassad domän för den korta länken |
| `custom_slug` | string | Nej | Anpassad slug för URL:en, slumpmässig om inget anges |

## Uppladdningshistorik

### Förfrågan

```bash
curl "https://s.ee/api/v1/files" \
  -H "Authorization: your-api-key"
```

### Parametrar

| Parameter | Typ | Krävs | Beskrivning |
|-----------|------|----------|-------------|
| `page` | integer | Nej | Sidnummer, standard 1, 30 filer per sida |

## Skillnader jämfört med SM.MS

Även om S.EE är kompatibelt med SM.MS API finns några förbättringar:

| Funktion | SM.MS | S.EE |
|---------|-------|------|
| Egen domän | Nej | Ja |
| Anpassad slug | Nej | Ja |
| Filstorleksgräns | 5MB | Varierar per abonnemang |
| Format som stöds | Endast bilder | Bilder, dokument och mer |

## Hämta API-nyckel

För att använda S.EE API behöver du en API-nyckel:

1. Registrera dig eller logga in på [s.ee](https://s.ee)
2. Gå till **[Tools > API Token](https://s.ee/user/developers/)**
3. Generera en ny API-nyckel

## Behöver du hjälp?

Om du stöter på problem under migreringen kan du [kontakta supporten](https://s.ee/contact) eller läsa vår [API-dokumentation](/docs/sv/developers/api).
