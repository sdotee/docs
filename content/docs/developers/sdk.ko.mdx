## 공식 SDK

S.EE 는 타입 안정성과 쉬운 연동을 위해 공식 SDK 를 제공합니다.

| Language | Repository | Description |
|----------|------------|-------------|
| PHP | [sdotee/sdk.php](https://github.com/sdotee/sdk.php) | PHP 애플리케이션용 개발자 툴킷 |
| TypeScript | [sdotee/sdk.ts](https://github.com/sdotee/sdk.ts) | JavaScript/TypeScript 프로젝트를 위한 타입 안전 연동 |
| Go | [sdotee/sdk.go](https://github.com/sdotee/sdk.go) | 링크 관리를 위한 Go 언어 지원 |
| Python | [sdotee/sdk.py](https://github.com/sdotee/sdk.py) | Python 기반 URL 단축 및 파일 공유 |
| Java | [sdotee/sdk.java](https://github.com/sdotee/sdk.java) | 엔터프라이즈 Java 구현 |
| Rust | [sdotee/sdk.rs](https://github.com/sdotee/sdk.rs) | 시스템 프로그래밍 언어 지원 |
| Zig | [sdotee/sdk.zig](https://github.com/sdotee/sdk.zig) | 신규 언어 지원 (실험적) |

## MCP 서버

AI/LLM 연동에는 MCP (Model Context Protocol) 서버를 사용할 수 있습니다.

- [sdotee/cli-mcp-server](https://github.com/sdotee/cli-mcp-server)

### 기능

- **URL Management**: 단축 URL 생성, 수정, 삭제 및 도메인 목록 조회
- **Content Sharing**: 만료 및 비밀번호 보호가 포함된 텍스트와 코드 스니펫 공유
- **File Operations**: 파일 업로드 및 삭제
- **Time Utilities**: 시간대 지원과 함께 현재 시간 정보 제공

### 요구 사항

- Go 1.25+
- 유효한 S.EE API Key

### 빌드

```bash
# Using Makefile (current OS)
make build

# Build for all platforms
make build-all

# Or using Go directly
go build -o bin/cli-mcp-server .
```

### 실행

```bash
./bin/cli-mcp-server -api-key "your-api-key"
```

환경 변수로 API Key 를 설정할 수도 있습니다.

```bash
export SEE_API_KEY="your-api-key"
./bin/cli-mcp-server
```

서버 기본 포트는 30101 이며, MCP 연결에는 Server-Sent Events 를 사용합니다. 연동하려면 MCP 서버 URL 을 LLM 클라이언트 설정 (예: Cherry Studio) 에 입력하세요.

## 설치

### PHP

```bash
composer require sdotee/sdk
```

### TypeScript

```bash

```

### Go

```bash
go get github.com/sdotee/sdk.go
```

### Python

