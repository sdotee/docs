Voici la documentation API destinée aux développeurs. Vous pouvez utiliser l’API pour créer, mettre à jour et supprimer des URL courtes, du partage de texte et du partage de fichiers, ainsi que pour récupérer des informations sur les domaines disponibles.

L’API est conçue pour être simple et facile à utiliser, afin de vous permettre d’intégrer rapidement les fonctionnalités de S.EE dans vos applications.
