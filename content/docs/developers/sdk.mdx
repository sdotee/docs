## Installation

### Java

```xml
<!-- Maven -->
<dependency>
    <groupId>s.ee</groupId>
    <artifactId>see-java-sdk</artifactId>
    <version>1.0.0</version>
</dependency>
```

### Rust

```toml
# Cargo.toml
[dependencies]
see-sdk = "1.0.0"
```

### TypeScript

```bash
npm install see-sdk
```

### Go

```bash
go get github.com/sdotee/sdk.go
```

### Python

```bash
pip install see-sdk
```
