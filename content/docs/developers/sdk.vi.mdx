## SDK chính thức

S.EE cung cấp SDK chính thức để đảm bảo type safety và dễ tích hợp:

| Ngôn ngữ | Kho mã nguồn | Mô tả |
|----------|------------|-------------|
| PHP | [sdotee/sdk.php](https://github.com/sdotee/sdk.php) | Bộ công cụ cho developer xây dựng ứng dụng PHP |
| TypeScript | [sdotee/sdk.ts](https://github.com/sdotee/sdk.ts) | Tích hợp an toàn kiểu cho dự án JavaScript/TypeScript |
| Go | [sdotee/sdk.go](https://github.com/sdotee/sdk.go) | Hỗ trợ Go cho quản lý liên kết |
| Python | [sdotee/sdk.py](https://github.com/sdotee/sdk.py) | Rút gọn URL và chia sẻ tệp bằng Python |
| Java | [sdotee/sdk.java](https://github.com/sdotee/sdk.java) | Triển khai Java cấp enterprise |
| Rust | [sdotee/sdk.rs](https://github.com/sdotee/sdk.rs) | Hỗ trợ cho ngôn ngữ lập trình hệ thống |
| Zig | [sdotee/sdk.zig](https://github.com/sdotee/sdk.zig) | Hỗ trợ cho ngôn ngữ mới nổi (thử nghiệm) |

## Máy chủ MCP

Đối với tích hợp AI/LLM, hãy dùng máy chủ MCP (Model Context Protocol) của chúng tôi:

- [sdotee/cli-mcp-server](https://github.com/sdotee/cli-mcp-server)

### Tính năng

- **Quản lý URL**: Tạo, cập nhật, xóa và liệt kê tên miền cho URL ngắn
- **Chia sẻ nội dung**: Chia sẻ đoạn mã và văn bản với tùy chọn hết hạn và bảo vệ bằng mật khẩu
- **Thao tác tệp**: Tải lên và xóa tệp
- **Tiện ích thời gian**: Truy cập thông tin thời gian hiện tại với hỗ trợ múi giờ

### Yêu cầu

- Go 1.25+
- API Key S.EE hợp lệ

### Biên dịch

```bash
# Dùng Makefile (hệ điều hành hiện tại)
make build

# Biên dịch cho mọi nền tảng
make build-all

# Hoặc dùng Go trực tiếp
go build -o bin/cli-mcp-server .
```

### Chạy

```bash
./bin/cli-mcp-server -api-key "your-api-key"
```

Bạn cũng có thể đặt khóa API bằng environment variable:

```bash
export SEE_API_KEY="your-api-key"
./bin/cli-mcp-server
```

Máy chủ mặc định chạy trên cổng 30101 và dùng Server-Sent Events cho kết nối MCP. Hãy cấu hình URL của máy chủ MCP trong phần cài đặt của ứng dụng LLM (ví dụ Cherry Studio) để tích hợp.

## Cài đặt

### PHP

```bash
composer require sdotee/sdk
```

### TypeScript

```bash
npm install see-sdk
```

### Go

```bash
go get github.com/sdotee/sdk.go
```

### Python

```bash
pip install see-sdk
```

### Java

Xem [repository sdk.java](https://github.com/sdotee/sdk.java) để biết hướng dẫn cài đặt.

### Rust

Xem [repository sdk.rs](https://github.com/sdotee/sdk.rs) để biết hướng dẫn cài đặt.

### Zig

Xem [repository sdk.zig](https://github.com/sdotee/sdk.zig) để biết hướng dẫn cài đặt.
