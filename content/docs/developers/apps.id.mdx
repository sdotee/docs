## Fitur

### Pemendekan URL

- Buat tautan pendek dengan slug kustom, judul, kata sandi, dan kedaluwarsa
- Pilih dari beberapa domain
- Lihat statistik tautan — hari ini, bulan ini, dan total kunjungan
- Cari berdasarkan slug, URL tujuan, judul, dan domain
- Sunting atau hapus tautan yang sudah ada

### Berbagi Teks

- Bagikan teks biasa, source code, atau Markdown
- Suntingor monospaced untuk kode
- Slug kustom, kata sandi, dan kedaluwarsa
- Pemilihan domain dan penetapan tag (hingga 5 tag)

### Unggah File

- Pilih file, pilih dari galeri, atau ambil dengan kamera
- Paste dari clipboard dengan konversi otomatis ke WebP (iOS)
- Progres unggah real-time
- 9 format tautan — Direct Link, Share Page, BBCode, HTML, Markdown, dan lainnya
- Salin batch dan hapus batch di semua daftar

### Dasbor Penggunaan

- Penggunaan penyimpanan dengan progress bar visual
- Rate limit untuk API Calls, Links, Text Shares, Uploads, dan QR Codes
- Jumlah penggunaan harian dan bulanan

### Pengaturan

- Konfigurasi URL dasar untuk instance self-hosted
- Manajemen API Key dengan verifikasi sekali ketuk
- Pemilihan domain default per layanan (tautan, teks, file)
- Tema: System / Light / Dark
- Dynamic Color / Material You (Android)

## iOS

Dibangun dengan **SwiftUI** dan **SwiftData** untuk pengalaman iOS native.

- **Photo Picker** — pilih dari Photos library
- **Camera** — ambil foto (JPG) atau video (MOV ke MP4)
- Pull-to-refresh untuk tag dan daftar
- Penyimpanan kunci API yang aman berbasis Keychain

## Android

Dibangun dengan **Jetpack Compose** dan **Material 3** (Material You) untuk pengalaman Android native.

- **Share Target** — menerima teks, URL, dan file yang dibagikan dari aplikasi lain
- **App Shortcuts** — tekan lama ikon aplikasi untuk aksi cepat
- Dukungan gesture **Predictive Back** (Android 14+)
- **Adaptive Icon** dengan dukungan themed icon
- Database Room untuk riwayat lokal
- EncryptedSharedPreferences untuk penyimpanan aman

## Tumpukan Teknologi

### iOS

| Lapisan | Teknologi |
|-------|-----------|
| UI | SwiftUI |
| Penyimpanan | SwiftData |
| Jaringan | URLSession (berbasis actor) |
| Keamanan | Keychain |
| Thumbnail | QuickLookThumbnailing |
| Pemrosesan Gambar | Swift-WebP |

### Android

| Lapisan | Teknologi |
|-------|-----------|
| UI | Jetpack Compose + Material 3 |
| Arsitektur | MVVM + Repository |
| Jaringan | Retrofit 3.0 + OkHttp 5 |
| DI | Hilt (Dagger) |
| DB Lokal | Room |
| Keamanan | EncryptedSharedPreferences |
| Pemuatan Gambar | Coil 3 |

## Mulai

1. Unduh dari [App Store](https://s.ee/ios) atau [Google Play](https://s.ee/android)
2. Buat akun di [s.ee](https://s.ee) jika Anda belum memilikinya
3. [Buat kunci API](https://s.ee/user/developers/) dari dasbor Anda
4. Buka aplikasi, masuk ke **Pengaturan**, masukkan kunci API Anda, lalu ketuk **Verifikasi**
