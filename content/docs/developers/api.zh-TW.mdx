這裡是面向開發者的 API 文件。您可以使用 API 建立、更新和刪除短網址、文字分享和檔案分享，也可以取得可用網域資訊。

該 API 設計簡潔、易於使用，便於您快速將 S.EE 功能整合到自己的應用中。
