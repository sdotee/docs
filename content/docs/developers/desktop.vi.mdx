Rút gọn liên kết, chia sẻ văn bản và tải tệp lên từ desktop của bạn — với ứng dụng native cho macOS và Linux.

- [sdotee/app](https://github.com/sdotee/app)

## Tải xuống

- **macOS**: [https://s.ee/macos](https://s.ee/macos)
- **Linux**: [https://s.ee/linux](https://s.ee/linux)

## Nền tảng được hỗ trợ

| Nền tảng | Trạng thái | Công nghệ |
|----------|--------|------------|
| macOS | Available | SwiftUI |
| Linux | Available | GTK4 + libadwaita + Rust |
| Windows | In Development | - |

## Tính năng

- **Rút gọn URL** — Slug tùy chỉnh, tiêu đề, mật khẩu và thời gian hết hạn
- **Chia sẻ văn bản** — Văn bản thuần, mã nguồn hoặc Markdown với trình soạn thảo chữ mono
- **Tải tệp lên** — Kéo thả, dán từ clipboard hoặc chọn tệp
- **Tạo mã QR** — Xuất dưới dạng PNG, SVG hoặc PDF
- **Lịch sử** — Lưu cục bộ mọi liên kết, văn bản và tệp đã tạo
- **Hỗ trợ nhiều tên miền** — Chọn từ nhiều tên miền cho từng dịch vụ
- **Bảng điều khiển sử dụng** — Xem nhanh dung lượng lưu trữ và giới hạn tốc độ

## macOS

Được xây dựng bằng **SwiftUI** và **SwiftData**. Hỗ trợ đầy đủ phím tắt và tích hợp thanh menu.

- [sdotee/app/macos](https://github.com/sdotee/app/tree/main/macos)
- Yêu cầu macOS 14.0+

### Điểm nổi bật

- **Thanh menu** — Rút gọn URL, dán & tải lên, và truy cập mục gần đây mà không cần mở cửa sổ chính
- **Kéo và thả** — Thả tệp ở bất kỳ đâu trong vùng tải lên
- **Dán từ clipboard** (Cmd+V) — Tự động chuyển ảnh sang WebP, dùng PNG dự phòng khi cần
- **Keychain** — Lưu khóa API an toàn qua macOS Security framework

### Phím tắt

| Phím tắt | Hành động |
|----------|--------|
| Cmd+N | Liên kết ngắn mới |
| Cmd+Shift+N | Bản chia sẻ văn bản mới |
| Cmd+V | Dán và tải lên từ clipboard |
| Cmd+, | Cài đặt |
| Cmd+Q | Thoát |

### Ngăn xếp công nghệ

| Lớp | Công nghệ |
|-------|-----------|
| UI | SwiftUI |
| Lưu trữ | SwiftData |
| Mạng | URLSession (dựa trên actor) |
| Bảo mật | Keychain |
| Ảnh thu nhỏ | QuickLookThumbnailing |
| Xử lý ảnh | Swift-WebP |

## Linux

Được xây dựng bằng **GTK4**, **libadwaita** và **Rust**. Tuân theo GNOME Human Interface Guidelines.

- [sdotee/app/linux](https://github.com/sdotee/app/tree/main/linux)
- Yêu cầu GTK 4.14+ và libadwaita 1.5+

### Điểm nổi bật

- **Trải nghiệm GNOME native** — Giao diện thích ứng tuân theo GNOME HIG
- **Xuất mã QR** — PNG, SVG hoặc PDF
- **Tệp cấu hình** — `~/.config/see/config.toml` cho khóa API, URL cơ sở và tên miền mặc định
- **Biến môi trường** — `SEE_API_KEY`, `SEE_BASE_URL`, `SEE_TIMEOUT` sẽ ghi đè tệp cấu hình

### Phím tắt

| Phím tắt | Hành động |
|----------|--------|
| Ctrl+1 | Chuyển sang chế độ xem liên kết |
| Ctrl+2 | Chuyển sang chế độ xem văn bản |
| Ctrl+3 | Chuyển sang chế độ xem tệp |
| Ctrl+, | Mở tùy chọn |
| Ctrl+? | Hiển thị phím tắt |
| Ctrl+Q | Thoát |

### Ngăn xếp công nghệ

| Lớp | Công nghệ |
|-------|-----------|
| UI | GTK4 + libadwaita |
| Ngôn ngữ | Rust (2024 edition) |
| Trình khách API | see-sdk |
| Mã QR | Tích hợp sẵn (xuất PNG, SVG, PDF) |

### Lưu trữ dữ liệu

- **Cấu hình**: `~/.config/see/config.toml`
- **Lịch sử**: `~/.local/share/see/history.json`

## Bắt đầu

1. Tải phiên bản cho [macOS](https://s.ee/macos) hoặc [Linux](https://s.ee/linux)
2. Tạo tài khoản tại [s.ee](https://s.ee) nếu bạn chưa có
3. [Tạo khóa API](https://s.ee/user/developers/) từ bảng điều khiển của bạn
4. Mở ứng dụng, vào **Tùy chọn** và nhập khóa API
