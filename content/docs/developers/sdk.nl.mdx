| Go | [sdotee/sdk.go](https://github.com/sdotee/sdk.go) | Ondersteuning in Go voor linkbeheer |
| Python | [sdotee/sdk.py](https://github.com/sdotee/sdk.py) | URL-verkorting en bestandsdeling in Python |
| Java | [sdotee/sdk.java](https://github.com/sdotee/sdk.java) | Java-implementatie voor enterprisegebruik |

## MCP-server

Gebruik voor AI- en LLM-integraties onze MCP-server, Model Context Protocol:

- [sdotee/cli-mcp-server](https://github.com/sdotee/cli-mcp-server)

### Functies

- **URL-beheer**: korte URL's maken, bijwerken, verwijderen en domeinen tonen
- **Content delen**: codefragmenten en tekst delen met vervaldatum en wachtwoordbeveiliging
- **Bestandsacties**: bestanden uploaden en verwijderen
- **Tijdhulpmiddelen**: huidige tijdinformatie met tijdzoneondersteuning

### Vereisten

- Go 1.25+
- Geldige S.EE API-sleutel

### Bouwen

```bash
# Met Makefile, huidig OS
make build

# Voor alle platforms bouwen
make build-all

# Of direct met Go
go build -o bin/cli-mcp-server .
```

### Uitvoeren

```bash
./bin/cli-mcp-server -api-key "your-api-key"
```

U kunt de API-sleutel ook via een omgevingsvariabele instellen:

```bash
export SEE_API_KEY="your-api-key"
./bin/cli-mcp-server
```

De server gebruikt standaard poort 30101 en Server-Sent Events voor MCP-verbindingen. Stel de MCP-server-URL in de instellingen van uw LLM-client in, bijvoorbeeld Cherry Studio, om te integreren.

## Installatie

### PHP

```bash
composer require sdotee/sdk
```

### TypeScript

```bash
npm install see-sdk
```

### Go

```bash
go get github.com/sdotee/sdk.go
```

### Python

```bash
pip install see-sdk
```
