Aqui encontra a documentação da API para desenvolvedores. Pode usar a API para criar, atualizar e eliminar URLs curtos, partilhas de texto e partilhas de ficheiros, bem como obter informações sobre domínios disponíveis.

A API foi concebida para ser simples e fácil de usar, permitindo-lhe integrar rapidamente as funcionalidades da S.EE nas suas aplicações.
