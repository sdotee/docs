Här är API-dokumentationen för utvecklare. Du kan använda API:t för att skapa, uppdatera och ta bort korta URL:er, textdelningar och fildelningar samt hämta information om tillgängliga domäner.

API:t är utformat för att vara enkelt att använda så att du snabbt kan integrera S.EE i dina applikationer.
