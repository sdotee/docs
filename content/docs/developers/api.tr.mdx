Bu sayfa geliştiriciler için API dokümantasyonunun genel bakışını sunar. API ile kısa URL'ler, metin paylaşımları ve dosya paylaşımları oluşturabilir, güncelleyebilir ve silebilir; ayrıca kullanılabilir alan adları hakkında bilgi alabilirsiniz.

API, uygulamalarınıza S.EE işlevlerini hızlı biçimde entegre edebilmeniz için basit ve kullanımı kolay olacak şekilde tasarlanmıştır.
