Aqui está a documentação da API para desenvolvedores. Você pode usar a API para criar, atualizar e excluir URLs curtas, compartilhamentos de texto e compartilhamentos de arquivos, além de obter informações sobre domínios disponíveis.

A API foi projetada para ser simples e fácil de usar, permitindo integrar rapidamente os recursos da S.EE em seus aplicativos.
