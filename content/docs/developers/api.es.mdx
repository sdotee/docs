Aquí encontrará la documentación de la API para desarrolladores. Puede usar la API para crear, actualizar y eliminar URL cortas, textos compartidos y archivos compartidos, así como para obtener información sobre los dominios disponibles.

La API está diseñada para ser simple y fácil de usar, lo que le permite integrar rápidamente las funciones de S.EE en sus aplicaciones.
