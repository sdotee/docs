Hier finden Sie die API-Dokumentation für Entwickler. Sie können die API verwenden, um Kurz-URLs, Textfreigaben und Dateifreigaben zu erstellen, zu aktualisieren und zu löschen sowie Informationen über verfügbare Domains abzurufen.

Die API ist einfach und leicht zu verwenden, sodass Sie S.EE-Funktionalität schnell in Ihre Anwendungen integrieren können.
