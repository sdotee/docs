## Возможности

### Сокращение URL

- Создание коротких ссылок с пользовательским коротким кодом, заголовком, паролем и сроком действия
- Выбор из нескольких доменов
- Просмотр статистики: сегодня, за месяц и всего переходов
- Поиск по короткому коду, URL назначения, заголовку и домену
- Редактирование и удаление существующих ссылок

### Обмен текстом

- Публикация обычного текста, исходного кода и Markdown
- Моноширинный редактор для кода
- Пользовательский короткий код, пароль и срок действия
- Выбор домена и назначение меток (до 5 меток)

### Загрузка файлов

- Выбор файлов, загрузка из галереи или съемка камерой
- Вставка из буфера обмена с автоматическим преобразованием в WebP (iOS)
- Отслеживание прогресса загрузки в реальном времени
- 9 форматов ссылок: Direct Link, Share Page, BBCode, HTML, Markdown и другие
- Пакетное копирование и удаление во всех списках

### Панель использования

- Использование хранилища с наглядной полосой прогресса
- Лимиты для вызовов API, ссылок, текстовых публикаций, загрузок и QR-кодов
- Счетчики дневного и месячного использования

### Настройки

- Настройка базового URL для self-hosted-инстансов
- Управление API-ключом с проверкой в одно касание
- Выбор домена по умолчанию для каждого сервиса (ссылки, текст, файлы)
- Тема: System / Light / Dark
- Dynamic Color / Material You (Android)

## iOS

Разработано на **SwiftUI** и **SwiftData** для нативного опыта на iOS.

- **Photo Picker** — выбор из фотобиблиотеки
- **Camera** — съемка фото (JPG) или видео (MOV в MP4)
- Обновление списков и меток жестом потянуть-вниз-для-обновления
- Безопасное хранение API-ключа в Keychain

## Android

Разработано на **Jetpack Compose** и **Material 3** (Material You) для нативного опыта на Android.

- **Share Target** — получает текст, URL и файлы из других приложений
- **App Shortcuts** — зажмите иконку приложения для быстрых действий
- Поддержка жеста **Predictive Back** (Android 14+)
- **Adaptive Icon** с поддержкой тематических иконок
- База данных Room для локальной истории
- Безопасное хранение через EncryptedSharedPreferences

## Технологический стек

### iOS

| Слой | Технология |
|------|------------|
| UI | SwiftUI |
| Хранилище | SwiftData |
| Сеть | URLSession (на акторах) |
| Безопасность | Keychain |
| Миниатюры | QuickLookThumbnailing |
| Обработка изображений | Swift-WebP |

### Android

| Слой | Технология |
|------|------------|
| UI | Jetpack Compose + Material 3 |
| Архитектура | MVVM + Repository |
| Сеть | Retrofit 3.0 + OkHttp 5 |
| DI | Hilt (Dagger) |
| Локальная БД | Room |
| Безопасность | EncryptedSharedPreferences |
| Загрузка изображений | Coil 3 |

## Первые шаги

1. Скачайте приложение из [App Store](https://s.ee/ios) или [Google Play](https://s.ee/android)
2. Создайте аккаунт на [s.ee](https://s.ee), если его еще нет
3. [Создайте API-ключ](https://s.ee/user/developers/) в панели управления
4. Откройте приложение, перейдите в **Settings**, введите API-ключ и нажмите **Verify**
