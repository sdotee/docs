## Descripción general del panel

Acceda a su [panel](https://s.ee/user/dashboard/) después de iniciar sesión. El panel de S.EE tiene dos áreas principales de navegación:

### Barra lateral izquierda

Acceda a sus funciones principales:

| Función | Descripción |
|---------|-------------|
| [Links](https://s.ee/user/links/) | Gestione sus enlaces cortos |
| [QR Codes](https://s.ee/user/qr/) | Cree y gestione códigos QR |
| [Text Sharing](https://s.ee/user/text/) | Comparta texto, fragmentos de código y Markdown |
| [File Sharing](https://s.ee/user/files/) | Suba y comparta archivos |
| [Domains](https://s.ee/user/domains/) | Configure dominios personalizados |
| [Analytics](https://s.ee/user/analytics/) | Vea estadísticas de clics y escaneos |
| [Team](https://s.ee/user/team/) | Gestione a los miembros del equipo |
| [UTM Builder](https://s.ee/user/library/utm/) | Cree y gestione parámetros UTM |
| [Usage](https://s.ee/user/usage/) | Supervise el uso de su plan |
| [Ajustes](https://s.ee/user/settings/) | Configuración de la cuenta y facturación |

En dispositivos móviles, la barra lateral se contrae automáticamente.

### Menú de perfil

Se encuentra en la barra lateral izquierda. Úselo para:
- Acceder a la documentación
- [Gestionar claves API](https://s.ee/user/developers/)
- [Actualizar su plan](https://s.ee/user/pricing/)
- Cambiar entre cuentas
- Cerrar sesión
