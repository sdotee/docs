S.EE brauserilaiendus toob URL-ide lühendamise, teksti jagamise ja failide üleslaadimise otse brauserisse. Saadaval Chrome'i, Firefoxi ja Edge'i jaoks.

Avatud lähtekoodiga projekt asub [GitHubis](https://github.com/sdotee/browser-extensions).

## Funktsioonid

| Funktsioon | Kirjeldus |
|---------|-------------|
| URL-ide lühendamine | Lühenda ükskõik milline URL ühe klõpsuga, kasuta kohandatud lühikoode ja mitut domeeni |
| Teksti jagamine | Jaga koodilõike ja teksti koos süntaksi esiletõstmisega |
| Failide jagamine | Laadi faile partiidena üles koos edenemise jälgimisega |
| Ekspordivormingud | Kopeeri tulemused tavalingi, Markdowni, HTML-i või BBCode'ina |
| QR-koodi loomine | Loo QR-koode igale lühendatud URL-ile |
| QR-koodi eksport | Laadi QR-koode alla PNG-, SVG- või PDF-vormingus (512x512) |
| Paremklõpsumenüü | Lühenda leht või link, kuva QR, jaga valitud teksti, laadi pildid üles |
| Ajaloo haldamine | Vahekaartidest teadlik ajalugu URL-idele, tekstidele ja failidele koos hulgikustutamisega |
| Mustandite säilitamine | Sisestusväljad salvestatakse automaatselt seansside vahel |
| Automaatne kopeerimine | Kopeeri tulemused automaatselt lõikelauale |
| Tume ja hele teema | Automaatne teematunnetus koos käsitsi lülitiga |
| Töölauateavitused | Kontekstimenüü toimingute lõppedes kuvatakse teavitus |
| Brauseriülene | Töötab Chrome'is (MV3), Firefoxis (MV3) ja Edge'is |
