[Plugin WordPress của S.EE](https://wordpress.org/plugins/sdotee/) cho phép bạn rút gọn URL, chia sẻ văn bản và tải tệp lên mà không cần rời khỏi WordPress. Khi bạn publish một bài viết, một liên kết ngắn sẽ được tạo tự động — không cần thêm bước nào.

- [WordPress Plugin Directory](https://wordpress.org/plugins/sdotee/)
- [Mã nguồn trên GitHub](https://github.com/sdotee/see-wordpress)
- [GitHub Releases](https://github.com/sdotee/see-wordpress/releases)

## Yêu cầu

- PHP 8.2+
- WordPress 6.0+
- Tài khoản S.EE có [khóa API令牌](https://s.ee/user/developers/)

---

## Cài đặt

### Từ WordPress Plugin Directory

1. Trong trang quản trị WordPress, truy cập **Gói mở rộng** > **Cài mới**
2. Tìm **S.EE**
3. Nhấp vào **Cài đặt ngay**, sau đó **Kích hoạt**

### Từ GitHub

1. Tải file ZIP mới nhất từ [GitHub Releases](https://github.com/sdotee/see-wordpress/releases)
2. Truy cập **Gói mở rộng** > **Cài mới** > **Tải plugin lên**
3. Tải ZIP lên rồi nhấp **Cài đặt ngay**, sau đó **Kích hoạt**

---

## Thiết lập

### 1. Lấy khóa API令牌 của bạn

1. Đăng nhập vào tài khoản S.EE của bạn
2. Truy cập **[Công cụ > khóa API令牌](https://s.ee/user/developers/)**
3. Tạo một khóa API令牌 mới

<Callout type="warn">
Không chia sẻ khóa API令牌 công khai hoặc commit nó vào version control.
</Callout>

### 2. Cấu hình plugin

1. Truy cập **Cài đặt** > **S.EE**
2. Dán khóa API令牌 của bạn rồi nhấp **Xác minh**
3. Chọn tên miền mặc định cho URL ngắn, văn bản và tệp
4. Bật auto-generation và auto-upload nếu bạn muốn mọi thứ chạy tự động

<Callout type="info">
Bạn cũng có thể đặt khóa API令牌 trong `wp-config.php` để tránh lưu nó trong database.
</Callout>

## Cách sử dụng

### URL ngắn

Khi bật auto-generation, mỗi bài viết và trang sẽ có một liên kết ngắn ngay khi bạn nhấn Publish. Bạn cũng có thể tạo hoặc xem URL ngắn từ cột trong danh sách bài viết/trang ở trang quản trị.

### Chia sẻ văn bản

Mở bảng điều khiển **S.EE** rồi dán nội dung của bạn. Các định dạng được hỗ trợ:

- Văn bản thuần
- Markdown
- Đoạn mã

### Lưu trữ tệp

Tải tệp lên thông qua bảng điều khiển **S.EE** hoặc Media Library. Sau khi tải lên, bạn có thể lấy liên kết ở định dạng mong muốn: URL, HTML, Markdown hoặc BBCode.
