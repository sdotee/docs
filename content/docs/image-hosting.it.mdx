---

La [pagina di caricamento immagini](https://s.ee/user/images/) gestisce upload multipli. Puoi trascinare e rilasciare file o cartelle, incollare dagli appunti oppure fare clic per scegliere i file. Le immagini incollate vengono convertite automaticamente in WebP (o PNG se il browser non supporta WebP).

I file caricati vengono ordinati per nome, così i batch restano sempre nell'ordine corretto.

---

## Codici di condivisione

Ogni immagine riceve un link di condivisione. Puoi copiare i link uno alla volta oppure ottenere un codice di condivisione per l'intero batch.

Formati: URL diretto, Markdown, HTML, BBCode.

---

## Gestione dei file

La pagina immagini serve solo per l'upload. Per eliminare immagini o gestire i file, vai su [s.ee/user/files/](https://s.ee/user/files/).
