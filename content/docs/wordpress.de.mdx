Das [S.EE WordPress-Plugin](https://wordpress.org/plugins/sdotee/) ermöglicht es Ihnen, URLs zu kürzen, Text zu teilen und Dateien hochzuladen, ohne WordPress zu verlassen. Veröffentlichen Sie einen Beitrag, und ein Kurzlink wird automatisch erstellt – ohne zusätzliche Schritte.

- [WordPress Plugin Directory](https://wordpress.org/plugins/sdotee/)
- [Quellcode auf GitHub](https://github.com/sdotee/see-wordpress)
- [GitHub Releases](https://github.com/sdotee/see-wordpress/releases)

---

## Funktionen

| Funktion | Beschreibung |
|---------|-------------|
| Kurz-URLs | Generiert automatisch Kurzlinks, wenn Sie Beiträge und Seiten veröffentlichen oder aktualisieren |
| Textfreigabe | Teilen Sie Klartext, Markdown und Code-Snippets über einen Link |
| Dateihosting | Laden Sie Dateien aus der Mediathek hoch und kopieren Sie den Link als URL, HTML, Markdown oder BBCode |
| Automatisierung | Automatischer Upload neuer Medien und automatische Kürzung beim Veröffentlichen – einmal einrichten und vergessen |
| Admin-Integration | Kurz-URLs direkt aus der Listenansicht von Beiträgen/Seiten anzeigen und verwalten |
| Dashboard-Panel | Eigenes S.EE-Panel für Text- und Dateifreigabe mit vollständiger Nutzungshistorie |
| 13 Sprachen | Englisch, Chinesisch (vereinfacht/traditionell), Japanisch, Koreanisch, Indonesisch, Vietnamesisch, Deutsch, Französisch, Spanisch, Portugiesisch (Brasilien/Portugal), Russisch |

## Voraussetzungen

- PHP 8.2+
- WordPress 6.0+
- Ein S.EE-Konto mit [API-Schlüssel令牌](https://s.ee/user/developers/)

---

## Installation

### Aus dem WordPress Plugin Directory

1. Gehen Sie in Ihrem WordPress-Admin zu **Plugins** > **Neu hinzufügen**
2. Suchen Sie nach **S.EE**
3. Klicken Sie auf **Jetzt installieren**, dann auf **Aktivieren**

### Von GitHub

1. Laden Sie die neueste ZIP-Datei aus den [GitHub Releases](https://github.com/sdotee/see-wordpress/releases) herunter
2. Gehen Sie zu **Plugins** > **Neu hinzufügen** > **Plugin hochladen**
3. Laden Sie die ZIP-Datei hoch und klicken Sie auf **Jetzt installieren**, dann auf **Aktivieren**

---

## Einrichtung

### 1. API-Schlüssel令牌 abrufen

1. Melden Sie sich bei Ihrem S.EE-Konto an
2. Gehen Sie zu **[Werkzeuge > API-Token](https://s.ee/user/developers/)**
3. Generieren Sie ein neues API-Schlüssel令牌

<Callout type="warn">
Teilen Sie Ihr API-Schlüssel令牌 nicht öffentlich und committen Sie es nicht in die Versionskontrolle.
</Callout>

### 2. Das Plugin konfigurieren

1. Gehen Sie zu **Einstellungen** > **S.EE**
2. Fügen Sie Ihr API-Token ein und klicken Sie auf **Verifizieren**
3. Wählen Sie Ihre Standard-Domains für Kurz-URLs, Text und Dateien
4. Aktivieren Sie automatische Generierung und automatische Uploads, wenn Sie einen möglichst automatischen Workflow möchten

<Callout type="info">
Sie können Ihr API-Schlüssel令牌 auch in `wp-config.php` setzen, um es aus der Datenbank herauszuhalten.
</Callout>

## Verwendung

### Kurz-URLs

Wenn die automatische Generierung aktiviert ist, erhält jeder Beitrag und jede Seite einen Kurzlink, sobald Sie auf Veröffentlichen klicken. Sie können Kurz-URLs auch über die Spalte in der Beitrags-/Seitenliste im Admin-Bereich erstellen oder anzeigen.

### Textfreigabe

Öffnen Sie das **S.EE**-Dashboard-Panel und fügen Sie Ihren Inhalt ein. Unterstützte Formate:

- Klartext
- Markdown
- Code-Snippets

### Dateihosting

Laden Sie Dateien über das **S.EE**-Dashboard-Panel oder die Mediathek hoch. Nach dem Upload können Sie den Link im gewünschten Format übernehmen: URL, HTML, Markdown oder BBCode.
