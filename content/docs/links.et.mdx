## Loo link

1. Ava [https://s.ee/user/links/](https://s.ee/user/links/)
2. Klõpsa **Create Link**
3. Sisesta siht-URL
4. Seadista valikud vastavalt vajadusele
5. Klõpsa **Create**

---

## Lingi valikud

Lingi loomisel saab seadistada järgmisi valikuid:

| Valik | Kirjeldus |
|--------|-------------|
| Kohandatud lühikood | Määra lingi kohandatud tagumine osa, näiteks `s.ee/link` |
| Domeen | Vali ametlike või enda kohandatud domeenide seast |
| Parool | Nõua sihtlehele pääsemiseks parooli |
| Aegumine | Määra kuupäev, millal link enam ei tööta |
| Sihtimine | Suuna kasutajaid ümber asukoha, seadme või keele järgi |
| Sildid | Korralda linke siltidega |
| UTM-parameetrid | Lisa kampaania jälgimise parameetrid |
| Parameetrite edastamine | Edasta URL-i parameetrid sihtkohta |

---

## Analüütika

Klõpsa mis tahes lingil, et näha selle analüütikat:

- Klõpsude koguarv
- Klõpsud asukoha järgi
- Klõpsud seadme ja brauseri järgi
- Viiteallikad

---

## Linkide haldamine

Linkide juhtpaneelilt saad:

- **Edit** - uuendada siht-URL-i või seadeid
- **Delete** - eemaldada lingi jäädavalt
- **Copy** - kopeerida lühilink lõikelauale
