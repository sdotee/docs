[ShareX](https://getsharex.com/) est un outil open source de capture d’écran et de partage de fichiers pour Windows. S.EE fournit des configurations d’téléverseurs personnalisés pour ShareX, ce qui vous permet de téléverser directement des captures d’écran, des fichiers, du texte et du code vers S.EE et d’obtenir ensuite un lien partageable.

Voir aussi : [Documentation des téléverseurs personnalisés ShareX](https://getsharex.com/docs/custom-téléverseur)

---

## Uploaders personnalisés

S.EE propose des configurations pour ces types de téléversement :

| Uploader | Fonction |
|----------|-------------|
| Raccourcisseur d’URL | Raccourcit les URL via S.EE |
| Téléverseur de texte | Téléverse des extraits de texte |
| Téléverseur de code | Téléverse du code avec coloration syntaxique |
| Téléverseur Markdown | Téléverse du Markdown avec aperçu rendu |
| Téléverseur d’images et de fichiers | Téléverse des images et des fichiers |

### Importeration

1. Ouvrez **ShareX**
2. Accédez à **Cibles** > **Paramètres du téléverseur personnalisé**
3. Cliquez sur **Importer**, puis :
   - Sélectionnez le fichier `.sxcu` téléchargé
   - Ou collez directement l’URL de téléchargement

## Configurer votre clé API

1. Obtenez votre clé API depuis **[Outils > Token API](https://s.ee/user/developers/)**
2. Dans ShareX, ouvrez **Paramètres du téléverseur personnalisé** et sélectionnez le téléverseur S.EE importé
3. Dans la section **En-têtes**, définissez la valeur `Authorization` avec votre clé API

<Callout type="warn">
Ne partagez pas votre clé API publiquement et ne la validez pas dans votre système de gestion de versions.
</Callout>

## Utilisation

### Choisir vos téléverseurs

Dans ShareX, accédez à **Cibles** et assignez les téléverseurs S.EE :

- Raccourcisseur d’URL → S.EE Raccourcisseur d’URL
- Téléverseur de texte → Téléverseur de texte S.EE, Téléverseur de code ou Téléverseur Markdown
- Téléverseur d’images → S.EE File Uploader
- Téléverseur de fichiers → S.EE File Uploader

### Téléverser

Utilisez les raccourcis clavier ou le menu contextuel de ShareX pour raccourcir une URL, téléverser une capture d’écran, partager un extrait de texte, etc. ShareX téléverse le contenu vers S.EE et copie le lien dans votre presse-papiers.

---

## Résolution des problèmes

| Problème | Correctif |
|-------|-----|
| Erreur 401 | Vérifiez que votre clé API est correctement définie dans l’en-tête Authorization |
| Erreur 403 | Assurez-vous que votre clé API dispose des permissions nécessaires |
| Fichier refusé | Le type de fichier n’est peut-être pas pris en charge. Consultez [Partage de fichiers](/docs/fr/file-sharing) pour les types autorisés |

Vous pouvez cliquer sur **Tester** dans **Paramètres du téléverseur personnalisé** pour vérifier que tout fonctionne.
