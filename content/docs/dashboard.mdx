## Dashboard Overview

Access your [Dashboard](https://s.ee/user/dashboard/) after logging in. The S.EE dashboard has two main navigation areas:

### Left Sidebar

Access your main features:

| Feature | Description |
|---------|-------------|
| [Links](https://s.ee/user/links/) | Manage your short links |
| [QR Codes](https://s.ee/user/qr/) | Create and manage QR codes |
| [Text Sharing](https://s.ee/user/text/) | Share text, code snippets, and markdown |
| [Domains](https://s.ee/user/domains/) | Configure custom domains |
| [Analytics](https://s.ee/user/analytics/) | View click and scan statistics |
| [Team](https://s.ee/user/team/) | Manage team members |
| [UTM Builder](https://s.ee/user/library/utm/) | Create and manage UTM parameters |
| [Usage](https://s.ee/user/usage/) | Monitor your plan usage |
| [Settings](https://s.ee/user/settings/) | Account and billing settings |

On mobile devices, the sidebar collapses automatically.

### Profile Menu

Located in the left sidebar. Use it to:
- Access documentation
- [Manage API keys](https://s.ee/user/developers/)
- [Upgrade your plan](https://s.ee/user/pricing/)
- Switch between accounts
- Log out
