## Creare un link

1. Vai su [https://s.ee/user/links/](https://s.ee/user/links/)
2. Fai clic su **Create Link**
3. Inserisci l'URL di destinazione
4. Configura le opzioni, se necessario
5. Fai clic su **Create**

---

## Opzioni del link

Quando crei un link, puoi configurare:

| Opzione | Descrizione |
|--------|-------------|
| Custom slug | Imposta una parte finale personalizzata per il tuo link, ad esempio `s.ee/link` |
| Domain | Scegli tra domini ufficiali o i tuoi domini personalizzati |
| Password | Richiedi una password per accedere alla destinazione |
| Expiration | Imposta una data in cui il link smette di funzionare |
| Targeting | Reindirizza gli utenti in base a posizione, dispositivo o lingua |
| Tags | Organizza i link con i tag |
| UTM parameters | Aggiungi parametri di tracciamento per le campagne |
| Parameter passing | Inoltra i parametri dell'URL alla destinazione |

---

## Analytics

Fai clic su qualsiasi link per vedere le statistiche:
- Clic totali
- Clic per posizione
- Clic per dispositivo e browser
- Origini dei referrer

---

## Gestire i link

Dal pannello dei link puoi:
- **Edit** - Aggiornare l'URL di destinazione o le impostazioni
- **Delete** - Rimuovere il link in modo permanente
- **Copy** - Copiare il link breve negli appunti
