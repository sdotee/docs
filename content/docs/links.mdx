## Create a Link

1. Go to [https://s.ee/user/links/](https://s.ee/user/links/)
2. Click **Create Link**
3. Enter your destination URL
4. Configure options (optional)
5. Click **Create**

---

## Link Options

When creating a link, you can configure:

| Option | Description |
|--------|-------------|
| Custom slug | Set a custom back-half for your link (e.g., `s.ee/link`) |
| Domain | Choose from official domains or your custom domains |
| Password | Require a password to access the destination |
| Expiration | Set a date when the link stops working |
| Targeting | Redirect users based on location, device, or language |
| Tags | Organize links with tags |
| UTM parameters | Add campaign tracking parameters |
| Parameter passing | Forward URL parameters to the destination |

---

## Analytics

Click any link to view its analytics:
- Total clicks
- Clicks by location
- Clicks by device and browser
- Referrer sources

---

## Manage Links

From the links dashboard, you can:
- **Edit** - Update destination URL or settings
- **Delete** - Remove the link permanently
- **Copy** - Copy the short link to clipboard
