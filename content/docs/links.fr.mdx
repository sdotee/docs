## Créer un lien

1. Accédez à [https://s.ee/user/links/](https://s.ee/user/links/)
2. Cliquez sur **Créer un lien**
3. Saisissez votre URL de destination
4. Configurez les options (facultatif)
5. Cliquez sur **Créer**

---

## Options du lien

Lors de la création d’un lien, vous pouvez configurer :

| Option | Description |
|--------|-------------|
| Slug personnalisé | Définissez une partie finale personnalisée pour votre lien (par exemple `s.ee/link`) |
| Domaine | Choisissez parmi les domaines officiels ou vos domaines personnalisés |
| Mot de passe | Exigez un mot de passe pour accéder à la destination |
| Expiration | Définissez une date à laquelle le lien cessera de fonctionner |
| Ciblage | Redirigez les utilisateurs selon le lieu, l’appareil ou la langue |
| Tags | Organisez les liens avec des tags |
| Paramètres UTM | Ajoutez des paramètres de suivi de campagne |
| Transmission de paramètres | Transmettez les paramètres d’URL à la destination |

---

## Statistiques

Cliquez sur n’importe quel lien pour consulter ses statistiques :
- Nombre total de clics
- Clics par lieu
- Clics par appareil et navigateur
- Sources de provenance

---

## Gérer les liens

Depuis le tableau de bord des liens, vous pouvez :
- **Modifier** - Mettre à jour l’URL de destination ou les paramètres
- **Supprimer** - Supprimer définitivement le lien
- **Copier** - Copier le lien court dans le presse-papiers
