## CLI-Tools

| Name | Description | Link |
|------|-------------|------|
| **see-cli** | S.EE über die Kommandozeile. Kürzen Sie URLs, teilen Sie Text und laden Sie Dateien hoch. | [sdotee/cli](https://github.com/sdotee/cli) |

---

## Plugins

| Name | Description | Link |
|------|-------------|------|
| **S.EE for WordPress** | URL-Kürzung, Textfreigabe und Dateihosting für WordPress. Erstellt Kurz-URLs automatisch beim Veröffentlichen, lädt Medien zu S.EE hoch und verfolgt alles im Dashboard. | [WordPress.org](https://wordpress.org/plugins/sdotee/) · [GitHub](https://github.com/sdotee/see-wordpress) · [Docs](/docs/de/wordpress) |
| **S.EE for Raycast** | URL-Kürzung, Textfreigabe und Datei-Uploads direkt aus Raycast. Erkennt Zwischenablage-Inhalte automatisch und unterstützt benutzerdefinierte Slugs, Passwörter, Ablaufdaten und durchsuchbare Historie. | [Raycast Store](https://www.raycast.com/missuo/sdotee) · [GitHub](https://github.com/missuo/raycast-sdotee) |
| **Typecho-Plugin-PicUp** | Ein Plugin für Typecho zum Hochladen von Bildern und Anhängen mit mehreren Backends. Unterstützt auch Bild-Uploads zu S.EE und erleichtert das Hochladen und Verwalten von Medien in Typecho. | [GitHub](https://github.com/lhl77/Typecho-Plugin-PicUp) · [Homepage](https://blog.lhl.one/artical/1026.html) |
