[S.EE WordPress eklentisi](https://wordpress.org/plugins/sdotee/), WordPress'ten çıkmadan URL kısaltmanıza, metin paylaşmanıza ve dosya yüklemenize olanak tanır. Bir yazı yayımladığınızda kısa bağlantı otomatik olarak oluşturulur; ek adım gerekmez.

- [WordPress Plugin Directory](https://wordpress.org/plugins/sdotee/)
- [GitHub üzerindeki kaynak kod](https://github.com/sdotee/see-wordpress)
- [GitHub sürümleri](https://github.com/sdotee/see-wordpress/releases)

## Gereksinimler

- PHP 8.2+
- WordPress 6.0+
- [API Token](https://s.ee/user/developers/) içeren bir S.EE hesabı

---

## Kurulum

### WordPress eklenti dizininden

1. WordPress yönetim panelinde **Eklentiler** > **Yeni Ekle** bölümüne gidin
2. **S.EE** için arama yapın
3. **Şimdi Kur**, ardından **Etkinleştir** düğmesine tıklayın

### GitHub üzerinden

1. En son ZIP dosyasını [GitHub Releases](https://github.com/sdotee/see-wordpress/releases) sayfasından indirin
2. **Eklentiler** > **Yeni Ekle** > **Eklenti Yükle** yoluna gidin
3. ZIP dosyasını yükleyin, ardından **Şimdi Kur** ve **Etkinleştir** düğmelerine tıklayın

---

## Kurulum ayarları

### 1. API Token alın

1. S.EE hesabınızda oturum açın
2. **[Tools > API Token](https://s.ee/user/developers/)** sayfasına gidin
3. Yeni bir API Token oluşturun

<Callout type="warn">
API Token'ınızı herkese açık olarak paylaşmayın ve sürüm kontrolüne eklemeyin.
</Callout>

### 2. Eklentiyi yapılandırın

1. **Ayarlar** > **S.EE** yoluna gidin
2. API Token'ınızı yapıştırın ve **Doğrula** düğmesine tıklayın
3. Kısa URL'ler, metin ve dosyalar için varsayılan alan adlarını seçin
4. Tam otomatik kullanım istiyorsanız otomatik oluşturma ve otomatik yükleme seçeneklerini açın

<Callout type="info">
API Token'ı veritabanı dışında tutmak için `wp-config.php` içinde de tanımlayabilirsiniz.
</Callout>

## Kullanım

### Kısa URL'ler

Otomatik oluşturma açıksa her yazı ve sayfa, **Publish** düğmesine bastığınız anda kısa bağlantı alır. Yönetim panelindeki yazı/sayfa listesi sütunundan da kısa URL oluşturabilir veya görüntüleyebilirsiniz.

### Metin paylaşımı

**S.EE** kontrol paneli bölümünü açın ve içeriğinizi yapıştırın. Desteklenen biçimler:

- Düz metin
- Markdown
- Kod parçacıkları

### Dosya barındırma

Dosyaları **S.EE** panelinden veya Medya Kitaplığı'ndan yükleyin. Yükleme tamamlandıktan sonra bağlantıyı istediğiniz biçimde alın: URL, HTML, Markdown veya BBCode.
