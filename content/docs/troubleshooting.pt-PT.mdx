## Problemas de conta e início de sessão

| Mensagem de erro | Causa | Solução |
|---------------|-------|----------|
| "Invalid email or password. Please check your credentials and try again." | Email ou palavra-passe incorretos | Verifique cuidadosamente o seu email e palavra-passe. Tenha em conta que as palavras-passe distinguem maiúsculas de minúsculas. |
| "We couldn't find an account with this email." | Email não registado | Verifique o seu endereço de email ou crie uma nova conta. |
| "Your account has been suspended. Please contact support for assistance." | Conta suspensa | Contacte a nossa equipa de suporte em [hi@s.ee](mailto:hi@s.ee). |
| "This login link has expired or is invalid. Please request a new one." | Link de início de sessão expirado | Peça um novo link de início de sessão na página de login. |
| "This verification code has expired or is invalid. Please request a new one." | Código de verificação expirado ou inválido | Peça um novo código de verificação. |
| "This email is already registered." | Email já utilizado | Use outro endereço de email ou tente iniciar sessão com esse email. |
| "The current password you entered is incorrect." | Palavra-passe atual incorreta ao alterá-la | Verifique a sua palavra-passe atual e tente novamente. |

**Nota:** só o seu endereço de email principal funciona para início de sessão. Outros emails verificados na conta não funcionam para autenticação. Os emails de reposição de palavra-passe também são enviados apenas para o email principal.

## Problemas com nome de utilizador

| Mensagem de erro | Causa | Solução |
|---------------|-------|----------|
| "This username is already taken. Please choose a different one." | Nome de utilizador já em uso | Escolha um nome de utilizador diferente. |
| "Username can only be changed once." | Limite de alteração atingido | Infelizmente, o nome de utilizador só pode ser alterado uma vez por conta. Esta alteração é permanente. |

## Problemas com links curtos

| Mensagem de erro | Causa | Solução |
|---------------|-------|----------|
| "This slug is already taken. Please choose a different one." | Slug personalizado já em uso | Tente outro slug personalizado para o seu link curto. |
| "This slug cannot be used. Please choose a different one." | O slug é reservado pelo sistema | Este slug está reservado. Escolha outro diferente. |
| "This short link does not exist or is no longer available." | O link curto não existe ou foi eliminado | Verifique se o URL do link está correto. O link pode ter sido eliminado. |
| "The domain linked to this short URL is unavailable or no longer exists." | Domínio indisponível | Contacte o suporte ou tente usar um domínio diferente. |
| "Please enter a valid destination URL starting with http:// or https://" | Formato de URL inválido | Certifique-se de que o URL de destino começa por `http://` ou `https://`. |
| "This destination URL is not allowed. Please use a different link." | URL de destino bloqueado | Este URL está na nossa lista de bloqueio. Use outro URL de destino. |
| "Incorrect password. Please try again." | Palavra-passe incorreta para link protegido | Introduza a palavra-passe correta para aceder a este link. |
| "URL has already been shortened." | O URL já foi encurtado | Este URL já tem um link curto associado. |

## Problemas de subscrição e plano

| Mensagem de erro | Causa | Solução |
|---------------|-------|----------|
| "You already have an active subscription plan." | Já subscreveu um plano | Se quiser mudar de plano, cancele primeiro a sua subscrição atual. |
| "You don't have an active subscription to cancel." | Sem subscrição ativa | Atualmente não tem nenhuma subscrição ativa para cancelar. |
| "This feature requires a premium subscription. Please upgrade your plan to access it." | A funcionalidade exige um nível superior | Atualize a sua subscrição para aceder a esta funcionalidade. |
| "Unable to find your subscription plan. Please contact support if this issue persists." | Plano não encontrado | Contacte a nossa equipa de suporte em [hi@s.ee](mailto:hi@s.ee). |
| "The selected pricing option is no longer available." | Opção de preço expirada | Atualize a página e selecione uma opção de preço atual. |

## Problemas de carregamento de ficheiros

| Mensagem de erro | Causa | Solução |
|---------------|-------|----------|
| "File size exceeds limit" | O ficheiro é demasiado grande | Comprima o ficheiro ou atualize o seu plano para permitir carregamentos maiores. |
| "Storage limit exceeded" | Quota de armazenamento atingida | Elimine alguns ficheiros para libertar espaço, ou atualize o seu plano para obter mais armazenamento. |
| "Unsupported file type" / "File type not allowed" | Formato de ficheiro não suportado | Use um formato de ficheiro suportado. |
| "Maximum concurrent uploads exceeded" | Demasiados carregamentos em simultâneo | Aguarde que os carregamentos atuais terminem antes de iniciar novos. |
| "This file format is not supported. Allowed formats: JPG, JPEG, PNG, GIF, WEBP, AVIF." | Formato de avatar não suportado | Carregue o seu avatar num dos formatos suportados: JPG, JPEG, PNG, GIF, WEBP ou AVIF. |

## Problemas de equipa

| Mensagem de erro | Causa | Solução |
|---------------|-------|----------|
| "This feature is not available on your current plan." | Funcionalidade de equipa não incluída | Atualize para um plano que inclua funcionalidades de equipa. |
| "You have already created a team." | A equipa já existe | Cada utilizador só pode criar uma equipa. Pode gerir a equipa existente a partir do painel. |
| "You don't have permission to manage this team." | Permissões insuficientes | Contacte o proprietário da sua equipa. |
| "The invitation has already been processed." | Convite já tratado | Este convite já foi aceite ou recusado. |
| "Please wait at least 24 hours before resending." | Reenvio demasiado cedo | Aguarde 24 horas antes de enviar outro convite a este utilizador. |

## Problemas ao eliminar a conta

| Mensagem de erro | Causa | Solução |
|---------------|-------|----------|
| "Your team has other members, please delete the team first." | A equipa ainda tem membros | Remova todos os membros da equipa ou elimine a equipa antes de eliminar a conta. |
| "Too many deletion requests, please try again later." | Demasiados pedidos | Aguarde algum tempo antes de tentar novamente. |
| "Invalid or expired verification code." | Código de verificação inválido ou expirado | Peça um novo código de verificação. |

## Problemas de gestão de email

| Mensagem de erro | Causa | Solução |
|---------------|-------|----------|
| "Email already exists." | Email já em uso | Use um endereço de email diferente. |
| "Main email cannot be deleted." | Não é possível eliminar o email principal | Defina outro email verificado como principal antes de eliminar este. |
| "You need to close two factor verification first." | Primeiro é preciso desativar 2FA | Desative a autenticação de dois fatores antes de alterar o seu email principal. |
| "This email has not been verified." | Email não verificado | Verifique primeiro este endereço de email. |
