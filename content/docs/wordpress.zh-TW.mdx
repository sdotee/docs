[S.EE WordPress 外掛](https://wordpress.org/plugins/sdotee/) 讓您無需離開 WordPress，就能建立短網址、分享文字和上傳檔案。釋出文章後，系統會自動建立短網址，無需額外步驟。

- [WordPress 外掛目錄](https://wordpress.org/plugins/sdotee/)
- [GitHub 原始碼](https://github.com/sdotee/see-wordpress)
- [GitHub Releases](https://github.com/sdotee/see-wordpress/releases)

---

## 功能

| 功能 | 說明 |
|---------|-------------|
| 短網址 | 釋出或更新文章、頁面時自動生成短網址 |
| 文字分享 | 透過連結分享純文字、Markdown 和程式碼片段 |
| 檔案託管 | 從媒體庫上傳檔案，並將連結複製為 URL、HTML、Markdown 或 BBCode |
| 自動化 | 自動上傳新媒體，並在釋出時自動生成短網址，設定一次即可長期使用 |
| 後臺整合 | 直接在文章 / 頁面列表中檢視和管理短網址 |
| S.EE 面板 | 專用的 S.EE 面板，用於文字和檔案分享，並提供完整使用歷史 |
| 13 種語言 | 英語、中文（簡體 / 繁體）、日語、韓語、印尼語、越南語、德語、法語、西班牙語、葡萄牙語（巴西 / 葡萄牙）、俄語 |

---

## 要求

- PHP 8.2+
- WordPress 6.0+
- 帶有 [API Token](https://s.ee/user/developers/) 的 S.EE 帳號

---

## 安裝

### 從 WordPress 外掛目錄安裝

1. 在 WordPress 後臺前往 **Plugins** > **Add New**
2. 搜尋 **S.EE**
3. 點擊 **Install Now**，然後點擊 **Activate**

### 從 GitHub 安裝

1. 從 [GitHub Releases](https://github.com/sdotee/see-wordpress/releases) 下載最新 ZIP
2. 前往 **Plugins** > **Add New** > **Upload Plugin**
3. 上傳 ZIP，點擊 **Install Now**，然後點擊 **Activate**

---

## 設定

### 1. 取得您的 API Token

1. 登入您的 S.EE 帳號
2. 前往 **[Tools > API Token](https://s.ee/user/developers/)**
3. 生成新的 API Token

<Callout type="warn">
不要公開分享您的 API Token，也不要將其提交到版本控制系統中。
</Callout>

### 2. 設定外掛

1. 前往 **Settings** > **S.EE**
2. 貼上您的 API Token 並點擊 **Verify**
3. 為短網址、文字和檔案選擇預設網域
4. 如果您希望完全自動化，可以啟用自動生成和自動上傳

<Callout type="info">
您也可以在 `wp-config.php` 中設定 API Token，以避免將其儲存在資料庫中。
</Callout>

## 使用方法

### 短網址

啟用自動生成功能後，每篇文章和頁面在釋出時都會立即獲得一個短網址。您也可以在後臺文章 / 頁面列表列中建立或檢視短網址。

### 文字分享

開啟 **S.EE** 面板並貼上內容。支援的格式包括：

- 純文字
- Markdown
- 程式碼片段

### 檔案託管

透過 **S.EE** 面板或媒體庫上傳檔案。上傳完成後，您可以按需要取得 URL、HTML、Markdown 或 BBCode 格式的連結。
