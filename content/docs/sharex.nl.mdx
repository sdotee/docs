[ShareX](https://getsharex.com/) is een open-sourcetool voor schermafbeeldingen en bestandsdeling op Windows. S.EE levert aangepaste uploaderconfiguraties voor ShareX, zodat u screenshots, bestanden, tekst en code rechtstreeks naar S.EE kunt uploaden en een deelbare link terugkrijgt.

Zie ook: [ShareX-documentatie voor custom uploaders](https://getsharex.com/docs/custom-uploader)

---

## Aangepaste uploaders

S.EE heeft configuraties voor de volgende uploadtypen:

| Uploader | Wat deze doet |
|----------|-------------|
| URL Shortener | Verkort URL's via S.EE |
| Text Uploader | Uploadt tekstfragmenten |
| Code Uploader | Uploadt code met syntax highlighting |
| Markdown Uploader | Uploadt Markdown met gerenderde preview |
| Image & File Uploader | Uploadt afbeeldingen en bestanden |

### Importeren

1. Open **ShareX**
2. Ga naar **Destinations** > **Custom uploader settings**
3. Klik op **Import**, en doe daarna een van de volgende dingen:
   - Selecteer het gedownloade `.sxcu`-bestand
   - Plak de download-URL direct in

## Uw API-sleutel instellen

1. Haal uw API-sleutel op via **[Tools > API Token](https://s.ee/user/developers/)**
2. Open in ShareX **Custom uploader settings** en selecteer de eerder geimporteerde S.EE-uploader
3. Stel in de sectie **Headers** de waarde van `Authorization` in op uw API-sleutel

<Callout type="warn">
Deel uw API-sleutel niet openbaar en commit die niet naar versiebeheer.
</Callout>

## Gebruik

### Uw uploaders kiezen

Ga in ShareX naar **Destinations** en wijs de S.EE-uploaders toe:

- URL shortener -> S.EE URL Shortener
- Text uploader -> S.EE Text Uploader, Code Uploader of Markdown Uploader
- Image uploader -> S.EE File Uploader
- File uploader -> S.EE File Uploader

### Uploaden

Gebruik de sneltoetsen van ShareX of het rechtsklikmenu om een URL te verkorten, een screenshot te uploaden, een tekstfragment te delen enzovoort. ShareX uploadt naar S.EE en kopieert de link naar uw klembord.

## Probleemoplossing

| Probleem | Oplossing |
|-------|-----|
| 401-fout | Controleer of uw API-sleutel correct in de Authorization-header staat |
| 403-fout | Controleer of uw API-sleutel de juiste rechten heeft |
| Bestand geweigerd | Het bestandstype wordt mogelijk niet ondersteund. Zie [Bestanden delen](/docs/nl/file-sharing) voor toegestane typen |

U kunt op **Test** klikken in **Custom uploader settings** om te controleren of alles werkt.
