[ShareX](https://getsharex.com/) is een open-sourcetool voor schermafbeeldingen en bestandsdeling op Windows. S.EE levert aangepaste uploadconfiguraties voor ShareX, zodat u screenshots, bestanden, tekst en code rechtstreeks naar S.EE kunt uploaden en een deelbare link terugkrijgt.

Zie ook: [ShareX-documentatie voor aangepaste uploadmodules](https://getsharex.com/docs/custom-uploadmodule)

---

## Aangepaste uploadmodules

S.EE heeft configuraties voor de volgende uploadtypen:

| Uploader | Wat deze doet |
|----------|-------------|
| URL-verkorter | Verkort URL's via S.EE |
| Tekstupload | Uploadt tekstfragmenten |
| Code-upload | Uploadt code met syntax highlighting |
| Markdown-upload | Uploadt Markdown met gerenderde preview |
| Afbeeldings- en bestandsupload | Uploadt afbeeldingen en bestanden |

### Importereneren

1. Open **ShareX**
2. Ga naar **Bestemmingen** > **Instellingen voor aangepaste uploadmodule**
3. Klik op **Importeren**, en doe daarna een van de volgende dingen:
   - Selecteer het gedownloade `.sxcu`-bestand
   - Plak de download-URL direct in

## Uw API-sleutel instellen

1. Haal uw API-sleutel op via **[Hulpmiddelen > API-token](https://s.ee/user/developers/)**
2. Open in ShareX **Instellingen voor aangepaste uploadmodule** en selecteer de eerder geimporteerde S.EE-uploadmodule
3. Stel in de sectie **Kopteksten** de waarde van `Authorization` in op uw API-sleutel

<Callout type="warn">
Deel uw API-sleutel niet openbaar en commit die niet naar versiebeheer.
</Callout>

## Gebruik

### Uw uploadmethoden kiezen

Ga in ShareX naar **Bestemmingen** en wijs de S.EE-uploadmethoden toe:

- URL-verkorter -> S.EE URL Shortener
- Tekstupload -> S.EE Text Uploader, Code Uploader of Markdown Uploader
- Afbeeldingsupload -> S.EE File Uploader
- Bestandsupload -> S.EE File Uploader

### Uploaden

Gebruik de sneltoetsen van ShareX of het rechtsklikmenu om een URL te verkorten, een screenshot te uploaden, een tekstfragment te delen enzovoort. ShareX uploadt naar S.EE en kopieert de link naar uw klembord.

## Probleemoplossing

| Probleem | Oplossing |
|-------|-----|
| 401-fout | Controleer of uw API-sleutel correct in de Authorization-header staat |
| 403-fout | Controleer of uw API-sleutel de juiste rechten heeft |
| Bestand geweigerd | Het bestandstype wordt mogelijk niet ondersteund. Zie [Bestanden delen](/docs/nl/file-sharing) voor toegestane typen |

U kunt op **Testen** klikken in **Instellingen voor aangepaste uploadmodule** om te controleren of alles werkt.
