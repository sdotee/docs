## Przegląd

S.EE Text Sharing pozwala udostępniać zwykły tekst, fragmenty kodu i dokumenty Markdown za pomocą krótkiego linku. To wygodne rozwiązanie do dzielenia się plikami konfiguracyjnymi, przykładami kodu, notatkami i dokumentacją.

---

## Utwórz udostępnienie tekstu

1. Zaloguj się na [https://s.ee/user/dashboard](https://s.ee/user/dashboard)
2. Kliknij **Sharing > Text** w lewym pasku bocznym lub przejdź bezpośrednio do [https://s.ee/user/text/](https://s.ee/user/text/)
3. Uzupełnij szczegóły:
   - **Title** (opcjonalnie) - używany jako nazwa pliku podczas pobierania
   - **Short Link** - wybierz preferowaną domenę
   - **Text Type** - wybierz Plain Text, Code lub Markdown
4. Wprowadź treść w polu **Edytuj**
5. Skonfiguruj opcjonalne ustawienia opisane poniżej
6. Kliknij **Utwórz tekst**

---

## Typy treści

| Typ | Opis |
|------|-------------|
| Plain Text | Prosty niesformatowany tekst |
| Code | Fragment kodu z podświetlaniem składni |
| Markdown | Wyrenderowany Markdown z formatowaniem |

---

## Ustawienia opcjonalne

| Opcja | Opis |
|--------|-------------|
| Password | Chroń tekst hasłem |
| Tags | Porządkuj udostępnione teksty za pomocą tagów |
| Expiration | Ustaw moment wygaśnięcia udostępnienia |
| Disable search engine indexing | Zablokuj indeksowanie przez wyszukiwarki |

---

## Wyświetlanie udostępnionego tekstu

Po utworzeniu otrzymasz krótki link, np. `ba.sh/demo`.

### Tryby widoku

| URL | Opis |
|-----|-------------|
| `ba.sh/demo` | Wyświetla sformatowaną stronę tekstu z podświetlaniem składni |
| `ba.sh/demo/raw` | Wyświetla bezpośrednio surowy zwykły tekst |

Endpoint `/raw` jest przydatny do:

- Pobierania skryptów bezpośrednio przez `curl` lub `wget`
- Osadzania treści w innych aplikacjach
- Podglądu bez dodatkowego formatowania
