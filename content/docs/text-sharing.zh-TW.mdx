## 概覽

S.EE 文字分享允許您透過簡短、可分享的連結來分享純文字、程式碼片段和 Markdown 檔案。非常適合分享設定檔案、程式碼範例、筆記和檔案。

---

## 建立文字分享

1. 登入 [https://s.ee/user/dashboard](https://s.ee/user/dashboard)
2. 在左側邊欄點擊 **Sharing > Text**，或直接前往 [https://s.ee/user/text/](https://s.ee/user/text/)
3. 填寫以下資訊：
   - **Title**（可選）- 下載時會作為檔名使用
   - **Short Link** - 選擇您偏好的網域
   - **Text Type** - 從 Plain Text、Code 或 Markdown 中選擇
4. 在 **編輯** 輸入框中填入內容
5. 設定可選設定（見下文）
6. 點擊 **建立文字**

---

## 文字型別

| 型別 | 說明 |
|------|-------------|
| 純文字 | 簡單的無格式文字 |
| 程式碼 | 帶語法高亮的程式碼片段 |
| Markdown | 渲染後的 Markdown 檔案 |

---

## 可選設定

| 選項 | 說明 |
|--------|-------------|
| 密碼 | 為文字設定密碼保護 |
| 標籤 | 使用標籤整理分享的文字 |
| 過期時間 | 設定文字分享的過期時間 |
| 禁用搜索引擎索引 | 防止搜尋引擎收錄您分享的文字 |

---

## 檢視分享的文字

建立完成後，您會得到類似 `ba.sh/demo` 的短網址。

### 檢視模式

| URL | 說明 |
|-----|-------------|
| `ba.sh/demo` | 檢視帶語法高亮的格式化文字頁面 |
| `ba.sh/demo/raw` | 直接檢視原始純文字 |

`/raw` 介面適用於：

- 使用 `curl` 或 `wget` 直接下載指令碼
- 嵌入到其他應用中
- 在無格式模式下檢視內容

## 支援的網域

所有套餐都可以在以下網域上使用文字分享：

| 網域 | 適用場景 |
|--------|----------|
| [ba.sh](https://ba.sh) | Bash 和 Shell 指令碼 |
| [z.sh](https://z.sh) | Shell 指令碼 |
| [fs.to](https://fs.to) | 通用檔案分享 |
| [files.to](https://files.to) | 通用檔案分享 |
| [fileshare.to](https://fileshare.to) | 通用檔案分享 |
| [filesharing.to](https://filesharing.to) | 通用檔案分享 |

如果您是 Premium 或 Enterprise 客戶，請參考我們的[官方網域](https://s.ee/docs/official-domains/#premium--enterprise)。
