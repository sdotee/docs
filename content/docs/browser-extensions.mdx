The S.EE browser extension brings URL shortening, text sharing, and file uploads directly to your browser. Available for Chrome, Firefox, and Edge.

Open source on [GitHub](https://github.com/sdotee/browser-extensions).

## Features

| Feature | Description |
|---------|-------------|
| URL Shortening | Shorten any URL with one click, custom slugs, and multiple domain options |
| Text Sharing | Share code snippets and text with syntax highlighting support |
| File Sharing | Batch upload files with progress tracking |
| Export Formats | Copy results in Plain URL, Markdown, HTML, or BBCode format |
| QR Code Generation | Generate QR codes for any shortened URL |
| QR Code Export | Download QR codes in PNG, SVG, or PDF format (512x512) |
| Right-Click Menu | Shorten page/link, show QR, share selected text, upload images |
| History Management | Tab-aware history for URLs, texts, and files with batch delete |
| Draft Persistence | Input fields are saved automatically across sessions |
| Auto-Copy | Automatically copy results to clipboard |
| Dark/Light Mode | Automatic theme detection with manual toggle |
| Desktop Notifications | Get notified when actions complete via context menu |
| Cross-Browser | Works on Chrome (MV3), Firefox (MV3), and Edge |
