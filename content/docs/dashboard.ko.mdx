## 대시보드 개요

로그인한 뒤 [대시보드](https://s.ee/user/dashboard/)에 접속할 수 있습니다. S.EE 대시보드에는 두 가지 주요 탐색 영역이 있습니다.

### 왼쪽 사이드바

주요 기능에 접근할 수 있습니다.

| Feature | Description |
|---------|-------------|
| [Links](https://s.ee/user/links/) | 단축 링크 관리 |
| [QR Codes](https://s.ee/user/qr/) | QR 코드 생성 및 관리 |
| [Text Sharing](https://s.ee/user/text/) | 텍스트, 코드 스니펫, Markdown 공유 |
| [File Sharing](https://s.ee/user/files/) | 파일 업로드 및 공유 |
| [Domains](https://s.ee/user/domains/) | 사용자 정의 도메인 설정 |
| [Analytics](https://s.ee/user/analytics/) | 클릭 및 스캔 통계 확인 |
| [Team](https://s.ee/user/team/) | 팀 구성원 관리 |
| [UTM Builder](https://s.ee/user/library/utm/) | UTM 파라미터 생성 및 관리 |
| [Usage](https://s.ee/user/usage/) | 요금제 사용량 확인 |
| [Settings](https://s.ee/user/settings/) | 계정 및 결제 설정 |

모바일 기기에서는 사이드바가 자동으로 접힙니다.

### 프로필 메뉴

왼쪽 사이드바에 있습니다. 다음 작업에 사용할 수 있습니다.

- 문서 열기
- [API 키 관리](https://s.ee/user/developers/)
- [요금제 업그레이드](https://s.ee/user/pricing/)
- 계정 전환
- 로그아웃
