## 建立短網址

1. 前往 [https://s.ee/user/links/](https://s.ee/user/links/)
2. 點擊 **Create Link**
3. 輸入目標 URL
4. 設定選項（可選）
5. 點擊 **Create**

---

## 短網址選項

建立短網址時，您可以設定：

| 選項 | 說明 |
|--------|-------------|
| 自訂短碼 | 為連結設定自訂字尾（例如 `s.ee/link`） |
| 網域 | 從官方網域或您的自訂網域中選擇 |
| 密碼 | 造訪目標頁面時需要輸入密碼 |
| 過期時間 | 設定短網址停止生效的日期 |
| 定向 | 根據地區、裝置或語言重定向使用者 |
| 標籤 | 使用標籤整理短網址 |
| UTM 參數 | 新增行銷活動追蹤參數 |
| 參數傳遞 | 將 URL 參數轉發到目標地址 |

---

## 資料分析

點擊任意短網址即可檢視其資料分析：

- 總點擊次數
- 按地區統計的點擊次數
- 按裝置和瀏覽器統計的點擊次數
- 來源

---

## 管理短網址

在短網址儀表板中，您可以：

- **編輯** - 更新目標 URL 或設定
- **刪除** - 永久移除短網址
- **複製** - 將短網址複製到剪貼簿
