## 儀表板概覽

登入後即可前往您的 [儀表板](https://s.ee/user/dashboard/)。S.EE 儀表板主要分為兩個導覽區域：

### 左側邊欄

查看主要功能：

| 功能 | 說明 |
|---------|-------------|
| [短網址](https://s.ee/user/links/) | 管理您的短網址 |
| [QR Code](https://s.ee/user/qr/) | 建立和管理 QR Code |
| [文字分享](https://s.ee/user/text/) | 分享文字、程式碼片段和 Markdown |
| [檔案分享](https://s.ee/user/files/) | 上傳並分享檔案 |
| [網域](https://s.ee/user/domains/) | 設定自訂網域 |
| [資料分析](https://s.ee/user/analytics/) | 檢視點擊和掃碼統計 |
| [團隊](https://s.ee/user/team/) | 管理團隊成員 |
| [UTM Builder](https://s.ee/user/library/utm/) | 建立和管理 UTM 參數 |
| [用量](https://s.ee/user/usage/) | 監控您的套餐用量 |
| [設定](https://s.ee/user/settings/) | 帳號與帳單設定 |

在移動裝置上，側邊欄會自動摺疊。

### 個人資料選單

位於左側邊欄，可用於：

- 查看文件
- [管理 API 金鑰](https://s.ee/user/developers/)
- [升級套餐](https://s.ee/user/pricing/)
- 切換帳號
- 退出登入
