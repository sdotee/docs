[ShareX](https://getsharex.com/) on avoimen lähdekoodin kuvakaappaus- ja tiedostonjakotyökalu Windowsille. S.EE tarjoaa ShareX:lle omat uploader-määritykset, joten voit lähettää kuvakaappauksia, tiedostoja, tekstiä ja koodia suoraan S.EE:hen ja saada takaisin jaettavan linkin.

Katso myös: [ShareX:n mukautettujen uploaderien dokumentaatio](https://getsharex.com/docs/custom-uploader)

---

## Mukautetut uploaderit

S.EE:llä on määritykset näille upload-tyypeille:

| Uploader | Mitä se tekee |
|----------|---------------|
| URL Shortener | Lyhentää URL-osoitteita S.EE:n kautta |
| Text Uploader | Lataa tekstikatkelmia |
| Code Uploader | Lataa koodia syntaksikorostuksella |
| Markdown Uploader | Lataa Markdownin renderöidyllä esikatselulla |
| Image & File Uploader | Lataa kuvia ja tiedostoja |

### Tuonti

1. Avaa **ShareX**
2. Siirry kohtaan **Destinations** > **Custom uploader settings**
3. Napsauta **Import** ja tee jompikumpi seuraavista:
   - Valitse ladattu `.sxcu`-tiedosto
   - Liitä lataus-URL suoraan

## API-avaimen määrittäminen

1. Hanki API-avaimesi kohdasta **[Tools > API Token](https://s.ee/user/developers/)**
2. Avaa ShareX:ssä **Custom uploader settings** ja valitse tuotu S.EE-uploader
3. Aseta **Headers**-osiossa `Authorization`-arvoksi API-avaimesi

<Callout type="warn">
Älä jaa API-avaintasi julkisesti äläkä lisää sitä versionhallintaan.
</Callout>

## Käyttö

### Valitse uploaderit

Siirry ShareX:ssä kohtaan **Destinations** ja määritä S.EE-uploaderit:

- URL shortener → S.EE URL Shortener
- Text uploader → S.EE Text Uploader, Code Uploader tai Markdown Uploader
- Image uploader → S.EE File Uploader
- File uploader → S.EE File Uploader

### Lataaminen

Käytä ShareX:n pikanäppäimiä tai hiiren oikean painikkeen valikkoa URL:n lyhentämiseen, kuvakaappauksen lataamiseen, tekstikatkelman jakamiseen jne. ShareX lataa sisällön S.EE:hen ja kopioi linkin leikepöydällesi.

---

## Vianmääritys

| Ongelma | Korjaus |
|---------|---------|
| 401-virhe | Tarkista, että API-avaimesi on asetettu oikein Authorization-otsakkeeseen |
| 403-virhe | Varmista, että API-avaimellasi on oikeat käyttöoikeudet |
| Tiedosto hylättiin | Tiedostotyyppiä ei ehkä tueta. Katso sallitut tiedostotyypit kohdasta [Tiedostojen jakaminen](/docs/fi/file-sharing) |

Voit napsauttaa **Test** kohdassa **Custom uploader settings** varmistaaksesi, että kaikki toimii.
