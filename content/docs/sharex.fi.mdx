[ShareX](https://getsharex.com/) on avoimen lähdekoodin kuvakaappaus- ja tiedostonjakotyökalu Windowsille. S.EE tarjoaa ShareX:lle omat lataaja-määritykset, joten voit lähettää kuvakaappauksia, tiedostoja, tekstiä ja koodia suoraan S.EE:hen ja saada takaisin jaettavan linkin.

Katso myös: [ShareX:n mukautettujen lataajaien dokumentaatio](https://getsharex.com/docs/custom-lataaja)

---

## Mukautetut lataajait

S.EE:llä on määritykset näille upload-tyypeille:

| Uploader | Mitä se tekee |
|----------|---------------|
| URL-lyhennin | Lyhentää URL-osoitteita S.EE:n kautta |
| Tekstin lataaja | Lataa tekstikatkelmia |
| Koodin lataaja | Lataa koodia syntaksikorostuksella |
| Markdown-lataaja | Lataa Markdownin renderöidyllä esikatselulla |
| Kuvien ja tiedostojen lataaja | Lataa kuvia ja tiedostoja |

### Tuonti

1. Avaa **ShareX**
2. Siirry kohtaan **Kohteet** > **Mukautetun lataajan asetukset**
3. Napsauta **Tuo** ja tee jompikumpi seuraavista:
   - Valitse ladattu `.sxcu`-tiedosto
   - Liitä lataus-URL suoraan

## API-avaimen määrittäminen

1. Hanki API-avaimesi kohdasta **[Työkalut > API Token](https://s.ee/user/developers/)**
2. Avaa ShareX:ssä **Mukautetun lataajan asetukset** ja valitse tuotu S.EE-lataaja
3. Aseta **Otsakkeet**-osiossa `Authorization`-arvoksi API-avaimesi

<Callout type="warn">
Älä jaa API-avaintasi julkisesti äläkä lisää sitä versionhallintaan.
</Callout>

## Käyttö

### Valitse lataajait

Siirry ShareX:ssä kohtaan **Kohteet** ja määritä S.EE-lataajait:

- URL-lyhennin → S.EE URL Shortener
- Tekstin lataaja → S.EE Text Uploader, Code Uploader tai Markdown Uploader
- Kuvien lataaja → S.EE File Uploader
- Tiedostojen lataaja → S.EE File Uploader

### Lataaminen

Käytä ShareX:n pikanäppäimiä tai hiiren oikean painikkeen valikkoa URL:n lyhentämiseen, kuvakaappauksen lataamiseen, tekstikatkelman jakamiseen jne. ShareX lataa sisällön S.EE:hen ja kopioi linkin leikepöydällesi.

---

## Vianmääritys

| Ongelma | Korjaus |
|---------|---------|
| 401-virhe | Tarkista, että API-avaimesi on asetettu oikein Authorization-otsakkeeseen |
| 403-virhe | Varmista, että API-avaimellasi on oikeat käyttöoikeudet |
| Tiedosto hylättiin | Tiedostotyyppiä ei ehkä tueta. Katso sallitut tiedostotyypit kohdasta [Tiedostojen jakaminen](/docs/fi/file-sharing) |

Voit napsauttaa **Testaa** kohdassa **Mukautetun lataajan asetukset** varmistaaksesi, että kaikki toimii.
