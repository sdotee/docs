## Overview

S.EE File Sharing allows you to upload and share files with a short, shareable link. Perfect for sharing images, documents, audio, video, and other files.

---

## Upload Files

### Quick Upload (Drag & Drop)

1. Log in to [https://s.ee/user/dashboard](https://s.ee/user/dashboard)
2. Click **Sharing > Files** in the left sidebar or go directly to [https://s.ee/user/files/](https://s.ee/user/files/)
3. Drag and drop files or folders into the **Drag & Drop** window
   - You can also click the window to browse and select files
   - Uploading a folder will upload all files within it
4. Files will be shared with the default domain and a random slug

### Advanced Upload (Create File)

For more control over your file share:

1. Click **Create File** button
2. Configure your settings:
   - **Domain** - Select your preferred domain for the share page
   - **Custom Slug** - Set a custom URL slug
   - **Other options** - Tags, expiration, etc.
3. Select your file and upload

---

## File Links

Each uploaded file has two types of links:

| Link Type | Format | Description |
|-----------|--------|-------------|
| Share Page | `fs.to/[your_slug]` | Customizable domain and slug |
| Direct Link | `i.see.you/[yyyy]/[mm]/[dd]/[random_slug]/[filename].[ext]` | Direct access to the file with original filename |

**Note:** Direct links currently do not support custom domains.

<Callout type="warn">
Executable files (.exe, .msi, .bat, .app, etc.) are **not allowed** for security reasons.
</Callout>

<Callout type="info">
For the following file extensions, we may force the response `Content-Type` to `text/plain` to reduce script abuse: `.js`, `.mjs`, `.html`, `.htm`, `.xhtml`, `.shtml`, `.svg`, `.xml`, `.json`, `.map`.
</Callout>

<Callout type="info">
For unknown or ambiguous file types, we will force the response `Content-Type` to `application/octet-stream`.
</Callout>

## File Retention

- Files are **permanently stored** as long as your subscription is active
- If your subscription expires, files are retained for **7 days** before deletion

## FAQ

### Who can upload files?

Only paid subscribers can upload files. A valid subscription is required.

### Are there download limits?

No, there are no download count or bandwidth limits for shared files.

### Can I customize the share page URL?

Yes, you can set a custom domain and slug for the share page when using the **Create File** upload method.

### Can I customize the direct link URL?

Currently, direct links use a fixed format and do not support custom domains.
