## Problèmes de compte et de connexion

| Message d’erreur | Cause | Solution |
|---------------|-------|----------|
| "Invalid email or password. Please check your credentials and try again." | E-mail ou mot de passe incorrect | Vérifiez votre e-mail et votre mot de passe. Les mots de passe sont sensibles à la casse. |
| "We couldn't find an account with this email." | E-mail non enregistré | Vérifiez votre adresse e-mail ou créez un nouveau compte. |
| "Your account has been suspended. Please contact support for assistance." | Compte suspendu | Contactez notre équipe d’assistance à [hi@s.ee](mailto:hi@s.ee). |
| "This login link has expired or is invalid. Please request a new one." | Lien de connexion expiré | Demandez un nouveau lien de connexion depuis la page de connexion. |
| "This verification code has expired or is invalid. Please request a new one." | Code de vérification expiré ou invalide | Demandez un nouveau code de vérification. |
| "This email is already registered." | E-mail déjà utilisé | Utilisez une autre adresse e-mail ou essayez de vous connecter avec cet e-mail. |
| "The current password you entered is incorrect." | Mot de passe actuel incorrect lors du changement de mot de passe | Vérifiez votre mot de passe actuel puis réessayez. |

**Remarque :** seule votre adresse e-mail principale fonctionne pour la connexion. Les autres e-mails vérifiés de votre compte ne peuvent pas être utilisés pour l’authentification. Les e-mails de réinitialisation du mot de passe sont également envoyés uniquement à votre adresse e-mail principale.

## Problèmes de nom d’utilisateur

| Message d’erreur | Cause | Solution |
|---------------|-------|----------|
| "This username is already taken. Please choose a different one." | Nom d’utilisateur déjà utilisé | Choisissez un autre nom d’utilisateur. |
| "Username can only be changed once." | Limite de changement atteinte | Malheureusement, le nom d’utilisateur ne peut être modifié qu’une seule fois par compte. Cette modification est permanente. |

## Problèmes de liens courts

| Message d’erreur | Cause | Solution |
|---------------|-------|----------|
| "This slug is already taken. Please choose a different one." | Slug personnalisé déjà utilisé | Essayez un autre slug personnalisé pour votre lien court. |
| "This slug cannot be used. Please choose a different one." | Slug réservé par le système | Ce slug est réservé. Choisissez-en un autre. |
| "This short link does not exist or is no longer available." | Le lien court n’existe pas ou a été supprimé | Vérifiez que l’URL du lien est correcte. Il a peut-être été supprimé. |
| "The domain linked to this short URL is unavailable or no longer exists." | Domaine indisponible | Contactez l’assistance ou essayez un autre domaine. |
| "Please enter a valid destination URL starting with http:// or https://" | Format d’URL invalide | Assurez-vous que l’URL de destination commence par `http://` ou `https://`. |
| "This destination URL is not allowed. Please use a different link." | URL de destination bloquée | Cette URL figure sur notre liste de blocage. Utilisez une autre URL de destination. |
| "Incorrect password. Please try again." | Mauvais mot de passe pour un lien protégé | Saisissez le bon mot de passe pour accéder à ce lien. |
| "URL has already been shortened." | URL déjà raccourcie | Cette URL possède déjà un lien court existant. |

## Problèmes d’abonnement et d’offre

| Message d’erreur | Cause | Solution |
|---------------|-------|----------|
| "You already have an active subscription plan." | Déjà abonné | Si vous souhaitez changer d’offre, annulez d’abord votre abonnement actuel. |
| "You don't have an active subscription to cancel." | Aucun abonnement actif | Vous n’avez actuellement aucun abonnement actif à annuler. |
| "This feature requires a premium subscription. Please upgrade your plan to access it." | Fonction réservée à une offre supérieure | Passez à une offre supérieure pour accéder à cette fonctionnalité. |
| "Unable to find your subscription plan. Please contact support if this issue persists." | Offre introuvable | Contactez notre équipe d’assistance à [hi@s.ee](mailto:hi@s.ee). |
| "The selected pricing option is no longer available." | Option tarifaire expirée | Actualisez la page et sélectionnez une option tarifaire disponible. |

## Problèmes de téléversement de fichiers

| Message d’erreur | Cause | Solution |
|---------------|-------|----------|
| "File size exceeds limit" | Fichier trop volumineux | Compressez votre fichier ou passez à une offre supérieure pour téléverser des fichiers plus volumineux. |
| "Storage limit exceeded" | Quota de stockage atteint | Supprimez quelques fichiers pour libérer de l’espace, ou passez à une offre supérieure pour obtenir plus de stockage. |
| "Unsupported file type" / "File type not allowed" | Format de fichier non pris en charge | Utilisez un format de fichier pris en charge. |
| "Maximum concurrent uploads exceeded" | Trop de téléversements simultanés | Attendez la fin de vos téléversements en cours avant d’en lancer de nouveaux. |
| "This file format is not supported. Allowed formats: JPG, JPEG, PNG, GIF, WEBP, AVIF." | Format d’avatar non pris en charge | Téléversez votre avatar dans l’un des formats suivants : JPG, JPEG, PNG, GIF, WEBP ou AVIF. |

## Problèmes d’équipe

| Message d’erreur | Cause | Solution |
|---------------|-------|----------|
| "This feature is not available on your current plan." | Fonction d’équipe non incluse | Passez à une offre incluant les fonctionnalités d’équipe. |
| "You have already created a team." | L’équipe existe déjà | Chaque utilisateur ne peut créer qu’une seule équipe. Vous pouvez gérer votre équipe existante depuis le tableau de bord. |
| "You don't have permission to manage this team." | Permissions insuffisantes | Contactez le propriétaire de votre équipe. |
| "The invitation has already been processed." | Invitation déjà traitée | Cette invitation a déjà été acceptée ou refusée. |
| "Please wait at least 24 hours before resending." | Renvoi trop rapide | Attendez 24 heures avant d’envoyer une nouvelle invitation à cet utilisateur. |

## Problèmes de gestion des e-mails
