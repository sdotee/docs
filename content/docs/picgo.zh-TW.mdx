[PicGo](https://picgo.app/) 是一款適用於 Windows、macOS 和 Linux 的開源圖片與檔案上傳工具。較新的版本已原生支援 S.EE，您只需要一個 API Token 即可開始使用。

- [下載 PicGo](https://github.com/Molunerfinn/PicGo/releases)
- [原始碼](https://github.com/Molunerfinn/PicGo)

---

## 設定

### 1. 取得您的 API Token

1. 登入您的 S.EE 帳號
2. 前往 **[Tools > API Token](https://s.ee/user/developers/)**
3. 生成新的 API Token

<Callout type="warn">
不要公開分享您的 API Token，也不要將其提交到版本控制系統中。
</Callout>

### 2. 設定 PicGo

1. 下載並安裝 [PicGo](https://github.com/Molunerfinn/PicGo/releases)
2. 開啟 PicGo
3. 前往 **Picbeds 設定** > **SM.MS (S.EE)**
4. 點擊 **+** 新增新設定
5. 在 **Configuration Name** 中輸入名稱（例如 `S.EE`）
6. 將您的 API Token 貼上到 **Set Token**
7. 點擊 **Confirm**

![PicGo S.EE 設定](https://i.see.you/2026/02/10/k6Nr/pasted-image-1770748236942.webp)

## 使用方法

您可以透過以下幾種方式上傳：

- 將檔案拖拽到上傳區域
- 點擊上傳區域，從電腦中選擇檔案
- 點擊懸浮迷你視窗直接上傳
- 右鍵懸浮迷你視窗並選擇 **Upload by Clipboard**

PicGo 會將生成的連結複製到您的剪貼簿。您可以選擇輸出格式，例如 Markdown、HTML、原始 URL 等。
