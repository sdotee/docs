## Konto ja sisselogimise probleemid

| Veateade | Põhjus | Lahendus |
|---------------|-------|----------|
| "Invalid email or password. Please check your credentials and try again." | Vale e-post või parool | Kontrolli oma e-posti aadress ja parool üle. Pane tähele, et paroolid on tõstutundlikud. |
| "We couldn't find an account with this email." | E-post ei ole registreeritud | Kontrolli e-posti aadressi või loo uus konto. |
| "Your account has been suspended. Please contact support for assistance." | Konto on peatatud | Abi saamiseks kirjuta meie toe aadressile [hi@s.ee](mailto:hi@s.ee). |
| "This login link has expired or is invalid. Please request a new one." | Sisselogimislink on aegunud | Küsi sisselogimislehelt uus sisselogimislink. |
| "This verification code has expired or is invalid. Please request a new one." | Kinnituskood on aegunud või vigane | Küsi uus kinnituskood. |
| "This email is already registered." | E-post on juba kasutusel | Kasuta teist e-posti aadressi või proovi selle e-postiga sisse logida. |
| "The current password you entered is incorrect." | Vale praegune parool parooli vahetamisel | Kontrolli praegune parool üle ja proovi uuesti. |

**Märkus:** sisselogimiseks töötab ainult sinu peamine e-posti aadress. Teised kontole lisatud kinnitatud aadressid autentimiseks ei sobi. Parooli lähtestamise kirjad saadetakse samuti ainult peamisele e-posti aadressile.

## Lühilinkide probleemid

| Veateade | Põhjus | Lahendus |
|---------------|-------|----------|
| "This slug is already taken. Please choose a different one." | Kohandatud lühikood on juba kasutusel | Proovi lühilingi jaoks teist kohandatud lühikoodi. |
| "This slug cannot be used. Please choose a different one." | Lühikood on süsteemi poolt reserveeritud | See lühikood on reserveeritud. Vali teine. |
| "This short link does not exist or is no longer available." | Lühilinki ei eksisteeri või see on kustutatud | Kontrolli, et lingi URL oleks õige. Link võis olla kustutatud. |
| "The domain linked to this short URL is unavailable or no longer exists." | Domeen ei ole saadaval | Võta ühendust toega või proovi teist domeeni. |
| "Please enter a valid destination URL starting with http:// or https://" | Vigane URL-i vorming | Veendu, et siht-URL algab `http://` või `https://`. |
| "This destination URL is not allowed. Please use a different link." | Siht-URL on blokeeritud | See URL on meie blokeerimisloendis. Kasuta teist siht-URL-i. |
| "Incorrect password. Please try again." | Vale parool parooliga kaitstud lingi puhul | Sisesta selle lingi avamiseks õige parool. |
| "URL has already been shortened." | URL on juba lühendatud | Selle URL-iga on juba seotud olemasolev lühilink. |

## Tellimuse ja paketi probleemid

| Veateade | Põhjus | Lahendus |
|---------------|-------|----------|
| "You already have an active subscription plan." | Tellimus on juba aktiivne | Kui tahad paketti vahetada, tühista esmalt praegune tellimus. |
| "You don't have an active subscription to cancel." | Aktiivset tellimust pole | Sul ei ole praegu aktiivset tellimust, mida tühistada. |
| "This feature requires a premium subscription. Please upgrade your plan to access it." | Funktsioon nõuab kõrgemat paketti | Selle funktsiooni kasutamiseks uuenda oma tellimust. |
| "Unable to find your subscription plan. Please contact support if this issue persists." | Paketti ei leitud | Kui probleem püsib, kirjuta meie toe aadressile [hi@s.ee](mailto:hi@s.ee). |
| "The selected pricing option is no longer available." | Hinnastuse valik on aegunud | Värskenda lehte ja vali kehtiv hinnastuse valik. |

## Failide üleslaadimise probleemid

| Veateade | Põhjus | Lahendus |
|---------------|-------|----------|
| "File size exceeds limit" | Fail on liiga suur | Tihenda faili või uuenda paketti, et saaksid suuremaid faile üles laadida. |
| "Storage limit exceeded" | Salvestuslimiit on täis | Kustuta ruumi vabastamiseks mõned failid või uuenda paketti. |
| "Unsupported file type" / "File type not allowed" | Failivorming ei ole toetatud | Kasuta toetatud failivormingut. |
| "Maximum concurrent uploads exceeded" | Liiga palju samaaegseid üleslaadimisi | Oota, kuni praegused üleslaadimised lõpevad, enne kui alustad uutega. |
| "This file format is not supported. Allowed formats: JPG, JPEG, PNG, GIF, WEBP, AVIF." | Profiilipildi vorming pole toetatud | Laadi profiilipilt üles ühes toetatud vormingus: JPG, JPEG, PNG, GIF, WEBP või AVIF. |
