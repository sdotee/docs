## CLI 도구

| Name | Description | Link |
|------|-------------|------|
| **see-cli** | 명령줄에서 S.EE 를 사용합니다. URL 단축, 텍스트 공유, 파일 업로드를 지원합니다. | [sdotee/cli](https://github.com/sdotee/cli) |

## 플러그인

| Name | Description | Link |
|------|-------------|------|
| **S.EE for WordPress** | WordPress 용 URL 단축, 텍스트 공유, 파일 호스팅 플러그인입니다. 게시 시 단축 URL 을 자동 생성하고, 미디어를 S.EE 로 업로드하며, 모든 활동을 대시보드에서 추적합니다. | [WordPress.org](https://wordpress.org/plugins/sdotee/) · [GitHub](https://github.com/sdotee/see-wordpress) · [Docs](/docs/ko/wordpress) |
| **S.EE for Raycast** | Raycast 에서 바로 URL 단축, 텍스트 공유, 파일 업로드를 수행할 수 있습니다. 클립보드 내용을 자동 감지하고, 사용자 정의 슬러그, 비밀번호, 만료일, 탐색 가능한 기록을 지원합니다. | [Raycast Store](https://www.raycast.com/missuo/sdotee) · [GitHub](https://github.com/missuo/raycast-sdotee) |
| **Typecho-Plugin-PicUp** | 여러 백엔드를 지원하는 Typecho용 이미지 및 첨부파일 업로드 플러그인입니다. S.EE 이미지 업로드도 지원하여 Typecho 사이트에서 미디어를 더 쉽게 업로드하고 관리할 수 있습니다. | [GitHub](https://github.com/lhl77/Typecho-Plugin-PicUp) · [Homepage](https://blog.lhl.one/artical/1026.html) |
