De browserextensie van S.EE brengt URL-verkorting, tekst delen en bestandsuploads rechtstreeks naar uw browser. Beschikbaar voor Chrome, Firefox en Edge.

Open source op [GitHub](https://github.com/sdotee/browser-extensions).

## Functies

| Functie | Beschrijving |
|---------|-------------|
| URL-verkorting | Verkort elke URL met een klik, met aangepaste slugs en meerdere domeinopties |
| Tekst delen | Deel codefragmenten en tekst met ondersteuning voor syntax highlighting |
| Bestanden delen | Upload bestanden in batches met voortgangsweergave |
| Exportformaten | Kopieer resultaten als gewone URL, Markdown, HTML of BBCode |
| QR-codegeneratie | Genereer QR-codes voor elke verkorte URL |
| QR-code-export | Download QR-codes als PNG, SVG of PDF (512x512) |
| Rechtsklikmenu | Verkort pagina of link, toon QR-code, deel geselecteerde tekst en upload afbeeldingen |
| Geschiedenisbeheer | Tabbladbewuste geschiedenis voor URL's, teksten en bestanden met batchverwijdering |
| Conceptbehoud | Invoervelden worden automatisch opgeslagen tussen sessies |
| Automatisch kopieren | Kopieer resultaten automatisch naar het klembord |
| Donkere/lichtmodus | Automatische themadetectie met handmatige schakelaar |
| Desktopmeldingen | Ontvang meldingen wanneer acties via het contextmenu zijn voltooid |
| Cross-browser | Werkt op Chrome (MV3), Firefox (MV3) en Edge |
