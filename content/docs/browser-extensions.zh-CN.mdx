S.EE 浏览器扩展将短网址、文本分享和文件上传直接带到您的浏览器中。支持 Chrome、Firefox 和 Edge。

开源地址：[GitHub](https://github.com/sdotee/browser-extensions)。

## 功能

| 功能 | 说明 |
|---------|-------------|
| URL 缩短 | 一键缩短任意 URL，支持自定义短码和多个域名选项 |
| 文本分享 | 支持带语法高亮的代码片段和文本分享 |
| 文件分享 | 支持批量上传文件并跟踪进度 |
| 导出格式 | 可将结果复制为 Plain URL、Markdown、HTML 或 BBCode |
| 二维码生成 | 为任意短网址生成二维码 |
| 二维码导出 | 将二维码下载为 PNG、SVG 或 PDF 格式（512x512） |
| 右键菜单 | 缩短页面 / 链接、显示二维码、分享选中文本、上传图片 |
| 历史记录管理 | 按标签页区分 URL、文本和文件历史，并支持批量删除 |
| 草稿持久化 | 输入字段会跨会话自动保存 |
| 自动复制 | 自动将结果复制到剪贴板 |
| 深色 / 浅色模式 | 自动检测主题，并支持手动切换 |
| 桌面通知 | 通过右键菜单在操作完成时接收通知 |
| 跨浏览器 | 支持 Chrome（MV3）、Firefox（MV3）和 Edge |
