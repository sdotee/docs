[PicGo](https://picgo.app/), Windows, macOS ve Linux'ta görsel ve dosya yüklemek için kullanılan açık kaynaklı bir araçtır.

Yeni sürümlerde S.EE desteği hazır gelir; başlamanız için yalnızca bir API anahtarı令牌 gerekir.

- [PicGo'yu indirin](https://github.com/Molunerfinn/PicGo/releases)
- [Kaynak kodu](https://github.com/Molunerfinn/PicGo)

---

## Kurulum

### 1. API anahtarı令牌 alın

1. S.EE hesabınıza giriş yapın
2. **[Araçlar > API anahtarı令牌](https://s.ee/user/developers/)** sayfasına gidin
3. Yeni bir API anahtarı令牌 oluşturun

<Callout type="warn">
API anahtarı令牌'ınızı herkese açık olarak paylaşmayın ve sürüm kontrolüne eklemeyin.
</Callout>

### 2. PicGo'yu yapılandırın

1. [PicGo](https://github.com/Molunerfinn/PicGo/releases) uygulamasını indirip kurun
2. PicGo'yu açın
3. **Picbeds ayarları** > **SM.MS (S.EE)** bölümüne gidin
4. Yeni bir yapılandırma eklemek için **+** düğmesine tıklayın
5. **Yapılandırma adı** alanına bir ad girin (ör. `S.EE`)
6. API anahtarı令牌'ınızı **Token ayarla** alanına yapıştırın
7. **Onayla** düğmesine tıklayın

![PicGo S.EE Yapılandırması](https://i.see.you/2026/02/10/k6Nr/pasted-image-1770748236942.webp)

## Kullanım

Yükleme için birkaç farklı yöntem vardır:

- Dosyaları Yükleme Alanı'na sürükleyip bırakın
- Bilgisayarınızdan dosya seçmek için Yükleme Alanı'na tıklayın
- Doğrudan yüklemek için küçük yüzen pencereye tıklayın
- Yüzen pencereye sağ tıklayıp **Panodan yükle** seçeneğini kullanın

PicGo, oluşan bağlantıyı panonuza kopyalar. Çıktı biçimi olarak Markdown, HTML veya ham URL gibi formatları seçebilirsiniz.
