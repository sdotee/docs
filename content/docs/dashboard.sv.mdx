## Översikt över instrumentpanelen

När du har loggat in kan du öppna [instrumentpanelen](https://s.ee/user/dashboard/). S.EE:s instrumentpanel har två huvudsakliga navigationsområden:

### Vänster sidofält

Här kommer du åt huvudfunktionerna:

| Funktion | Beskrivning |
|---------|-------------|
| [Länkar](https://s.ee/user/links/) | Hantera dina korta länkar |
| [QR-koder](https://s.ee/user/qr/) | Skapa och hantera QR-koder |
| [Textdelning](https://s.ee/user/text/) | Dela text, kodsnuttar och Markdown |
| [Fildelning](https://s.ee/user/files/) | Ladda upp och dela filer |
| [Domäner](https://s.ee/user/domains/) | Konfigurera egna domäner |
| [Analys](https://s.ee/user/analytics/) | Visa klick- och skanningsstatistik |
| [Team](https://s.ee/user/team/) | Hantera teammedlemmar |
| [UTM Builder](https://s.ee/user/library/utm/) | Skapa och hantera UTM-parametrar |
| [Användning](https://s.ee/user/usage/) | Följ användningen för ditt abonnemang |
| [Inställningar](https://s.ee/user/settings/) | Konto- och faktureringsinställningar |

På mobila enheter fälls sidofältet ihop automatiskt.

### Profilmeny

Den finns i vänstra sidofältet. Därifrån kan du:

- Öppna dokumentationen
- [Hantera API-nycklar](https://s.ee/user/developers/)
- [Uppgradera abonnemanget](https://s.ee/user/pricing/)
- Växla mellan konton
- Logga ut
