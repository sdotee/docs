## Что такое S.EE?

S.EE — это платформа управления ссылками для сокращения URL, создания QR-кодов и обмена контентом со встроенной аналитикой и поддержкой пользовательских доменов. В платформу входят:

- **[S.EE Links](https://s.ee/user/links/)** — создание коротких ссылок с пользовательскими доменами, защитой паролем и умным таргетингом
- **[S.EE QR Codes](https://s.ee/user/qr/)** — создание настраиваемых QR-кодов с аналитикой
- **[S.EE Text Sharing](https://s.ee/user/text/)** — публикация обычного текста, фрагментов кода и Markdown по коротким ссылкам
- **[S.EE File Sharing](https://s.ee/user/files/)** — загрузка и обмен файлами по коротким ссылкам
- **[S.EE Image Hosting](https://s.ee/user/images/)** — загрузка и хостинг изображений по коротким ссылкам

## Ключевые возможности

### Короткие ссылки

- Пользовательские короткие коды и домены
- Защита паролем
- Срок действия
- Таргетинг по местоположению, устройству или языку
- UTM-параметры
- Передача параметров

### QR-коды

- Настройка цветов и логотипов
- Скачивание в PNG, SVG или PDF
- Встроенная аналитика

### Обмен текстом

- Поддержка обычного текста, фрагментов кода и Markdown
- Подсветка синтаксиса для кода
- Защита паролем и срок действия
- Endpoint с raw-текстом для удобного скачивания

### Обмен файлами

- Загрузка перетаскиванием, вставкой из буфера обмена или папкой
- Пользовательские домены и короткие коды
- Защита приватных файлов паролем и сроком действия
- Прямые ссылки на скачивание

### Хостинг изображений

- Загрузка перетаскиванием, вставкой из буфера обмена или папкой
- Автоматическая оптимизация изображений при вставке из буфера обмена
- Прямые ссылки на изображения для встраивания

### Аналитика

- Количество кликов
- Геоданные
- Информация об устройствах и браузерах
- Источники переходов
