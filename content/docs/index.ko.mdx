## S.EE 는 무엇인가요?

S.EE 는 URL 단축, QR 코드 생성, 콘텐츠 공유를 위한 링크 관리 플랫폼입니다. 기본 제공되는 분석 기능과 사용자 정의 도메인도 함께 지원합니다. 플랫폼에는 다음 기능이 포함됩니다.

- **[S.EE Links](https://s.ee/user/links/)** - 사용자 정의 도메인, 비밀번호 보호, 스마트 타기팅을 지원하는 단축 링크 생성
- **[S.EE QR Codes](https://s.ee/user/qr/)** - 분석 기능이 포함된 맞춤형 QR 코드 생성
- **[S.EE Text Sharing](https://s.ee/user/text/)** - 일반 텍스트, 코드 스니펫, Markdown 을 단축 링크로 공유
- **[S.EE File Sharing](https://s.ee/user/files/)** - 파일 업로드 및 단축 링크 공유
- **[S.EE Image Hosting](https://s.ee/user/images/)** - 이미지 업로드 및 단축 링크 호스팅

## 주요 기능

### 링크 관리

다음 옵션으로 단축 링크를 만들 수 있습니다.

- 사용자 정의 슬러그와 도메인
- 비밀번호 보호
- 만료일 설정
- 위치, 기기, 언어 기반 타기팅
- UTM 파라미터
- 파라미터 전달

### QR 코드

다음 옵션으로 QR 코드를 생성할 수 있습니다.

- 사용자 정의 색상과 로고
- PNG, SVG, PDF 다운로드
- 기본 제공 분석 기능

### 텍스트 공유

다음 형식의 텍스트를 공유할 수 있습니다.

- 일반 텍스트, 코드 스니펫, Markdown 지원
- 코드 구문 강조
- 비밀번호 보호 및 만료 설정
- 간편 다운로드를 위한 원본 텍스트 엔드포인트

### 파일 공유

다음 방식으로 파일을 업로드하고 공유할 수 있습니다.

- 드래그 앤 드롭, 붙여넣기, 폴더 업로드
- 사용자 정의 도메인과 슬러그
- 비공개 파일용 비밀번호 보호 및 만료 설정
- 직접 다운로드 링크

### 이미지 호스팅

다음 방식으로 이미지를 호스팅할 수 있습니다.

- 드래그 앤 드롭, 붙여넣기, 폴더 업로드
- 클립보드에서 붙여넣을 때 자동 이미지 최적화
- 임베드를 위한 직접 이미지 링크

### 분석

다음 항목으로 성과를 추적할 수 있습니다.

- 클릭 수
- 지역 데이터
- 기기 및 브라우저 정보
- 리퍼러 소스
