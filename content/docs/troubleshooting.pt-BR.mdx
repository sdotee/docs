## Problemas de conta e login

| Mensagem de erro | Causa | Solução |
|---------------|-------|----------|
| "Invalid email or password. Please check your credentials and try again." | Email ou senha incorretos | Verifique com cuidado seu email e sua senha. Lembre-se de que senhas diferenciam maiúsculas de minúsculas. |
| "We couldn't find an account with this email." | Email não cadastrado | Verifique seu endereço de email ou crie uma nova conta. |
| "Your account has been suspended. Please contact support for assistance." | Conta suspensa | Entre em contato com nossa equipe de suporte em [hi@s.ee](mailto:hi@s.ee). |
| "This login link has expired or is invalid. Please request a new one." | Link de login expirado | Solicite um novo link de login na página de login. |
| "This verification code has expired or is invalid. Please request a new one." | Código de verificação expirado ou inválido | Solicite um novo código de verificação. |
| "This email is already registered." | Email já usado | Use outro endereço de email ou tente fazer login com esse email. |
| "The current password you entered is incorrect." | Senha atual incorreta ao alterá-la | Verifique sua senha atual e tente novamente. |

**Observação:** apenas o seu endereço de email principal funciona para login. Outros emails verificados na conta não funcionam para autenticação. Emails de redefinição de senha também são enviados apenas para o email principal.

| Mensagem de erro | Causa | Solução |
|---------------|-------|----------|
| "This username is already taken. Please choose a different one." | Nome de usuário já está em uso | Escolha um nome de usuário diferente. |

## Problemas com URLs curtas

| Mensagem de erro | Causa | Solução |
|---------------|-------|----------|
| "This slug is already taken. Please choose a different one." | Slug personalizado já está em uso | Tente outro slug personalizado para sua URL curta. |
| "This slug cannot be used. Please choose a different one." | O slug é reservado pelo sistema | Esse slug está reservado. Escolha outro. |
| "This short link does not exist or is no longer available." | A URL curta não existe ou foi excluída | Verifique se a URL do link está correta. O link pode ter sido excluído. |
| "The domain linked to this short URL is unavailable or no longer exists." | Domínio indisponível | Entre em contato com o suporte ou tente usar outro domínio. |
| "Please enter a valid destination URL starting with http:// or https://" | Formato de URL inválido | Verifique se a URL de destino começa com `http://` ou `https://`. |
| "This destination URL is not allowed. Please use a different link." | URL de destino bloqueada | Essa URL está na nossa lista de bloqueio. Use outra URL de destino. |
| "Incorrect password. Please try again." | Senha incorreta para link protegido | Digite a senha correta para acessar esse link. |
| "URL has already been shortened." | A URL já foi encurtada | Essa URL já tem uma URL curta associada. |

## Problemas de assinatura e plano

| Mensagem de erro | Causa | Solução |
|---------------|-------|----------|
| "You already have an active subscription plan." | Você já assinou um plano | Se quiser mudar de plano, cancele primeiro sua assinatura atual. |
| "You don't have an active subscription to cancel." | Sem assinatura ativa | No momento você não tem nenhuma assinatura ativa para cancelar. |
| "This feature requires a premium subscription. Please upgrade your plan to access it." | O recurso exige um nível superior | Faça upgrade da sua assinatura para acessar esse recurso. |
| "Unable to find your subscription plan. Please contact support if this issue persists." | Plano não encontrado | Entre em contato com nossa equipe de suporte em [hi@s.ee](mailto:hi@s.ee). |
| "The selected pricing option is no longer available." | Opção de preço expirada | Atualize a página e selecione uma opção de preço atual. |

## Problemas no envio de arquivos

| Mensagem de erro | Causa | Solução |
|---------------|-------|----------|
| "File size exceeds limit" | O arquivo é grande demais | Comprima o arquivo ou faça upgrade do plano para permitir envios maiores. |
| "Storage limit exceeded" | Cota de armazenamento atingida | Exclua alguns arquivos para liberar espaço ou faça upgrade do plano para obter mais armazenamento. |
| "Unsupported file type" / "File type not allowed" | Formato de arquivo não compatível | Use um formato de arquivo compatível. |
| "Maximum concurrent uploads exceeded" | Envios simultâneos demais | Aguarde os envios atuais terminarem antes de iniciar novos. |
| "This file format is not supported. Allowed formats: JPG, JPEG, PNG, GIF, WEBP, AVIF." | Formato de avatar não compatível | Envie seu avatar em um dos formatos compatíveis: JPG, JPEG, PNG, GIF, WEBP ou AVIF. |

## Problemas de equipe

| Mensagem de erro | Causa | Solução |
|---------------|-------|----------|
| "This feature is not available on your current plan." | Recurso de equipe não incluído | Faça upgrade para um plano que inclua recursos de equipe. |
| "You have already created a team." | A equipe já existe | Cada usuário pode criar apenas uma equipe. Você pode gerenciar a equipe existente pelo painel. |
| "You don't have permission to manage this team." | Permissões insuficientes | Entre em contato com o proprietário da sua equipe. |
| "The invitation has already been processed." | Convite já processado | Esse convite já foi aceito ou recusado. |
| "Please wait at least 24 hours before resending." | Reenvio cedo demais | Aguarde 24 horas antes de reenviar outro convite a esse usuário. |

## Problemas ao excluir a conta

| Mensagem de erro | Causa | Solução |
|---------------|-------|----------|
| "Your team has other members, please delete the team first." | A equipe ainda tem membros | Remova todos os membros da equipe ou exclua a equipe antes de excluir a conta. |
| "Too many deletion requests, please try again later." | Pedidos demais | Aguarde um tempo antes de tentar novamente. |
| "Invalid or expired verification code." | Código de verificação inválido ou expirado | Solicite um novo código de verificação. |

## Problemas de gerenciamento de email

| Mensagem de erro | Causa | Solução |
|---------------|-------|----------|
| "Email already exists." | Email já está em uso | Use outro endereço de email. |
| "Main email cannot be deleted." | Não é possível excluir o email principal | Defina outro email verificado como principal antes de excluir este. |
| "You need to close two factor verification first." | Primeiro é preciso desativar o 2FA | Desative a autenticação de dois fatores antes de alterar seu email principal. |
| "This email has not been verified." | Email não verificado | Verifique primeiro esse endereço de email. |
