[PicGo](https://picgo.app/) は、Windows、macOS、Linux 向けのオープンソース画像 / ファイルアップロードツールです。最近のバージョンには S.EE のサポートが標準搭載されているため、API キー令牌 があればすぐに使い始められます。

- [PicGo をダウンロード](https://github.com/Molunerfinn/PicGo/releases)
- [ソースコード](https://github.com/Molunerfinn/PicGo)

---

## 設定

### 1. API キー令牌 を取得する

1. S.EE アカウントにログインします
2. **[ツール > API キー令牌](https://s.ee/user/developers/)** にアクセスします
3. 新しい API キー令牌 を発行します

<Callout type="warn">
API キー令牌 を公開したり、バージョン管理システムへコミットしたりしないでください。
</Callout>

### 2. PicGo を設定する

1. [PicGo](https://github.com/Molunerfinn/PicGo/releases) をダウンロードしてインストールします
2. PicGo を起動します
3. **Picbeds 設定** > **SM.MS (S.EE)** を開きます
4. **+** をクリックして新しい設定を追加します
5. **設定名** に名前（例: `S.EE`）を入力します
6. **Token を設定** に API キー令牌 を貼り付けます
7. **確認** をクリックします

## 使い方

アップロード方法は複数あります。

- ファイルをアップロードエリアへドラッグ＆ドロップする
- アップロードエリアをクリックしてファイルを選択する
- フローティングミニウィンドウをクリックして直接アップロードする
- フローティングミニウィンドウを右クリックし、**クリップボードからアップロード** を選択する

PicGo は生成されたリンクをクリップボードへコピーします。Markdown、HTML、Raw URL など、出力形式も選択できます。
