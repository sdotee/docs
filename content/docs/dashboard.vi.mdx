## Tổng quan bảng điều khiển

Truy cập [Bảng điều khiển](https://s.ee/user/dashboard/) sau khi đăng nhập. Bảng điều khiển S.EE có hai khu vực điều hướng chính:

### Thanh bên trái

Truy cập các tính năng chính của bạn:

| Feature | Description |
|---------|-------------|
| [Links](https://s.ee/user/links/) | Quản lý các liên kết ngắn |
| [QR Codes](https://s.ee/user/qr/) | Tạo và quản lý mã QR |
| [Text Sharing](https://s.ee/user/text/) | Chia sẻ văn bản, đoạn mã và Markdown |
| [File Sharing](https://s.ee/user/files/) | Tải lên và chia sẻ tệp |
| [Domains](https://s.ee/user/domains/) | Cấu hình tên miền tùy chỉnh |
| [Analytics](https://s.ee/user/analytics/) | Xem thống kê lượt nhấp và quét |
| [Team](https://s.ee/user/team/) | Quản lý thành viên nhóm |
| [UTM Builder](https://s.ee/user/library/utm/) | Tạo và quản lý tham số UTM |
| [Usage](https://s.ee/user/usage/) | Theo dõi mức sử dụng gói |
| [Cài đặt](https://s.ee/user/settings/) | Cài đặt tài khoản và thanh toán |

Trên thiết bị di động, thanh bên sẽ tự động thu gọn.

### Menu hồ sơ

Nằm ở thanh bên trái. Dùng menu này để:

- Truy cập tài liệu
- [Quản lý API key](https://s.ee/user/developers/)
- [Nâng cấp gói](https://s.ee/user/pricing/)
- Chuyển đổi giữa các tài khoản
- Đăng xuất
