[ShareX](https://getsharex.com/) on Windowsi avatud lähtekoodiga ekraanipildi- ja failijagamise tööriist. S.EE pakub ShareX-i jaoks kohandatud üleslaadija konfiguratsioone, nii et saad ekraanipilte, faile, teksti ja koodi otse S.EE-sse üles laadida ning vastu jagamislingi.

Vaata ka: [ShareX kohandatud üleslaadija docs](https://getsharex.com/docs/custom-üleslaadija)

---

## Kohandatud üleslaadijad

S.EE-l on konfiguratsioonid järgmiste üleslaadimistüüpide jaoks:

| Üleslaadija | Mida see teeb |
|----------|-------------|
| URL-i lühendaja | Lühendab URL-e S.EE kaudu |
| Teksti üleslaadija | Laadib üles tekstikatkeid |
| Koodi üleslaadija | Laadib üles koodi koos süntaksi esiletõstmisega |
| Markdowni üleslaadija | Laadib üles Markdowni renderdatud eelvaatega |
| Pildi- ja failiüleslaadija | Laadib üles pilte ja faile |

---

## Paigaldamine

Võta allolevad `.sxcu` konfiguratsioonifailid. Saad need alla laadida või ShareX-i URL-i kaudu importida.

| Üleslaadija | Allalaadimis- või impordilink |
|----------|------------------------|
| URL-i lühendaja | [https://assets.seecdn.com/files/sharex/S.EE_URL_Shortener.sxcu](https://assets.seecdn.com/files/sharex/S.EE_URL_Shortener.sxcu) |
| Teksti üleslaadija | [https://assets.seecdn.com/files/sharex/S.EE_Text_Uploader.sxcu](https://assets.seecdn.com/files/sharex/S.EE_Text_Uploader.sxcu) |
| Koodi üleslaadija | [https://assets.seecdn.com/files/sharex/S.EE_Code_Uploader.sxcu](https://assets.seecdn.com/files/sharex/S.EE_Code_Uploader.sxcu) |
| Markdowni üleslaadija | [https://assets.seecdn.com/files/sharex/S.EE_Markdown_Uploader.sxcu](https://assets.seecdn.com/files/sharex/S.EE_Markdown_Uploader.sxcu) |
| Pildi- ja failiüleslaadija | [https://assets.seecdn.com/files/sharex/S.EE_File_Uploader.sxcu](https://assets.seecdn.com/files/sharex/S.EE_File_Uploader.sxcu) |

### Importimineimine

1. Ava **ShareX**
2. Vali **Sihtkohad** > **Kohandatud üleslaadija seaded**
3. Klõpsa **Impordi** ja tee üks järgmistest:
   - vali allalaaditud `.sxcu` fail
   - kleebi otse allalaadimis-URL

## Seadista oma API võti

1. Hangi API võti menüüst **[Tööriistad > API token](https://s.ee/user/developers/)**
2. Ava ShareX-is **Kohandatud üleslaadija seaded** ja vali imporditud S.EE üleslaadija
3. Määra jaotises **Päised** väärtuseks `Authorization` oma API võti

<Callout type="warn">
Ära jaga oma API võtit avalikult ega lisa seda versioonihaldusse.
</Callout>

## Kasutamine

### Vali üleslaadijad

ShareX-is ava **Sihtkohad** ja määra S.EE üleslaadijad:

- URL-i lühendaja -> S.EE URL Shortener
- Teksti üleslaadija -> S.EE Text Uploader, Code Uploader või Markdown Uploader
- Pildi üleslaadija -> S.EE File Uploader
- Faili üleslaadija -> S.EE File Uploader

### Laadi üles

Kasuta URL-i lühendamiseks, ekraanipildi üleslaadimiseks või tekstikatke jagamiseks ShareX-i kiirklahve või paremklõpsumenüüd. ShareX laeb sisu S.EE-sse üles ja kopeerib lingi lõikelauale.

## Tõrkeotsing

| Probleem | Lahendus |
|-------|-----|
| 401 viga | Kontrolli, et API võti on `Authorization` päises õigesti seadistatud |
| 403 error | Veendu, et API võtmel on õiged õigused |
| File rejected | Failitüüp ei pruugi olla toetatud. Lubatud tüübid on kirjas lehel [Failide jagamine](/docs/et/file-sharing) |

Töötamise kinnitamiseks saad **Kohandatud üleslaadija seaded** all klõpsata **Testi**.
