Ekstensi browser S.EE menghadirkan pemendekan URL, berbagi teks, dan unggah file langsung ke browser Anda. Tersedia untuk Chrome, Firefox, dan Edge.

Open source di [GitHub](https://github.com/sdotee/browser-extensions).

## Fitur

| Feature | Description |
|---------|-------------|
| URL Shortening | Pendekkan URL apa pun dengan satu klik, dukungan slug kustom, dan beberapa pilihan domain |
| Text Sharing | Bagikan cuplikan kode dan teks dengan dukungan sorotan sintaks |
| File Sharing | Unggah file secara batch dengan pelacakan progres |
| Export Formats | Salin hasil sebagai Plain URL, Markdown, HTML, atau BBCode |
| QR Code Generation | Buat kode QR untuk URL pendek apa pun |
| QR Code Export | Unduh kode QR dalam format PNG, SVG, atau PDF (512x512) |
| Right-Click Menu | Pendekkan halaman/tautan, tampilkan QR, bagikan teks terpilih, unggah gambar |
| History Management | Riwayat per tab untuk URL, teks, dan file dengan dukungan hapus batch |
| Draft Persistence | Kolom input disimpan otomatis antar sesi |
| Auto-Copy | Otomatis menyalin hasil ke clipboard |
| Dark/Light Mode | Deteksi tema otomatis dengan tombol ubah manual |
| Desktop Notifications | Dapatkan notifikasi saat aksi selesai melalui menu konteks |
| Cross-Browser | Berfungsi di Chrome (MV3), Firefox (MV3), dan Edge |
