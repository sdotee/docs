## Was ist S.EE?

S.EE ist eine Plattform für Link-Management zum Kürzen von URLs, Erstellen von QR-Codes und Teilen von Inhalten – mit integrierter Analyse und Unterstützung für benutzerdefinierte Domains. Die Plattform umfasst:

- **[S.EE Links](https://s.ee/user/links/)** - Erstellen Sie Kurzlinks mit benutzerdefinierten Domains, Passwortschutz und intelligentem Targeting
- **[S.EE QR Codes](https://s.ee/user/qr/)** - Erstellen Sie anpassbare QR-Codes mit Analysefunktionen
- **[S.EE Text Sharing](https://s.ee/user/text/)** - Teilen Sie Klartext, Code-Snippets und Markdown über Kurzlinks
- **[S.EE File Sharing](https://s.ee/user/files/)** - Laden Sie Dateien hoch und teilen Sie sie über Kurzlinks
- **[S.EE Image Hosting](https://s.ee/user/images/)** - Laden Sie Bilder hoch und hosten Sie sie über Kurzlinks

---

## Wichtige Funktionen

### Link-Management

Erstellen Sie Kurzlinks mit:

- Benutzerdefinierten Slugs und Domains
- Passwortschutz
- Ablaufdatum
- Targeting nach Standort, Gerät oder Sprache
- UTM-Parametern
- Parameterweitergabe

### QR-Codes

Erstellen Sie QR-Codes mit:

- Benutzerdefinierten Farben und Logos
- Download als PNG, SVG oder PDF
- Integrierter Analyse

### Textfreigabe

Teilen Sie Textinhalte mit:

- Unterstützung für Klartext, Code-Snippets und Markdown
- Syntaxhervorhebung für Code
- Passwortschutz und Ablaufdatum
- Raw-Text-Endpunkt für einfache Downloads

### Dateifreigabe

Laden Sie Dateien hoch und teilen Sie sie mit:

- Drag-and-drop, Einfügen oder Ordner-Upload
- Benutzerdefinierten Domains und Slugs
- Passwortschutz für private Dateien sowie Ablaufdatum
- Direkten Download-Links

### Bild-Hosting

Hosten Sie Bilder mit:

- Drag-and-drop, Einfügen oder Ordner-Upload
- Automatischer Bildoptimierung beim Einfügen aus der Zwischenablage
- Direkten Bild-Links zum Einbetten

### Analyse

Verfolgen Sie die Leistung mit:

- Klickzahlen
- Geografischen Daten
- Geräte- und Browserinformationen
- Referrer-Quellen
