[ShareX](https://getsharex.com/) är ett verktyg med öppen källkod för skärminspelning och fildelning i Windows. S.EE erbjuder anpassade uppladdarkonfigurationer för ShareX, så att du kan ladda upp skärmbilder, filer, text och kod direkt till S.EE och få tillbaka en delbar länk.

Se även: [ShareX-dokumentation om anpassade uppladdare](https://getsharex.com/docs/custom-uppladdare)

---

## Anpassade uppladdare

S.EE har konfigurationer för följande uppladdningstyper:

| Uppladdare | Vad den gör |
|----------|-------------|
| URL-förkortare | Förkortar URL:er via S.EE |
| Textuppladdare | Laddar upp textsnuttar |
| Koduppladdare | Laddar upp kod med syntaxmarkering |
| Markdown-uppladdare | Laddar upp Markdown med renderad förhandsvisning |
| Bild- och filuppladdare | Laddar upp bilder och filer |

### Importeraera

1. Öppna **ShareX**
2. Gå till **Destinationer** > **Inställningar för anpassad uppladdare**
3. Klicka på **Importera** och gör sedan något av följande:
   - Välj den nedladdade `.sxcu`-filen
   - Klistra in nedladdnings-URL:en direkt

## Konfigurera API-nyckeln

1. Hämta din API-nyckel från **[Verktyg > API-token](https://s.ee/user/developers/)**
2. Öppna **Inställningar för anpassad uppladdare** i ShareX och välj den S.EE-uppladdare du importerade
3. I avsnittet **Rubriker** anger du din API-nyckel som värde för `Authorization`

<Callout type="warn">
Dela inte din API-nyckel offentligt och checka inte in den i versionshantering.
</Callout>

## Användning

### Välj uppladdare

I ShareX går du till **Destinationer** och kopplar S.EE-uppladdarna:

- URL-förkortare -> S.EE URL Shortener
- Textuppladdare -> S.EE Text Uploader, Code Uploader eller Markdown Uploader
- Bilduppladdare -> S.EE File Uploader
- Filuppladdare -> S.EE File Uploader

### Ladda upp

Använd ShareX-genvägarna eller högerklicksmenyn för att förkorta en URL, ladda upp en skärmbild, dela en textsnutt och så vidare. ShareX laddar upp till S.EE och kopierar länken till urklipp.

## Felsökning

| Problem | Lösning |
|-------|-----|
| 401-fel | Kontrollera att API-nyckeln är korrekt angiven i `Authorization`-headern |
| 403-fel | Kontrollera att API-nyckeln har rätt behörigheter |
| Filen nekas | Filtypen kanske inte stöds. Se [Fildelning](/docs/sv/file-sharing) för tillåtna typer |

Du kan klicka på **Testa** i **Inställningar för anpassad uppladdare** för att kontrollera att allt fungerar.
