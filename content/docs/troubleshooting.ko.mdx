## 계정 및 로그인 문제

| Error Message | Cause | Solution |
|---------------|-------|----------|
| "Invalid email or password. Please check your credentials and try again." | 이메일 또는 비밀번호가 올바르지 않습니다 | 이메일과 비밀번호를 다시 확인해 주세요. 비밀번호는 대소문자를 구분합니다. |
| "We couldn't find an account with this email." | 등록되지 않은 이메일입니다 | 이메일 주소를 다시 확인하거나 새 계정을 만들어 주세요. |
| "Your account has been suspended. Please contact support for assistance." | 계정이 정지되었습니다 | [hi@s.ee](mailto:hi@s.ee) 로 지원팀에 문의해 주세요. |
| "This login link has expired or is invalid. Please request a new one." | 로그인 링크가 만료되었거나 유효하지 않습니다 | 로그인 페이지에서 새 로그인 링크를 요청해 주세요. |
| "This verification code has expired or is invalid. Please request a new one." | 인증 코드가 만료되었거나 유효하지 않습니다 | 새 인증 코드를 요청해 주세요. |
| "This email is already registered." | 이미 사용 중인 이메일입니다 | 다른 이메일을 사용하거나 이 이메일로 로그인을 시도해 주세요. |
| "The current password you entered is incorrect." | 비밀번호 변경 시 현재 비밀번호가 틀렸습니다 | 현재 비밀번호를 확인한 뒤 다시 시도해 주세요. |

**참고:** 로그인에는 기본 이메일 주소만 사용할 수 있습니다. 계정에 추가로 인증된 다른 이메일 주소로는 인증할 수 없습니다. 비밀번호 재설정 메일도 기본 이메일 주소로만 전송됩니다.

## 사용자 이름 문제

| Error Message | Cause | Solution |
|---------------|-------|----------|
| "This username is already taken. Please choose a different one." | 이미 사용 중인 사용자 이름입니다 | 다른 사용자 이름을 선택해 주세요. |
| "Username can only be changed once." | 사용자 이름 변경 한도에 도달했습니다 | 사용자 이름은 계정당 한 번만 변경할 수 있습니다. 이 변경은 영구적입니다. |

## 단축 링크 문제

| Error Message | Cause | Solution |
|---------------|-------|----------|
| "This slug is already taken. Please choose a different one." | 사용자 정의 슬러그가 이미 사용 중입니다 | 다른 사용자 정의 슬러그를 사용해 주세요. |
| "This slug cannot be used. Please choose a different one." | 시스템 예약 슬러그입니다 | 이 슬러그는 예약되어 있습니다. 다른 슬러그를 선택해 주세요. |
| "This short link does not exist or is no longer available." | 단축 링크가 존재하지 않거나 삭제되었습니다 | 링크 URL 이 올바른지 확인해 주세요. 링크가 삭제되었을 수 있습니다. |
| "The domain linked to this short URL is unavailable or no longer exists." | 도메인을 사용할 수 없습니다 | 지원팀에 문의하거나 다른 도메인을 사용해 주세요. |
| "Please enter a valid destination URL starting with http:// or https://" | URL 형식이 올바르지 않습니다 | 대상 URL 이 `http://` 또는 `https://` 로 시작하는지 확인해 주세요. |
| "This destination URL is not allowed. Please use a different link." | 대상 URL 이 차단되었습니다 | 이 URL 은 차단 목록에 있습니다. 다른 대상 URL 을 사용해 주세요. |
| "Incorrect password. Please try again." | 비밀번호 보호 링크의 비밀번호가 틀렸습니다 | 올바른 비밀번호를 입력해 주세요. |
| "URL has already been shortened." | 이미 단축된 URL 입니다 | 이 URL 에는 이미 생성된 단축 링크가 연결되어 있습니다. |

## 구독 및 요금제 문제

| Error Message | Cause | Solution |
|---------------|-------|----------|
| "You already have an active subscription plan." | 이미 구독 중입니다 | 요금제를 변경하려면 먼저 현재 구독을 취소해 주세요. |
| "You don't have an active subscription to cancel." | 활성 구독이 없습니다 | 현재 취소할 수 있는 활성 구독이 없습니다. |
| "This feature requires a premium subscription. Please upgrade your plan to access it." | 더 높은 요금제가 필요한 기능입니다 | 이 기능을 사용하려면 요금제를 업그레이드해 주세요. |
| "Unable to find your subscription plan. Please contact support if this issue persists." | 요금제 정보를 찾을 수 없습니다 | 문제가 계속되면 [hi@s.ee](mailto:hi@s.ee) 로 문의해 주세요. |
| "The selected pricing option is no longer available." | 선택한 가격 옵션이 만료되었습니다 | 페이지를 새로고침한 뒤 현재 사용 가능한 옵션을 선택해 주세요. |

## 파일 업로드 문제

| Error Message | Cause | Solution |
|---------------|-------|----------|
| "File size exceeds limit" | 파일이 너무 큽니다 | 파일을 압축하거나 더 큰 업로드를 지원하는 상위 요금제로 업그레이드해 주세요. |
| "Storage limit exceeded" | 저장 공간 한도에 도달했습니다 | 일부 파일을 삭제해 공간을 확보하거나 더 많은 저장 공간이 포함된 요금제로 업그레이드해 주세요. |
| "Unsupported file type" / "File type not allowed" | 지원하지 않는 파일 형식입니다 | 지원되는 파일 형식을 사용해 주세요. |
| "Maximum concurrent uploads exceeded" | 동시에 업로드가 너무 많이 실행되었습니다 | 현재 업로드가 끝난 뒤 새 업로드를 시작해 주세요. |
| "This file format is not supported. Allowed formats: JPG, JPEG, PNG, GIF, WEBP, AVIF." | 아바타 형식이 지원되지 않습니다 | JPG, JPEG, PNG, GIF, WEBP, AVIF 형식 중 하나로 업로드해 주세요. |

## 팀 관련 문제

| Error Message | Cause | Solution |
|---------------|-------|----------|
| "This feature is not available on your current plan." | 현재 요금제에 팀 기능이 포함되어 있지 않습니다 | 팀 기능이 포함된 요금제로 업그레이드해 주세요. |
| "You have already created a team." | 이미 팀을 만들었습니다 | 사용자당 하나의 팀만 만들 수 있습니다. 기존 팀은 대시보드에서 관리할 수 있습니다. |
| "You don't have permission to manage this team." | 권한이 부족합니다 | 팀 소유자에게 문의해 주세요. |
| "The invitation has already been processed." | 초대가 이미 처리되었습니다 | 이 초대는 이미 수락되었거나 거절되었습니다. |
| "Please wait at least 24 hours before resending." | 초대 재전송이 너무 빠릅니다 | 이 사용자에게 다시 초대하려면 24 시간 기다려 주세요. |

| Error Message | Cause | Solution |
|---------------|-------|----------|
| "Your team has other members, please delete the team first." | 팀에 다른 구성원이 남아 있습니다 | 계정을 삭제하기 전에 모든 팀 구성원을 제거하거나 팀을 먼저 삭제해 주세요. |

| "Invalid or expired verification code." | 인증 코드가 유효하지 않거나 만료되었습니다 | 새 인증 코드를 요청해 주세요. |

## 이메일 관리 문제

| Error Message | Cause | Solution |
|---------------|-------|----------|
| "Email already exists." | 이미 사용 중인 이메일입니다 | 다른 이메일 주소를 사용해 주세요. |
| "Main email cannot be deleted." | 기본 이메일은 삭제할 수 없습니다 | 삭제하기 전에 다른 인증된 이메일을 기본 이메일로 설정해 주세요. |
| "You need to close two factor verification first." | 먼저 2단계 인증을 비활성화해야 합니다 | 기본 이메일을 변경하기 전에 2단계 인증을 꺼 주세요. |
| "This email has not been verified." | 이메일이 인증되지 않았습니다 | 먼저 이 이메일 주소를 인증해 주세요. |

## 더 도움이 필요하신가요?

여기에 없는 문제가 발생했거나 추가 지원이 필요하면 [hi@s.ee](mailto:hi@s.ee) 로 문의하시거나 [문의 페이지](https://s.ee/contact/)를 이용해 주세요. 도와드리겠습니다.
