## Konto- und Login-Probleme

| Error Message | Cause | Solution |
|---------------|-------|----------|
| "Invalid email or password. Please check your credentials and try again." | Falsche E-Mail oder falsches Passwort | Bitte überprüfen Sie Ihre E-Mail und Ihr Passwort. Passwörter unterscheiden Groß- und Kleinschreibung. |
| "We couldn't find an account with this email." | E-Mail nicht registriert | Bitte prüfen Sie Ihre E-Mail-Adresse oder erstellen Sie ein neues Konto. |
| "Your account has been suspended. Please contact support for assistance." | Konto gesperrt | Bitte kontaktieren Sie unser Support-Team unter [hi@s.ee](mailto:hi@s.ee). |
| "This login link has expired or is invalid. Please request a new one." | Login-Link abgelaufen | Bitte fordern Sie auf der Login-Seite einen neuen Login-Link an. |
| "This verification code has expired or is invalid. Please request a new one." | Bestätigungscode abgelaufen oder ungültig | Bitte fordern Sie einen neuen Bestätigungscode an. |
| "This email is already registered." | E-Mail wird bereits verwendet | Bitte verwenden Sie eine andere E-Mail-Adresse oder versuchen Sie, sich mit dieser E-Mail anzumelden. |
| "The current password you entered is incorrect." | Falsches aktuelles Passwort beim Ändern des Passworts | Bitte prüfen Sie Ihr aktuelles Passwort und versuchen Sie es erneut. |

**Hinweis:** Nur Ihre primäre E-Mail-Adresse funktioniert für die Anmeldung. Andere bestätigte E-Mail-Adressen in Ihrem Konto funktionieren nicht für die Authentifizierung. E-Mails zum Zurücksetzen des Passworts werden ebenfalls nur an Ihre primäre E-Mail-Adresse gesendet.

## Benutzername-Probleme

| Error Message | Cause | Solution |
|---------------|-------|----------|
| "This username is already taken. Please choose a different one." | Benutzername bereits vergeben | Bitte wählen Sie einen anderen Benutzernamen. |
| "Username can only be changed once." | Limit für Namensänderung erreicht | Benutzernamen können pro Konto leider nur einmal geändert werden. Diese Änderung ist dauerhaft. |

## Probleme mit Kurzlinks

| Error Message | Cause | Solution |
|---------------|-------|----------|
| "This slug is already taken. Please choose a different one." | Benutzerdefinierter Slug wird bereits verwendet | Bitte versuchen Sie einen anderen benutzerdefinierten Slug für Ihren Kurzlink. |
| "This slug cannot be used. Please choose a different one." | Slug ist vom System reserviert | Dieser Slug ist reserviert. Bitte wählen Sie einen anderen. |
| "This short link does not exist or is no longer available." | Kurzlink existiert nicht oder wurde gelöscht | Bitte prüfen Sie, ob die Link-URL korrekt ist. Der Link wurde möglicherweise gelöscht. |
| "The domain linked to this short URL is unavailable or no longer exists." | Domain ist nicht verfügbar | Bitte kontaktieren Sie den Support oder verwenden Sie eine andere Domain. |
| "Please enter a valid destination URL starting with http:// or https://" | Ungültiges URL-Format | Bitte stellen Sie sicher, dass Ihre Ziel-URL mit `http://` oder `https://` beginnt. |
| "This destination URL is not allowed. Please use a different link." | Ziel-URL ist blockiert | Diese URL befindet sich auf unserer Blockliste. Bitte verwenden Sie eine andere Ziel-URL. |
| "Incorrect password. Please try again." | Falsches Passwort für passwortgeschützten Link | Bitte geben Sie das richtige Passwort ein, um auf diesen Link zuzugreifen. |
| "URL has already been shortened." | URL wurde bereits gekürzt | Diese URL hat bereits einen bestehenden Kurzlink. |

## Probleme mit Abonnements & Tarifen

| Error Message | Cause | Solution |
|---------------|-------|----------|
| "You already have an active subscription plan." | Bereits abonniert | Wenn Sie den Tarif wechseln möchten, kündigen Sie bitte zuerst Ihr aktuelles Abonnement. |
| "You don't have an active subscription to cancel." | Kein aktives Abonnement | Sie haben derzeit kein aktives Abonnement, das gekündigt werden kann. |
| "This feature requires a premium subscription. Please upgrade your plan to access it." | Funktion erfordert höheren Tarif | Bitte aktualisieren Sie Ihren Tarif, um auf diese Funktion zuzugreifen. |
| "Unable to find your subscription plan. Please contact support if this issue persists." | Tarif nicht gefunden | Bitte kontaktieren Sie unser Support-Team unter [hi@s.ee](mailto:hi@s.ee), wenn das Problem bestehen bleibt. |
| "The selected pricing option is no longer available." | Preisoption abgelaufen | Bitte aktualisieren Sie die Seite und wählen Sie eine aktuell verfügbare Preisoption. |

## Probleme beim Datei-Upload

| Error Message | Cause | Solution |
|---------------|-------|----------|
| "File size exceeds limit" | Datei ist zu groß | Bitte komprimieren Sie Ihre Datei oder aktualisieren Sie Ihren Tarif für größere Uploads. |
| "Storage limit exceeded" | Speicherlimit erreicht | Bitte löschen Sie einige Dateien, um Speicherplatz freizugeben, oder aktualisieren Sie Ihren Tarif für mehr Speicher. |
| "Unsupported file type" / "File type not allowed" | Dateiformat wird nicht unterstützt | Bitte verwenden Sie ein unterstütztes Dateiformat. |
| "Maximum concurrent uploads exceeded" | Zu viele gleichzeitige Uploads | Bitte warten Sie, bis Ihre aktuellen Uploads abgeschlossen sind, bevor Sie neue starten. |
| "This file format is not supported. Allowed formats: JPG, JPEG, PNG, GIF, WEBP, AVIF." | Avatar-Format wird nicht unterstützt | Bitte laden Sie Ihren Avatar in einem der unterstützten Formate hoch: JPG, JPEG, PNG, GIF, WEBP oder AVIF. |

## Team-Probleme

| Error Message | Cause | Solution |
|---------------|-------|----------|
| "This feature is not available on your current plan." | Team-Funktion ist im Tarif nicht enthalten | Bitte aktualisieren Sie auf einen Tarif, der Team-Funktionen enthält. |
| "You have already created a team." | Team existiert bereits | Jeder Nutzer kann nur ein Team erstellen. Ihr bestehendes Team können Sie im Dashboard verwalten. |
| "You don't have permission to manage this team." | Unzureichende Berechtigungen | Bitte kontaktieren Sie den Eigentümer Ihres Teams. |
| "The invitation has already been processed." | Einladung bereits bearbeitet | Diese Einladung wurde bereits angenommen oder abgelehnt. |
| "Please wait at least 24 hours before resending." | Einladung zu früh erneut gesendet | Bitte warten Sie 24 Stunden, bevor Sie dieser Person erneut eine Einladung senden. |

## Probleme beim Löschen des Kontos

| Error Message | Cause | Solution |
|---------------|-------|----------|
| "Your team has other members, please delete the team first." | Team hat noch weitere Mitglieder | Bitte entfernen Sie alle Teammitglieder oder löschen Sie zuerst das Team, bevor Sie Ihr Konto löschen. |
| "Too many deletion requests, please try again later." | Zu viele Anfragen | Bitte warten Sie etwas und versuchen Sie es später erneut. |
| "Invalid or expired verification code." | Bestätigungscode ungültig oder abgelaufen | Bitte fordern Sie einen neuen Bestätigungscode an. |

## Probleme bei der E-Mail-Verwaltung

| Error Message | Cause | Solution |
|---------------|-------|----------|
| "Email already exists." | E-Mail bereits verwendet | Bitte verwenden Sie eine andere E-Mail-Adresse. |
| "Main email cannot be deleted." | Primäre E-Mail kann nicht gelöscht werden | Bitte legen Sie zuerst eine andere bestätigte E-Mail als primäre E-Mail fest, bevor Sie diese löschen. |
| "You need to close two factor verification first." | 2FA muss zuerst deaktiviert werden | Bitte deaktivieren Sie die Zwei-Faktor-Authentifizierung, bevor Sie Ihre primäre E-Mail ändern. |
| "This email has not been verified." | E-Mail wurde nicht bestätigt | Bitte bestätigen Sie diese E-Mail-Adresse zuerst. |
