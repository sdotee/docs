## Criar um link

1. Acesse [https://s.ee/user/links/](https://s.ee/user/links/)
2. Clique em **Create Link**
3. Informe a URL de destino
4. Configure as opções (opcional)
5. Clique em **Create**

---

## Opções do link

Ao criar um link, você pode configurar:

| Opção | Descrição |
|--------|-------------|
| Slug personalizado | Defina uma parte final personalizada para seu link (por exemplo, `s.ee/link`) |
| Domínio | Escolha entre domínios oficiais ou seus domínios personalizados |
| Senha | Exija uma senha para acessar o destino |
| Expiração | Defina uma data em que o link deixará de funcionar |
| Segmentação | Redirecione usuários com base em localização, dispositivo ou idioma |
| Etiquetas | Organize links com etiquetas |
| Parâmetros UTM | Adicione parâmetros de rastreamento de campanha |
| Encaminhamento de parâmetros | Encaminhe parâmetros da URL para o destino |

---

## Análise

Clique em qualquer link para ver sua análise:
- Total de cliques
- Cliques por localização
- Cliques por dispositivo e navegador
- Fontes de referência

---

## Gerenciar links

No painel de links, você pode:
- **Edit** - Atualizar a URL de destino ou as configurações
- **Delete** - Remover o link permanentemente
- **Copy** - Copiar o link curto para a área de transferência
