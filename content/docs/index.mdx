## What is S.EE?

S.EE is a link management platform for shortening URLs, generating QR codes, and sharing contentâ€”with built-in analytics and custom domain support. The platform includes:

- **[S.EE Links](https://s.ee/user/links/)** - Create short links with custom domains, password protection, and smart targeting
- **[S.EE QR Codes](https://s.ee/user/qr/)** - Generate customizable QR codes with analytics
- **[S.EE Text Sharing](https://s.ee/user/text/)** - Share plain text, code snippets, and markdown with short links

- **[S.EE Image Hosting](https://s.ee/user/images/)** - Upload and host images with short links

---

## Key Features

### Link Management

Create short links with:
- Custom slugs and domains
- Password protection
- Expiration dates
- Targeting by location, device, or language
- UTM parameters
- Parameter passing

### QR Codes

Generate QR codes with:
- Custom colors and logos
- Download in PNG, SVG, or PDF
- Built-in analytics

### Text Sharing

Share text content with:
- Plain text, code snippets, and markdown support
- Syntax highlighting for code
- Password protection and expiration
- Raw text endpoint for easy downloads

### File Sharing

Upload and share files with:
- Drag-and-drop, paste, or folder upload
- Custom domains and slugs
- Password protection for private files, and expiration
- Direct download links

### Image Hosting

Host images with:
- Drag-and-drop, paste, or folder upload
- Automatic image optimization when pasting from clipboard
- Direct image links for embedding

### Analytics

Track performance with:
- Click counts
- Geographic data
- Device and browser info
- Referrer sources
