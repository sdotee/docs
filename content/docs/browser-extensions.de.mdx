Die S.EE-Browser-Erweiterung bringt URL-Kürzung, Textfreigabe und Datei-Uploads direkt in Ihren Browser. Verfügbar für Chrome, Firefox und Edge.

Open Source auf [GitHub](https://github.com/sdotee/browser-extensions).

## Funktionen

| Feature | Description |
|---------|-------------|
| URL Shortening | Kürzen Sie beliebige URLs mit einem Klick, benutzerdefinierten Slugs und mehreren Domain-Optionen |
| Text Sharing | Teilen Sie Code-Snippets und Texte mit Syntaxhervorhebung |
| File Sharing | Laden Sie Dateien stapelweise hoch, inklusive Fortschrittsanzeige |
| Export Formats | Kopieren Sie Ergebnisse als Plain URL, Markdown, HTML oder BBCode |
| QR Code Generation | Erstellen Sie QR-Codes für jede gekürzte URL |
| QR Code Export | Laden Sie QR-Codes als PNG, SVG oder PDF herunter (512x512) |
| Right-Click Menu | Seite/Link kürzen, QR anzeigen, markierten Text teilen, Bilder hochladen |
| History Management | Tab-bezogene Historie für URLs, Texte und Dateien mit Stapellöschung |
| Entwurfspeicherung | Eingabefelder werden sitzungsübergreifend automatisch gespeichert |
| Automatisches Kopieren | Ergebnisse automatisch in die Zwischenablage kopieren |
| Hell-/Dunkelmodus | Automatische Theme-Erkennung mit manuellem Umschalten |
| Desktop-Benachrichtigungen | Benachrichtigungen über abgeschlossene Aktionen per Kontextmenü |
| Cross-Browser | Funktioniert in Chrome (MV3), Firefox (MV3) und Edge |
