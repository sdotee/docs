L'estensione del browser di S.EE porta l'accorciamento URL, la condivisione di testo e il caricamento di file direttamente nel browser. È disponibile per Chrome, Firefox ed Edge.

Codice open source su [GitHub](https://github.com/sdotee/browser-extensions).

## Funzionalità

| Funzionalità | Descrizione |
|---------|-------------|
| Accorciamento URL | Accorcia qualsiasi URL con un clic, slug personalizzati e più domini |
| Condivisione di testo | Condividi testo e codice con evidenziazione sintattica |
| Condivisione file | Carica file in batch con tracciamento del progresso |
| Formati di esportazione | Copia i risultati in formato URL, Markdown, HTML o BBCode |
| QR code | Genera QR code per qualsiasi URL abbreviato |
| Esportazione QR | Scarica i QR code in PNG, SVG o PDF (512x512) |
| Menu contestuale | Accorcia pagina o link, mostra QR, condividi testo selezionato o carica immagini |
| Gestione cronologia | Cronologia distinta per scheda per URL, testi e file con eliminazione in batch |
| Persistenza bozze | I campi di input vengono salvati automaticamente tra una sessione e l'altra |
| Copia automatica | Copia automaticamente i risultati negli appunti |
| Modalità chiara/scura | Rilevamento automatico del tema con interruttore manuale |
| Notifiche desktop | Ricevi una notifica quando un'azione viene completata dal menu contestuale |
| Compatibilità multi-browser | Funziona con Chrome (MV3), Firefox (MV3) ed Edge |
