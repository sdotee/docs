L'estensione del browser di S.EE porta l'accorciamento URL, la condivisione di testo e il caricamento file direttamente nel browser.

Codice open source su [GitHub](https://github.com/sdotee/browser-extensions).

## Funzionalità

| Funzionalità | Descrizione |
|---------|-------------|
| Accorciamento URL | Accorcia qualsiasi URL con un clic, slug personalizzati e più domini |
| Condivisione di testo | Condividi testo e codice con evidenziazione sintattica |
| Condivisione file | Carica file in batch con tracciamento del progresso |
| Formati di esportazione | Copia i risultati in formato URL, Markdown, HTML o BBCode |
| QR code | Genera QR code per qualsiasi URL abbreviato |
| Esportazione QR | Scarica i QR code in PNG, SVG o PDF |
| Menu contestuale | Accorcia pagina o link, mostra QR, condividi testo selezionato o carica immagini |
| Gestione cronologia | Cronologia per URL, testi e file con eliminazione in batch |
| Copia automatica | Copia automaticamente i risultati negli appunti |
