## Hallintapaneelin yleiskuva

Kirjautumisen jälkeen pääset [hallintapaneeliin](https://s.ee/user/dashboard/). S.EE-hallintapaneelissa on kaksi pääasiallista navigointialuetta:

### Vasen sivupalkki

Pääset tärkeimpiin ominaisuuksiisi:

| Ominaisuus | Kuvaus |
|---------|-------------|
| [Linkit](https://s.ee/user/links/) | Hallitse lyhytlinkkejäsi |
| [QR-koodit](https://s.ee/user/qr/) | Luo ja hallitse QR-koodeja |
| [Tekstin jakaminen](https://s.ee/user/text/) | Jaa tekstiä, koodikatkelmia ja Markdownia |
| [Tiedostojen jakaminen](https://s.ee/user/files/) | Lataa ja jaa tiedostoja |
| [Verkkotunnukset](https://s.ee/user/domains/) | Määritä mukautettuja verkkotunnuksia |
| [Analytiikka](https://s.ee/user/analytics/) | Tarkastele klikkaus- ja skannaustilastoja |
| [Tiimi](https://s.ee/user/team/) | Hallitse tiimin jäseniä |
| [UTM-rakentaja](https://s.ee/user/library/utm/) | Luo ja hallitse UTM-parametreja |
| [Käyttö](https://s.ee/user/usage/) | Seuraa pakettisi käyttöä |
| [Asetukset](https://s.ee/user/settings/) | Tilin ja laskutuksen asetukset |

Mobiililaitteilla sivupalkki supistuu automaattisesti.

### Profiilivalikko

Se sijaitsee vasemmassa sivupalkissa. Voit sillä:

- Siirtyä dokumentaatioon
- [Hallita API-avaimia](https://s.ee/user/developers/)
- [Päivittää pakettisi](https://s.ee/user/pricing/)
- Vaihtaa tilien välillä
- Kirjautua ulos
