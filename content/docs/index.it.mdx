## Che cos'è S.EE?

S.EE è una piattaforma di gestione dei link per accorciare URL, generare QR code e condividere contenuti, con analisi integrate e supporto per domini personalizzati. La piattaforma include:

- **[S.EE Links](https://s.ee/user/links/)** - Crea link brevi con domini personalizzati, protezione con password e targeting intelligente
- **[S.EE QR Codes](https://s.ee/user/qr/)** - Genera QR code personalizzabili con analisi integrate
- **[S.EE Text Sharing](https://s.ee/user/text/)** - Condividi testo semplice, snippet di codice e Markdown con link brevi
- **[S.EE File Sharing](https://s.ee/user/files/)** - Carica e condividi file con link brevi
- **[S.EE Image Hosting](https://s.ee/user/images/)** - Carica e ospita immagini con link brevi

---

## Funzionalità principali

### Gestione dei link

Crea link brevi con:

- Slug e domini personalizzati
- Protezione con password
- Date di scadenza
- Targeting per posizione, dispositivo o lingua
- Parametri UTM
- Inoltro dei parametri

### QR code

Genera QR code con:

- Colori e loghi personalizzati
- Download in PNG, SVG o PDF
- Analisi integrate

### Condivisione di testo

Condividi contenuti testuali con:

- Supporto per testo semplice, snippet di codice e Markdown
- Evidenziazione della sintassi per il codice
- Protezione con password e scadenza
- Endpoint raw text per download più semplici

### Condivisione di file

Carica e condividi file con:

- Drag-and-drop, incolla o caricamento di cartelle
- Domini e slug personalizzati
- Protezione con password per file privati e scadenza
- Link di download diretto

### Hosting immagini

Ospita immagini con:

- Drag-and-drop, incolla o caricamento di cartelle
- Ottimizzazione automatica delle immagini quando incolli dagli appunti
- Link diretti alle immagini da incorporare

### Analisi

Monitora le prestazioni con:

- Numero di clic
- Dati geografici
- Informazioni su dispositivi e browser
- Fonti di provenienza
