## Buat Tautan

1. Buka [https://s.ee/user/links/](https://s.ee/user/links/)
2. Klik **Buat Tautan**
3. Masukkan URL tujuan Anda
4. Atur opsi (opsional)
5. Klik **Buat**

---

## Opsi Tautan

Saat membuat tautan, Anda dapat mengatur:

| Option | Description |
|--------|-------------|
| Custom slug | Atur bagian belakang tautan Anda (misalnya `s.ee/link`) |
| Domain | Pilih dari domain resmi atau domain kustom Anda |
| Password | Wajibkan kata sandi untuk mengakses tujuan |
| Expiration | Atur tanggal saat tautan berhenti berfungsi |
| Targeting | Arahkan pengguna berdasarkan lokasi, perangkat, atau bahasa |
| Tags | Atur tautan dengan tag |
| UTM parameters | Tambahkan parameter pelacakan kampanye |
| Parameter passing | Teruskan parameter URL ke tujuan |

---

## Analitik

Klik salah satu tautan untuk melihat analitiknya:

- Total klik
- Klik berdasarkan lokasi
- Klik berdasarkan perangkat dan browser
- Sumber perujuk

---

## Kelola Tautan

Dari dasbor tautan, Anda dapat:

- **Ubah** - Perbarui URL tujuan atau pengaturan
- **Hapus** - Hapus tautan secara permanen
- **Salin** - Salin tautan pendek ke clipboard
