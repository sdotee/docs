## Tilin ja kirjautumisen ongelmat

| Virheilmoitus | Syy | Ratkaisu |
|---------------|-------|----------|
| "Invalid email or password. Please check your credentials and try again." | Virheellinen sähköpostiosoite tai salasana | Tarkista sähköpostiosoite ja salasana huolellisesti. |
| "We couldn't find an account with this email." | Sähköpostia ei ole rekisteröity | Tarkista sähköpostiosoite tai luo uusi tili. |
| "Your account has been suspended. Please contact support for assistance." | Tili on jäädytetty | Ota yhteyttä tukitiimiimme osoitteessa [hi@s.ee](mailto:hi@s.ee). |
| "This login link has expired or is invalid. Please request a new one." | Kirjautumislinkki on vanhentunut | Pyydä uusi kirjautumislinkki kirjautumissivulta. |
| "This verification code has expired or is invalid. Please request a new one." | Vahvistuskoodi on vanhentunut tai virheellinen | Pyydä uusi vahvistuskoodi. |
| "This email is already registered." | Sähköposti on jo käytössä | Käytä toista sähköpostiosoitetta tai yritä kirjautua sisään tällä osoitteella. |
| "The current password you entered is incorrect." | Nykyinen salasana on väärä salasanan vaihtamisen yhteydessä | Tarkista nykyinen salasanasi ja yritä uudelleen. |

**Huom:** Vain ensisijainen sähköpostiosoitteesi toimii kirjautumisessa. Muut tilillesi vahvistetut sähköpostit eivät kelpaa tunnistautumiseen. Myös salasanan palautusviestit lähetetään vain ensisijaiseen sähköpostiosoitteeseen.

## Lyhyiden linkkien ongelmat

| Virheilmoitus | Syy | Ratkaisu |
|---------------|-------|----------|
| "This slug is already taken. Please choose a different one." | Mukautettu slug on jo käytössä | Kokeile toista slugia lyhyelle linkillesi. |
| "This slug cannot be used. Please choose a different one." | Järjestelmä on varannut slugin | Slug on varattu. Valitse toinen. |
| "This short link does not exist or is no longer available." | Lyhyttä linkkiä ei ole olemassa tai se on poistettu | Tarkista, että linkin URL on oikein. Linkki on voitu poistaa. |
| "The domain linked to this short URL is unavailable or no longer exists." | Verkkotunnus ei ole käytettävissä | Ota yhteyttä tukeen tai kokeile toista verkkotunnusta. |
| "Please enter a valid destination URL starting with http:// or https://" | Kohde-URL on virheellisessä muodossa | Varmista, että kohde-URL alkaa `http://` tai `https://`. |
| "This destination URL is not allowed. Please use a different link." | Kohde-URL on estolistalla | Tämä URL on estolistalla. Käytä toista kohde-URL:ää. |
| "Incorrect password. Please try again." | Salasana on väärä salasanasuojatulle linkille | Syötä oikea salasana päästäksesi linkkiin. |
| "URL has already been shortened." | URL on jo lyhennetty | Tälle URL:lle on jo olemassa lyhyt linkki. |

## Tiedoston latausongelmat

| Virheilmoitus | Syy | Ratkaisu |
|---------------|-------|----------|
| "File size exceeds limit" | Tiedosto on liian suuri | Pakkaa tiedostosi tai päivitä suunnitelmaasi suurempia latauksia varten. |
| "Storage limit exceeded" | Tallennuskiintiö on täynnä | Poista joitakin tiedostoja vapauttaaksesi tilaa tai päivitä suunnitelmaasi saadaksesi lisää tallennustilaa. |
| "Unsupported file type" / "File type not allowed" | Tiedostomuotoa ei tueta | Käytä tuettua tiedostomuotoa. |
| "Maximum concurrent uploads exceeded" | Liian monta samanaikaista latausta | Odota, että nykyiset lataukset valmistuvat ennen uusien aloittamista. |
| "This file format is not supported. Allowed formats: JPG, JPEG, PNG, GIF, WEBP, AVIF." | Avatarin tiedostomuotoa ei tueta | Lataa avatar jossakin tuetuista muodoista: JPG, JPEG, PNG, GIF, WEBP tai AVIF. |
