---

La [página de subida de imágenes](https://s.ee/user/images/) permite subir imágenes en lote. Arrastre y suelte archivos o carpetas, pegue desde el portapapeles o haga clic para seleccionar archivos. Las imágenes pegadas se convierten automáticamente a WebP (o a PNG si su navegador no es compatible con WebP).

Los archivos subidos se ordenan por nombre de archivo, para que los lotes mantengan el orden.

---

## Códigos para compartir

Cada imagen recibe un enlace para compartir. Puede copiar los enlaces uno por uno o tomar un código de uso compartido para todo el lote.

Formatos: URL directa, Markdown, HTML, BBCode.

---

## Gestión de archivos

La página de imágenes es solo para subidas. Para eliminar imágenes o gestionar archivos, vaya a [s.ee/user/files/](https://s.ee/user/files/).
