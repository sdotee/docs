[ShareX](https://getsharex.com/) adalah alat screen capture dan berbagi file open source untuk Windows. S.EE menyediakan konfigurasi pengunggah kustom untuk ShareX, sehingga Anda dapat mengunggah screenshot, file, teks, dan kode langsung ke S.EE lalu mendapatkan tautan berbagi.

Lihat juga: [Dokumentasi pengunggah kustom ShareX](https://getsharex.com/docs/custom-pengunggah)

---

## Pengunggah kustom

S.EE memiliki konfigurasi untuk jenis unggahan berikut:

| Uploader | Fungsinya |
|----------|-------------|
| Pemendek URL | Memendekkan URL melalui S.EE |
| Pengunggah teks | Mengunggah cuplikan teks |
| Pengunggah kode | Mengunggah kode dengan sorotan sintaks |
| Pengunggah Markdown | Mengunggah Markdown dengan pratinjau yang dirender |
| Pengunggah gambar dan file | Mengunggah gambar dan file |

---

## Instalasi

Ambil file konfigurasi `.sxcu` di bawah ini. Anda dapat mengunduhnya atau mengimpornya melalui URL di ShareX.

| Uploader | Tautan unduh / impor |
|----------|------------------------|

### Impor

1. Buka **ShareX**
2. Buka **Tujuan** > **Pengaturan pengunggah kustom**
3. Klik **Impor**, lalu pilih salah satu:
   - Pilih file `.sxcu` yang sudah diunduh
   - Tempelkan URL unduhan secara langsung

![Pengaturan pengunggah kustom ShareX](https://i.see.you/2026/01/18/5dkS/image.webp)

## Siapkan API key Anda

1. Dapatkan API key Anda dari **[Alat > API Token](https://s.ee/user/developers/)**
2. Di ShareX, buka **Pengaturan pengunggah kustom** lalu pilih pengunggah S.EE yang Anda impor
3. Pada bagian **Header**, isi nilai `Authorization` dengan API key Anda

<Callout type="warn">
Jangan membagikan API key Anda secara publik atau menyimpannya ke version control.
</Callout>

## Penggunaan

### Pilih pengunggah

Di ShareX, buka **Tujuan** dan tetapkan pengunggah S.EE:

- Pemendek URL → S.EE URL Shortener
- Pengunggah teks → S.EE Text Uploader, Code Uploader, atau Markdown Uploader
- Pengunggah gambar → S.EE File Uploader
- Pengunggah file → S.EE File Uploader

### Unggah

Gunakan hotkey atau menu klik kanan ShareX untuk memendekkan URL, mengunggah screenshot, membagikan cuplikan teks, dan sebagainya. ShareX akan mengunggah ke S.EE dan menyalin tautan ke clipboard Anda.

---

## Pemecahan Masalah

| Masalah | Solusi |
|-------|-----|
| Error 401 | Pastikan API key Anda telah diatur dengan benar di header Authorization |
| 403 error | Pastikan API key Anda memiliki izin yang tepat |
| File rejected | Jenis file mungkin tidak didukung. Lihat [Berbagi File](/docs/id/file-sharing) untuk tipe file yang diizinkan |

Anda dapat mengklik **Uji** di **Pengaturan pengunggah kustom** untuk memastikan semuanya bekerja dengan benar.
