## 什麼是 S.EE？

S.EE 是一個連結管理平台，提供短網址生成、QR Code 生成和內容分享功能，並內建資料分析與自訂網域支援。平台包含：

- **[S.EE 短網址](https://s.ee/user/links/)** - 建立支援自訂網域、密碼保護和智慧定向的短網址
- **[S.EE QR Code](https://s.ee/user/qr/)** - 生成可自訂且附帶資料分析的 QR Code
- **[S.EE 文字分享](https://s.ee/user/text/)** - 透過短網址分享純文字、程式碼片段和 Markdown
- **[S.EE 檔案分享](https://s.ee/user/files/)** - 透過短網址上傳和分享檔案
- **[S.EE 圖床](https://s.ee/user/images/)** - 透過短網址上傳並託管圖片

---

## 主要功能

### 短網址管理

建立短網址時，您可以使用：
- 自訂短碼和自訂網域
- 密碼保護
- 過期時間
- 按地區、裝置或語言進行定向
- UTM 參數
- 參數傳遞

### QR Code

生成 QR Code 時，您可以使用：
- 自訂顏色和 Logo
- 下載為 PNG、SVG 或 PDF
- 內建資料分析

### 文字分享

分享文字內容時，您可以使用：
- 純文字、程式碼片段和 Markdown
- 程式碼語法高亮
- 密碼保護與過期時間
- 原始文字端點，便於下載

### 檔案分享

上傳和分享檔案時，您可以使用：
- 拖拽、貼上或上傳資料夾
- 自訂網域和自訂短碼
- 為私有檔案設定密碼和過期時間
- 直接下載連結

### 圖床

託管圖片時，您可以使用：
- 拖拽、貼上或上傳資料夾
- 從剪貼簿貼上時自動最佳化圖片
- 直接圖片連結便於嵌入

### 資料分析

追蹤表現，包括：
- 點擊次數
- 地理位置資料
- 裝置與瀏覽器資訊
- 來源
