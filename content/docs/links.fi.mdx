## Luo lyhyt linkki

1. Avaa [https://s.ee/user/links/](https://s.ee/user/links/)
2. Napsauta **Luo linkki**
3. Syötä kohde-URL
4. Määritä asetukset (valinnainen)
5. Napsauta **Luo**

---

## Linkin asetukset

Kun luot linkkiä, voit määrittää:

| Asetus | Kuvaus |
|--------|--------|
| Custom slug | Aseta linkille oma loppuosa, esimerkiksi `s.ee/link` |
| Domain | Valitse virallisista verkkotunnuksista tai omista verkkotunnuksistasi |
| Password | Vaadi salasana kohteeseen pääsyyn |
| Expiration | Määritä päivämäärä, jolloin linkki lakkaa toimimasta |
| Targeting | Ohjaa käyttäjiä sijainnin, laitteen tai kielen mukaan |
| Tags | Järjestä linkit tunnisteilla |
| UTM parameters | Lisää kampanjaseurannan parametreja |
| Parameter passing | Välitä URL-parametrit kohteelle |

---

## Analytiikka

Voit avata minkä tahansa linkin analytiikan napsauttamalla sitä:
- Kokonaisklikkaukset
- Klikkaukset sijainnin mukaan
- Klikkaukset laitteen ja selaimen mukaan
- Viittaavat lähteet

---

## Linkkien hallinta

Linkkien hallintanäkymästä voit:
- **Muokkaa** - Päivitä kohde-URL tai asetukset
- **Poista** - Poista linkki pysyvästi
- **Kopioi** - Kopioi lyhyt linkki leikepöydälle
