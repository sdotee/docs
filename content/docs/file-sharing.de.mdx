## Überblick

Mit S.EE File Sharing können Sie Dateien über einen kurzen, leicht teilbaren Link hochladen und teilen. Ideal zum Teilen von Bildern, Dokumenten, Audio, Videos und anderen Dateien.

---

## Dateien hochladen

### Schneller Upload (Drag & Drop)

1. Melden Sie sich bei [https://s.ee/user/dashboard](https://s.ee/user/dashboard) an
2. Klicken Sie in der linken Seitenleiste auf **Sharing > Files** oder gehen Sie direkt zu [https://s.ee/user/files/](https://s.ee/user/files/)
3. Ziehen Sie Dateien oder Ordner in das **Drag & Drop**-Fenster
   - Sie können auch auf das Fenster klicken, um Dateien auszuwählen
   - Beim Hochladen eines Ordners werden alle darin enthaltenen Dateien hochgeladen
4. Dateien werden mit der Standard-Domain und einem zufälligen Slug geteilt

### Erweiterter Upload (Datei erstellen)

Für mehr Kontrolle über Ihre Dateifreigabe:

1. Klicken Sie auf **Datei erstellen**
2. Konfigurieren Sie Ihre Einstellungen:
   - **Domain** - Wählen Sie Ihre bevorzugte Domain für die Freigabeseite
   - **Custom Slug** - Legen Sie einen benutzerdefinierten URL-Slug fest
   - **Other options** - Tags, Ablaufdatum usw.
3. Wählen Sie Ihre Datei und laden Sie sie hoch

## Dateilinks

Jede hochgeladene Datei hat zwei Arten von Links:

| Link Type | Format | Description |
|-----------|--------|-------------|
| Share Page | `fs.to/[your_slug]` | Domain und Slug können angepasst werden |
| Direct Link | `i.see.you/[yyyy]/[mm]/[dd]/[random_slug]/[filename].[ext]` | Direkter Zugriff auf die Datei mit dem ursprünglichen Dateinamen |

**Hinweis:** Direct Links unterstützen derzeit keine benutzerdefinierten Domains.

## Dateiaufbewahrung

- Dateien werden **dauerhaft gespeichert**, solange Ihr Abonnement aktiv ist
- Wenn Ihr Abonnement endet, werden Dateien vor dem Löschen **7 Tage** lang aufbewahrt

## FAQ

### Wer kann Dateien hochladen?

Nur zahlende Abonnenten können Dateien hochladen. Ein aktives Abonnement ist erforderlich.

### Gibt es Download-Limits?

Nein, es gibt weder Limits für die Anzahl der Downloads noch für die Bandbreite geteilter Dateien.

### Kann ich die URL der Freigabeseite anpassen?

Ja, Sie können eine benutzerdefinierte Domain und einen Slug für die Freigabeseite festlegen, wenn Sie die Upload-Methode **Datei erstellen** verwenden.

### Kann ich die URL des Direct Links anpassen?

Derzeit verwenden Direct Links ein festes Format und unterstützen keine benutzerdefinierten Domains.
