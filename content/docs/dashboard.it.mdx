Dopo aver effettuato l'accesso a S.EE, verrai portato alla dashboard: [https://s.ee/user/dashboard/](https://s.ee/user/dashboard/)

Qui puoi:

- Creare e gestire link brevi
- Visualizzare le analisi
- Caricare file e immagini
- Pubblicare testo e Markdown

---

## Navigazione principale

| Sezione | Descrizione |
|------|--------|
| Dashboard | Panoramica di storage e utilizzo |
| Links | Creazione, modifica e analisi dei link brevi |
| Texts | Condivisione di testo, Markdown e codice |
| Files | Upload e gestione di file e immagini |
| QR Codes | Creazione ed esportazione di QR code dinamici |
| Domains | Gestione dei domini personalizzati |
| Developers | API key, strumenti e integrazioni |

---

## Utilizzo e statistiche

Nella dashboard puoi vedere anche:

- utilizzo dello storage
- conteggi giornalieri e mensili
- limiti per API, link, testi, upload e QR code
