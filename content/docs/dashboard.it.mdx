## Panoramica della dashboard

Accedi alla [dashboard](https://s.ee/user/dashboard/) dopo aver effettuato l'accesso. La dashboard di S.EE ha due aree di navigazione principali:

### Barra laterale sinistra

Da qui accedi alle funzioni principali:

| Funzionalita | Descrizione |
|---------|-------------|
| [Link](https://s.ee/user/links/) | Gestisci i tuoi link brevi |
| [Codici QR](https://s.ee/user/qr/) | Crea e gestisci i codici QR |
| [Condivisione testo](https://s.ee/user/text/) | Condividi testo, snippet di codice e Markdown |
| [Condivisione file](https://s.ee/user/files/) | Carica e condividi file |
| [Domini](https://s.ee/user/domains/) | Configura domini personalizzati |
| [Analisi](https://s.ee/user/analytics/) | Visualizza le statistiche di clic e scansioni |
| [Team](https://s.ee/user/team/) | Gestisci i membri del team |
| [UTM Builder](https://s.ee/user/library/utm/) | Crea e gestisci i parametri UTM |
| [Utilizzo](https://s.ee/user/usage/) | Monitora l'utilizzo del piano |
| [Impostazioni](https://s.ee/user/settings/) | Gestisci account e fatturazione |

Sui dispositivi mobili, la barra laterale si comprime automaticamente.

### Menu del profilo

Si trova nella barra laterale sinistra. Puoi usarlo per:

- Accedere alla documentazione
- [Gestire le API key](https://s.ee/user/developers/)
- [Passare a un piano superiore](https://s.ee/user/pricing/)
- Passare da un account all'altro
- Disconnetterti
