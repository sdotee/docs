## 控制台概览

登录后即可访问您的 [控制台](https://s.ee/user/dashboard/)。S.EE 控制台主要分为两个导航区域：

### 左侧边栏

访问主要功能：

| 功能 | 说明 |
|---------|-------------|
| [短网址](https://s.ee/user/links/) | 管理您的短网址 |
| [二维码](https://s.ee/user/qr/) | 创建和管理二维码 |
| [文本分享](https://s.ee/user/text/) | 分享文本、代码片段和 Markdown |
| [文件分享](https://s.ee/user/files/) | 上传并分享文件 |
| [域名](https://s.ee/user/domains/) | 配置自定义域名 |
| [数据分析](https://s.ee/user/analytics/) | 查看点击和扫码统计 |
| [团队](https://s.ee/user/team/) | 管理团队成员 |
| [UTM Builder](https://s.ee/user/library/utm/) | 创建和管理 UTM 参数 |
| [用量](https://s.ee/user/usage/) | 监控您的套餐用量 |
| [设置](https://s.ee/user/settings/) | 账号和账单设置 |

在移动设备上，侧边栏会自动折叠。

### 个人资料菜单

位于左侧边栏，可用于：

- 访问文档
- [管理 API 密钥](https://s.ee/user/developers/)
- [升级套餐](https://s.ee/user/pricing/)
- 切换账号
- 退出登录
