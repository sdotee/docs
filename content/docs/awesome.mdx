## Plugins

| Name | Description | Link |
|------|-------------|------|
| **S.EE for WordPress** | URL shortening, text sharing, and file hosting for WordPress. Auto-generates short URLs on publish, uploads media to S.EE, and tracks everything in a dashboard. | [WordPress.org](https://wordpress.org/plugins/sdotee/) · [GitHub](https://github.com/sdotee/see-wordpress) · [Docs](/docs/wordpress) |
| **S.EE for Raycast** | URL shortening, text sharing, and file uploading right from Raycast. Auto-detects clipboard content, supports custom slugs, passwords, expiry dates, and browsable history. | [Raycast Store](https://www.raycast.com/missuo/sdotee) · [GitHub](https://github.com/missuo/raycast-sdotee) |
| **Typecho-Plugin-PicUp** | A multi-backend image and attachment upload plugin for Typecho. Supports S.EE image uploads alongside other storage backends, making it easy to upload and manage media from your Typecho site. | [GitHub](https://github.com/lhl77/Typecho-Plugin-PicUp) · [Homepage](https://blog.lhl.one/artical/1026.html) |
