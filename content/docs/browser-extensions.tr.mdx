S.EE tarayıcı uzantısı, URL kısaltmayı, metin paylaşımını ve dosya yüklemeyi doğrudan tarayıcınıza taşır.

Kaynak kodu [GitHub](https://github.com/sdotee/browser-extensions) üzerinde açıktır.

## Özellikler

| Özellik | Açıklama |
|---------|-------------|
| URL kısaltma | Tek tıklamayla URL kısaltın, özel slug ve farklı alan adları kullanın |
| Metin paylaşımı | Kod parçalarını ve metni sözdizimi vurgusuyla paylaşın |
| Dosya paylaşımı | İlerleme takibi ile toplu dosya yükleyin |
| Dışa aktarma biçimleri | Sonucu düz URL, Markdown, HTML veya BBCode olarak kopyalayın |
| QR kod üretimi | Her kısa bağlantı için QR kodu oluşturun |
| QR kod dışa aktarma | QR kodlarını PNG, SVG veya PDF olarak indirin (512x512) |
| Sağ tık menüsü | Sayfa veya bağlantıyı kısaltın, QR kodu gösterin, seçili metni paylaşın, görsel yükleyin |
| Geçmiş yönetimi | URL, metin ve dosyalar için sekme bazlı geçmiş ve toplu silme |
| Taslakları koruma | Girdi alanları oturumlar arasında otomatik olarak kaydedilir |
| Otomatik kopyalama | Sonuçları otomatik olarak panoya kopyalayın |
| Açık / koyu tema | Otomatik tema algılama ve manuel geçiş |
| Masaüstü bildirimleri | Bağlam menüsünden başlatılan işlemler tamamlandığında bildirim alın |
| Tarayıcılar arası destek | Chrome (MV3), Firefox (MV3) ve Edge üzerinde çalışır |
