A extensão de navegador da S.EE leva o encurtamento de URL, a partilha de texto e o carregamento de ficheiros diretamente para o seu navegador. Disponível para Chrome, Firefox e Edge.

Código aberto em [GitHub](https://github.com/sdotee/browser-extensions).

## Funcionalidades

| Funcionalidade | Descrição |
|---------|-------------|
| Encurtamento de URL | Encurte qualquer URL com um clique, slugs personalizados e várias opções de domínio |
| Partilha de texto | Partilhe fragmentos de código e texto com suporte para realce de sintaxe |
| Partilha de ficheiros | Carregue ficheiros em lote com acompanhamento do progresso |
| Formatos de exportação | Copie resultados em URL simples, Markdown, HTML ou BBCode |
| Geração de código QR | Gere códigos QR para qualquer URL encurtado |
| Exportação de código QR | Transfira códigos QR em PNG, SVG ou PDF (512x512) |
| Menu de contexto | Encurte página / link, mostre QR, partilhe texto selecionado, carregue imagens |
| Gestão de histórico | Histórico por separador para URLs, textos e ficheiros com eliminação em lote |
| Persistência de rascunhos | Os campos de introdução são guardados automaticamente entre sessões |
| Cópia automática | Copie automaticamente os resultados para a área de transferência |
| Modo escuro / claro | Deteção automática do tema com comutação manual |
| Notificações de desktop | Receba notificações quando as ações forem concluídas através do menu de contexto |
| Multinavegador | Funciona em Chrome (MV3), Firefox (MV3) e Edge |
