## Vue d’ensemble du tableau de bord

Accédez à votre [tableau de bord](https://s.ee/user/dashboard/) après vous être connecté. Le tableau de bord S.EE comporte deux zones de navigation principales :

### Barre latérale gauche

Accédez à vos principales fonctionnalités :

| Fonctionnalité | Description |
|---------|-------------|
| [Links](https://s.ee/user/links/) | Gérez vos liens courts |
| [QR Codes](https://s.ee/user/qr/) | Créez et gérez des codes QR |
| [Text Sharing](https://s.ee/user/text/) | Partagez du texte, des extraits de code et du Markdown |
| [File Sharing](https://s.ee/user/files/) | Téléversez et partagez des fichiers |
| [Domains](https://s.ee/user/domains/) | Configurez des domaines personnalisés |
| [Analytics](https://s.ee/user/analytics/) | Consultez les statistiques de clics et de scans |
| [Team](https://s.ee/user/team/) | Gérez les membres de l’équipe |
| [UTM Builder](https://s.ee/user/library/utm/) | Créez et gérez des paramètres UTM |
| [Usage](https://s.ee/user/usage/) | Surveillez l’utilisation de votre offre |
| [Settings](https://s.ee/user/settings/) | Paramètres du compte et de facturation |

Sur les appareils mobiles, la barre latérale se replie automatiquement.

### Menu du profil

Situé dans la barre latérale gauche. Il vous permet de :
- Accéder à la documentation
- [Gérer les clés API](https://s.ee/user/developers/)
- [Mettre à niveau votre offre](https://s.ee/user/pricing/)
- Basculer entre plusieurs comptes
- Vous déconnecter
