## Een link maken

1. Ga naar [https://s.ee/user/links/](https://s.ee/user/links/)
2. Klik op **Create Link**
3. Voer uw doel-URL in
4. Stel opties in indien nodig
5. Klik op **Create**

---

## Linkopties

Bij het maken van een link kunt u deze opties instellen:

| Optie | Beschrijving |
|--------|-------------|
| Custom slug | Stel een aangepaste slug in voor uw link, bijvoorbeeld `s.ee/link` |
| Domain | Kies uit officiele domeinen of uw aangepaste domeinen |
| Password | Vereis een wachtwoord om de bestemming te openen |
| Expiration | Stel een datum in waarop de link stopt met werken |
| Targeting | Stuur gebruikers door op basis van locatie, apparaat of taal |
| Tags | Organiseer links met tags |
| UTM parameters | Voeg parameters voor campagnetracking toe |
| Parameter passing | Geef URL-parameters door aan de bestemming |

---

## Analytics

Klik op een link om de analytics te bekijken:

- Totaal aantal klikken
- Klikken per locatie
- Klikken per apparaat en browser
- Verwijzingsbronnen

---

## Links beheren

Vanuit het linkdashboard kunt u:

- **Edit** - De doel-URL of instellingen bijwerken
- **Delete** - De link permanent verwijderen
- **Copy** - De korte link naar het klembord kopieren
