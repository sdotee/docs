## Masalah Akun & Login

| Error Message | Cause | Solution |
|---------------|-------|----------|
| "Invalid email or password. Please check your credentials and try again." | Email atau kata sandi salah | Periksa kembali email dan kata sandi Anda. Kata sandi peka huruf besar/kecil. |
| "We couldn't find an account with this email." | Email belum terdaftar | Verifikasi alamat email Anda atau buat akun baru. |
| "Your account has been suspended. Please contact support for assistance." | Akun ditangguhkan | Hubungi tim dukungan kami di [hi@s.ee](mailto:hi@s.ee). |
| "This login link has expired or is invalid. Please request a new one." | Tautan login kedaluwarsa | Minta tautan login baru dari halaman login. |
| "This verification code has expired or is invalid. Please request a new one." | Kode verifikasi kedaluwarsa atau tidak valid | Minta kode verifikasi baru. |
| "This email is already registered." | Email sudah digunakan | Gunakan alamat email lain atau coba login dengan email ini. |
| "The current password you entered is incorrect." | Kata sandi saat ini salah saat mengganti kata sandi | Verifikasi kata sandi saat ini lalu coba lagi. |

**Catatan:** Hanya alamat email utama Anda yang dapat digunakan untuk login. Email terverifikasi lain di akun Anda tidak dapat digunakan untuk autentikasi. Email reset kata sandi juga hanya dikirim ke email utama Anda.

## Masalah Tautan Pendek

| Error Message | Cause | Solution |
|---------------|-------|----------|
| "This slug is already taken. Please choose a different one." | Slug kustom sudah digunakan | Coba gunakan slug kustom lain untuk tautan pendek Anda. |
| "This slug cannot be used. Please choose a different one." | Slug dicadangkan oleh sistem | Slug ini dicadangkan. Silakan pilih slug lain. |
| "This short link does not exist or is no longer available." | Tautan pendek tidak ada atau telah dihapus | Pastikan URL tautan benar. Tautan mungkin sudah dihapus. |
| "The domain linked to this short URL is unavailable or no longer exists." | Domain tidak tersedia | Hubungi dukungan atau coba gunakan domain lain. |
| "Please enter a valid destination URL starting with http:// or https://" | Format URL tidak valid | Pastikan URL tujuan Anda diawali dengan `http://` atau `https://`. |
| "This destination URL is not allowed. Please use a different link." | URL tujuan diblokir | URL ini ada dalam blocklist kami. Gunakan URL tujuan lain. |
| "Incorrect password. Please try again." | Kata sandi untuk tautan terlindungi salah | Masukkan kata sandi yang benar untuk mengakses tautan ini. |
| "URL has already been shortened." | URL sudah pernah dipendekkan | URL ini sudah memiliki tautan pendek yang terkait. |

## Masalah Unggah File

| Error Message | Cause | Solution |
|---------------|-------|----------|
| "File size exceeds limit" | File terlalu besar | Kompres file Anda atau upgrade paket untuk unggahan file yang lebih besar. |
| "Storage limit exceeded" | Kuota penyimpanan habis | Hapus beberapa file untuk mengosongkan ruang, atau upgrade paket untuk mendapatkan penyimpanan lebih besar. |
| "Unsupported file type" / "File type not allowed" | Format file tidak didukung | Gunakan format file yang didukung. |
| "Maximum concurrent uploads exceeded" | Terlalu banyak unggahan sekaligus | Tunggu hingga unggahan saat ini selesai sebelum memulai unggahan baru. |
| "This file format is not supported. Allowed formats: JPG, JPEG, PNG, GIF, WEBP, AVIF." | Format avatar tidak didukung | Unggah avatar Anda dalam salah satu format yang didukung: JPG, JPEG, PNG, GIF, WEBP, atau AVIF. |
