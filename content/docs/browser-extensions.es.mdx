La extensión del navegador de S.EE lleva el acortamiento de URL, el uso compartido de texto y la subida de archivos directamente a su navegador. Está disponible para Chrome, Firefox y Edge.

Código abierto en [GitHub](https://github.com/sdotee/browser-extensions).

## Funciones

| Función | Descripción |
|---------|-------------|
| Acortamiento de URL | Acorte cualquier URL con un clic, slugs personalizados y múltiples opciones de dominio |
| Compartir texto | Comparta fragmentos de código y texto con compatibilidad con resaltado de sintaxis |
| Compartir archivos | Suba archivos por lotes con seguimiento del progreso |
| Formatos de exportación | Copie resultados en formato URL simple, Markdown, HTML o BBCode |
| Generación de códigos QR | Genere códigos QR para cualquier URL acortada |
| Exportación de códigos QR | Descargue códigos QR en formato PNG, SVG o PDF (512x512) |
| Menú contextual | Acorte página / enlace, muestre QR, comparta texto seleccionado, suba imágenes |
| Gestión del historial | Historial por pestaña para URL, textos y archivos con eliminación por lotes |
| Persistencia de borradores | Los campos de entrada se guardan automáticamente entre sesiones |
| Copia automática | Copia automáticamente los resultados al portapapeles |
| Modo oscuro / claro | Detección automática del tema con cambio manual |
| Notificaciones del escritorio | Reciba notificaciones cuando las acciones terminen desde el menú contextual |
| Multinavegador | Funciona en Chrome (MV3), Firefox (MV3) y Edge |
