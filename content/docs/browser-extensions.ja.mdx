S.EE のブラウザー拡張機能を使うと、短縮URL、テキスト共有、ファイルアップロードをブラウザー上で直接利用できます。Chrome、Firefox、Edge に対応しています。

ソースコードは [GitHub](https://github.com/sdotee/browser-extensions) で公開しています。

## 機能

| 機能 | 説明 |
|---------|-------------|
| URL 短縮 | 任意の URL をワンクリックで短縮できます。カスタムスラッグや複数ドメインにも対応しています |
| テキスト共有 | シンタックスハイライト付きでコードやテキストを共有できます |
| ファイル共有 | 複数ファイルを進捗表示付きでアップロードできます |
| 出力形式 | プレーン URL、Markdown、HTML、BBCode 形式で結果をコピーできます |
| QRコード生成 | 短縮した URL ごとに QRコードを生成できます |
| QRコード書き出し | QRコードを PNG、SVG、PDF（512x512）でダウンロードできます |
| 右クリックメニュー | ページ / リンクの短縮、QR 表示、選択テキスト共有、画像アップロードが可能です |
| 履歴管理 | URL、テキスト、ファイルの履歴をタブ単位で管理し、一括削除も行えます |
| 下書き保持 | 入力内容が自動保存され、セッションをまたいで保持されます |
| 自動コピー | 結果を自動でクリップボードへコピーします |
| ダーク / ライトモード | テーマの自動判定と手動切り替えに対応しています |
| デスクトップ通知 | コンテキストメニュー経由の操作完了時に通知を受け取れます |
| クロスブラウザー対応 | Chrome（MV3）、Firefox（MV3）、Edge で利用できます |
