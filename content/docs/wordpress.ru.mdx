[Плагин S.EE для WordPress](https://wordpress.org/plugins/sdotee/) позволяет сокращать URL, делиться текстом и загружать файлы, не покидая WordPress. Опубликуйте запись — и короткая ссылка будет создана автоматически, без лишних шагов.

- [Каталог плагинов WordPress](https://wordpress.org/plugins/sdotee/)
- [Исходный код на GitHub](https://github.com/sdotee/see-wordpress)
- [GitHub Releases](https://github.com/sdotee/see-wordpress/releases)

---

## Возможности

| Возможность | Описание |
|---------|-------------|
| Короткие URL | Автоматически создает короткие ссылки при публикации или обновлении записей и страниц |
| Обмен текстом | Делитесь обычным текстом, Markdown и фрагментами кода по ссылке |
| Хостинг файлов | Загружайте файлы из медиабиблиотеки и копируйте ссылку в форматах URL, HTML, Markdown или BBCode |
| Автоматизация | Автоматическая загрузка новых медиа и автоматическое сокращение при публикации: настройте один раз и используйте |
| Интеграция с админкой | Просматривайте и управляйте короткими ссылками прямо в списке записей / страниц |
| Отдельная панель | Отдельная панель S.EE для обмена текстом и файлами с полной историей использования |
| 13 языков | Английский, китайский (упрощенный / традиционный), японский, корейский, индонезийский, вьетнамский, немецкий, французский, испанский, португальский (Бразилия / Португалия), русский |

## Требования

- PHP 8.2+
- WordPress 6.0+
- Аккаунт S.EE с [API Token](https://s.ee/user/developers/)

---

## Установка

### Через каталог плагинов WordPress

1. В админке WordPress откройте **Plugins** > **Add New**
2. Найдите **S.EE**
3. Нажмите **Install Now**, затем **Activate**

### Через GitHub

1. Скачайте последний ZIP-файл из [GitHub Releases](https://github.com/sdotee/see-wordpress/releases)
2. Перейдите в **Plugins** > **Add New** > **Upload Plugin**
3. Загрузите ZIP, нажмите **Install Now**, затем **Activate**

---

## Настройка

### 1. Получите ваш API Token

1. Войдите в аккаунт S.EE
2. Откройте **[Tools > API Token](https://s.ee/user/developers/)**
3. Создайте новый API Token

<Callout type="warn">
Не публикуйте API Token и не отправляйте его в систему контроля версий.
</Callout>

### 2. Настройте плагин

1. Перейдите в **Settings** > **S.EE**
2. Вставьте API Token и нажмите **Verify**
3. Выберите домены по умолчанию для коротких ссылок, текста и файлов
4. Включите автоматическое создание и автоматическую загрузку, если хотите полностью автоматический процесс

<Callout type="info">
API Token также можно задать в `wp-config.php`, чтобы не хранить его в базе данных.
</Callout>

## Использование

### Короткие URL

Если включено автоматическое создание, каждая запись и страница получает короткую ссылку сразу после нажатия Publish. Вы также можете создавать и просматривать короткие ссылки через колонку в списке записей / страниц в админке.

### Обмен текстом

Откройте панель **S.EE** в панели управления и вставьте содержимое. Поддерживаются:

- Обычный текст
- Markdown
- Фрагменты кода

### Хостинг файлов

Загружайте файлы через панель **S.EE** или медиабиблиотеку. После загрузки получите ссылку в нужном формате: URL, HTML, Markdown или BBCode.
