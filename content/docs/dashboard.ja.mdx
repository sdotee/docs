## ダッシュボード概要

ログイン後、[ダッシュボード](https://s.ee/user/dashboard/) にアクセスできます。S.EE ダッシュボードは主に 2 つのナビゲーション領域で構成されています。

### 左サイドバー

主な機能は次のとおりです。

| 機能 | 説明 |
|---------|-------------|
| [URL短縮](https://s.ee/user/links/) | URL短縮を管理します |
| [QR Codes](https://s.ee/user/qr/) | QRコードを作成・管理します |
| [テキスト共有](https://s.ee/user/text/) | テキスト、コードスニペット、Markdown を共有します |
| [ファイル共有](https://s.ee/user/files/) | ファイルをアップロードして共有します |
| [ドメイン](https://s.ee/user/domains/) | カスタムドメインを設定します |
| [アナリティクス](https://s.ee/user/analytics/) | クリック数やスキャン数を確認します |
| [チーム](https://s.ee/user/team/) | チームメンバーを管理します |
| [UTM Builder](https://s.ee/user/library/utm/) | UTM パラメータを作成・管理します |
| [利用状況](https://s.ee/user/usage/) | プランの利用状況を確認します |
| [設定](https://s.ee/user/settings/) | アカウントや請求設定を管理します |

モバイル端末では、サイドバーは自動的に折りたたまれます。

### プロフィールメニュー

左サイドバーにあり、次の操作に利用できます。

- ドキュメントを確認する
- [API キーを管理する](https://s.ee/user/developers/)
- [プランをアップグレードする](https://s.ee/user/pricing/)
- アカウントを切り替える
- ログアウトする
