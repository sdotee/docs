## 创建短网址

1. 前往 [https://s.ee/user/links/](https://s.ee/user/links/)
2. 点击 **创建链接**
3. 输入目标 URL
4. 配置选项（可选）
5. 点击 **创建**

---

## 短网址选项

创建短网址时，您可以配置：

| 选项 | 说明 |
|--------|-------------|
| 自定义短码 | 为链接设置自定义后缀（例如 `s.ee/link`） |
| 域名 | 从官方域名或您的自定义域名中选择 |
| 密码 | 访问目标页面时需要输入密码 |
| 过期时间 | 设置短网址停止生效的日期 |
| 定向 | 根据地区、设备或语言重定向用户 |
| 标签 | 使用标签整理短网址 |
| UTM 参数 | 添加营销活动跟踪参数 |
| 参数传递 | 将 URL 参数转发到目标地址 |

---

## 数据分析

点击任意短网址即可查看其数据分析：

- 总点击量
- 按地区统计的点击量
- 按设备和浏览器统计的点击量
- 来源

---

## 管理短网址

在短网址控制台中，您可以：

- **编辑** - 更新目标 URL 或设置
- **删除** - 永久移除短网址
- **复制** - 将短网址复制到剪贴板
