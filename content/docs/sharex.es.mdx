[ShareX](https://getsharex.com/) es una herramienta de código abierto para capturar pantalla y compartir archivos en Windows. S.EE ofrece configuraciones de cargadores personalizados para ShareX, para que pueda subir capturas, archivos, texto y código directamente a S.EE y obtener un enlace para compartir.

Vea también: [Documentación de cargadores personalizados de ShareX](https://getsharex.com/docs/custom-cargador)

---

## Cargadores personalizados

S.EE tiene configuraciones para estos tipos de subida:

| Cargador | Qué hace |
|----------|-------------|
| Acortador de URL | Acorta URL mediante S.EE |
| Cargador de texto | Sube fragmentos de texto |
| Cargador de código | Sube código con resaltado de sintaxis |
| Cargador de Markdown | Sube Markdown con vista previa renderizada |
| Cargador de imágenes y archivos | Sube imágenes y archivos |

## Instalación

Obtenga los archivos de configuración `.sxcu` que aparecen abajo. Puede descargarlos o importarlos por URL en ShareX.

| Cargador | Enlace de descarga / importación |
|----------|------------------------|

| Cargador de texto | [https://assets.seecdn.com/files/sharex/S.EE_Text_Uploader.sxcu](https://assets.seecdn.com/files/sharex/S.EE_Text_Uploader.sxcu) |
| Cargador de código | [https://assets.seecdn.com/files/sharex/S.EE_Code_Uploader.sxcu](https://assets.seecdn.com/files/sharex/S.EE_Code_Uploader.sxcu) |
| Cargador de Markdown | [https://assets.seecdn.com/files/sharex/S.EE_Markdown_Uploader.sxcu](https://assets.seecdn.com/files/sharex/S.EE_Markdown_Uploader.sxcu) |
| Cargador de imágenes y archivos | [https://assets.seecdn.com/files/sharex/S.EE_File_Uploader.sxcu](https://assets.seecdn.com/files/sharex/S.EE_File_Uploader.sxcu) |

### Importaración

1. Abra **ShareX**
2. Vaya a **Destinos** > **Configuración del cargador personalizado**
3. Haga clic en **Importar** y luego:
   - Seleccione el archivo `.sxcu` descargado
   - O pegue directamente la URL de descarga

## Configure su clave API

1. Obtenga su clave API en **[Herramientas > Token de API](https://s.ee/user/developers/)**
2. En ShareX, abra **Configuración del cargador personalizado** y seleccione el cargador de S.EE que importó
3. En la sección **Encabezados**, establezca el valor de `Authorization` con su clave API

<Callout type="warn">
No comparta su clave API públicamente ni la suba a su sistema de control de versiones.
</Callout>

## Uso

### Elija sus cargadores

En ShareX, vaya a **Destinos** y asigne los cargadores de S.EE:

- Acortador de URL → S.EE URL Shortener
- Cargador de texto → S.EE Text Uploader, Code Uploader o Markdown Uploader
- Cargador de imágenes → S.EE File Uploader
- Cargador de archivos → S.EE File Uploader

### Subir

Use las teclas rápidas de ShareX o el menú contextual para acortar una URL, subir una captura, compartir un fragmento de texto, etc. ShareX sube el contenido a S.EE y copia el enlace al portapapeles.

## Solución de problemas

| Problema | Solución |
|-------|-----|
| Error 401 | Compruebe que su clave API esté bien configurada en el encabezado Authorization |
| Error 403 | Asegúrese de que su clave API tenga los permisos correctos |
| Archivo rechazado | Puede que el tipo de archivo no sea compatible. Consulte [Compartir archivos](/docs/es/file-sharing) para ver los tipos permitidos |

Puede hacer clic en **Probar** en **Configuración del cargador personalizado** para verificar que todo funciona.
