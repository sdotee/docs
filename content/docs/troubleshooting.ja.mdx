## アカウントとログインの問題

| エラーメッセージ | 原因 | 解決方法 |
|---------------|-------|----------|
| "Invalid email or password. Please check your credentials and try again." | メールアドレスまたはパスワードが正しくありません | メールアドレスとパスワードをもう一度ご確認ください。パスワードは大文字と小文字を区別します。 |
| "We couldn't find an account with this email." | メールアドレスが登録されていません | メールアドレスをご確認いただくか、新しいアカウントを作成してください。 |
| "Your account has been suspended. Please contact support for assistance." | アカウントが停止されています | サポートチーム `hi@s.ee` までご連絡ください。 |
| "This login link has expired or is invalid. Please request a new one." | ログインリンクの有効期限が切れているか無効です | ログインページから新しいログインリンクをリクエストしてください。 |
| "This verification code has expired or is invalid. Please request a new one." | 認証コードの有効期限切れまたは無効です | 新しい認証コードをリクエストしてください。 |
| "This email is already registered." | このメールアドレスはすでに使用されています | 別のメールアドレスをご利用いただくか、このメールアドレスでログインをお試しください。 |
| "The current password you entered is incorrect." | パスワード変更時の現在のパスワードが誤っています | 現在のパスワードをご確認のうえ、再度お試しください。 |

**注意:** ログインに使用できるのはメインのメールアドレスのみです。アカウントに追加済みの他の認証済みメールアドレスでは認証できません。パスワード再設定メールも、メインのメールアドレスにのみ送信されます。

## ユーザー名の問題

| エラーメッセージ | 原因 | 解決方法 |
|---------------|-------|----------|
| "This username is already taken. Please choose a different one." | ユーザー名がすでに使用されています | 別のユーザー名を選択してください。 |
| "Username can only be changed once." | ユーザー名変更の上限に達しています | ユーザー名はアカウントごとに 1 回しか変更できません。この変更は恒久的です。 |

## 短縮 URL の問題

| エラーメッセージ | 原因 | 解決方法 |
|---------------|-------|----------|
| "This slug is already taken. Please choose a different one." | カスタムスラッグがすでに使用されています | 別のカスタムスラッグをお試しください。 |
| "This slug cannot be used. Please choose a different one." | システム予約済みのスラッグです | このスラッグは予約済みです。別のスラッグを選択してください。 |
| "This short link does not exist or is no longer available." | 短縮 URL が存在しない、または削除されています | URL が正しいかご確認ください。リンクが削除されている可能性があります。 |
| "The domain linked to this short URL is unavailable or no longer exists." | ドメインが利用できません | サポートへお問い合わせいただくか、別のドメインをご利用ください。 |
| "Please enter a valid destination URL starting with http:// or https://" | URL の形式が正しくありません | リンク先 URL が `http://` または `https://` で始まっていることをご確認ください。 |
| "This destination URL is not allowed. Please use a different link." | リンク先 URL がブロックされています | この URL はブロックリストに登録されています。別のリンク先 URL をご利用ください。 |
| "Incorrect password. Please try again." | パスワード保護されたリンクのパスワードが誤っています | 正しいパスワードを入力してください。 |
| "URL has already been shortened." | この URL はすでに短縮されています | この URL には、すでに既存の短縮 URL が関連付けられています。 |

## サブスクリプションとプランの問題

| エラーメッセージ | 原因 | 解決方法 |
|---------------|-------|----------|
| "You already have an active subscription plan." | すでに契約中です | プランを変更したい場合は、先に現在のサブスクリプションを解約してください。 |
| "You don't have an active subscription to cancel." | 有効なサブスクリプションがありません | 現在、解約できる有効なサブスクリプションはありません。 |
| "This feature requires a premium subscription. Please upgrade your plan to access it." | 上位プランが必要な機能です | この機能を利用するには、プランをアップグレードしてください。 |
| "Unable to find your subscription plan. Please contact support if this issue persists." | プラン情報が見つかりません | 問題が続く場合は、サポートチーム `hi@s.ee` までご連絡ください。 |
| "The selected pricing option is no longer available." | 価格オプションの有効期限が切れています | ページを更新し、現在利用可能な価格オプションを選択してください。 |

## ファイルアップロードの問題

| エラーメッセージ | 原因 | 解決方法 |
|---------------|-------|----------|
| "File size exceeds limit" | ファイルが大きすぎます | ファイルを圧縮するか、より大きなファイルを扱える上位プランへアップグレードしてください。 |
| "Storage limit exceeded" | ストレージ上限に達しています | 不要なファイルを削除して空き容量を確保するか、より多くのストレージを含むプランへアップグレードしてください。 |
| "Unsupported file type" / "File type not allowed" | 対応していないファイル形式です | 対応しているファイル形式をご利用ください。 |
| "Maximum concurrent uploads exceeded" | 同時アップロード数が多すぎます | 現在のアップロードが完了するまでお待ちいただいてから、新しいアップロードを開始してください。 |
| "This file format is not supported. Allowed formats: JPG, JPEG, PNG, GIF, WEBP, AVIF." | アバター画像の形式が未対応です | アバター画像は JPG、JPEG、PNG、GIF、WEBP、AVIF のいずれかでアップロードしてください。 |

## チームの問題

| エラーメッセージ | 原因 | 解決方法 |
|---------------|-------|----------|
| "This feature is not available on your current plan." | ご利用中のプランではチーム機能が含まれていません | チーム機能を含むプランへアップグレードしてください。 |
| "You have already created a team." | すでにチームが存在します | ユーザーごとに作成できるチームは 1 つだけです。既存のチームはダッシュボードから管理できます。 |
| "You don't have permission to manage this team." | 権限が不足しています | チーム所有者へお問い合わせください。 |
| "The invitation has already been processed." | 招待はすでに処理済みです | この招待は、すでに承諾または拒否されています。 |
| "Please wait at least 24 hours before resending." | 招待の再送が早すぎます | このユーザーへ再度招待を送る前に、24 時間お待ちください。 |

## アカウント削除の問題

| エラーメッセージ | 原因 | 解決方法 |
|---------------|-------|----------|
| "Your team has other members, please delete the team first." | チームに他のメンバーが残っています | アカウントを削除する前に、すべてのチームメンバーを削除するか、チーム自体を削除してください。 |
| "Too many deletion requests, please try again later." | リクエストが多すぎます | しばらく時間を置いてから再度お試しください。 |
| "Invalid or expired verification code." | 認証コードが無効または期限切れです | 新しい認証コードをリクエストしてください。 |

## メール管理の問題

| エラーメッセージ | 原因 | 解決方法 |
|---------------|-------|----------|
| "Email already exists." | メールアドレスがすでに使用されています | 別のメールアドレスをご利用ください。 |
| "Main email cannot be deleted." | メインメールアドレスは削除できません | 削除前に、別の認証済みメールアドレスをメインメールアドレスとして設定してください。 |
| "You need to close two factor verification first." | 先に 2 要素認証を無効にする必要があります | メインメールアドレスを変更する前に、2 要素認証を無効にしてください。 |
| "This email has not been verified." | メールアドレスが未認証です | 先にこのメールアドレスを認証してください。 |
