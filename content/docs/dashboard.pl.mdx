## Przegląd panelu

Po zalogowaniu przejdziesz do [panelu](https://s.ee/user/dashboard/). Panel S.EE ma dwa główne obszary nawigacyjne:

### Lewy pasek boczny

Tutaj znajdziesz główne funkcje:

| Funkcja | Opis |
|---------|-------------|
| [Links](https://s.ee/user/links/) | Zarządzanie krótkimi linkami |
| [QR Codes](https://s.ee/user/qr/) | Tworzenie i zarządzanie kodami QR |
| [Text Sharing](https://s.ee/user/text/) | Udostępnianie tekstu, fragmentów kodu i Markdown |
| [File Sharing](https://s.ee/user/files/) | Przesyłanie i udostępnianie plików |
| [Domains](https://s.ee/user/domains/) | Konfiguracja własnych domen |
| [Analityka](https://s.ee/user/analytics/) | Podgląd statystyk kliknięć i skanów |
| [Team](https://s.ee/user/team/) | Zarządzanie członkami zespołu |
| [UTM Builder](https://s.ee/user/library/utm/) | Tworzenie i zarządzanie parametrami UTM |
| [Usage](https://s.ee/user/usage/) | Monitorowanie wykorzystania planu |
| [Ustawienia](https://s.ee/user/settings/) | Ustawienia konta i rozliczeń |

Na urządzeniach mobilnych pasek boczny zwija się automatycznie.

### Menu profilu

Znajduje się w lewym pasku bocznym. Możesz z niego:

- Przejść do dokumentacji
- [Zarządzać kluczami API](https://s.ee/user/developers/)
- [Zmienić plan](https://s.ee/user/pricing/)
- Przełączać się między kontami
- Wylogować się
