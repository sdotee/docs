[ShareX](https://getsharex.com/) to otwartoźródłowe narzędzie do przechwytywania ekranu i udostępniania plików dla Windows. S.EE udostępnia własne konfiguracje uploaderów do ShareX, dzięki czemu możesz przesyłać zrzuty ekranu, pliki, tekst i kod bezpośrednio do S.EE i od razu otrzymywać link do udostępnienia.

Zobacz też: [dokumentację własnych uploaderów ShareX](https://getsharex.com/docs/custom-uploader)

---

## Własne uploadery

S.EE ma konfiguracje dla następujących typów przesyłania:

| Uploader | Działanie |
|----------|-------------|
| URL Shortener | Skraca URL-e przez S.EE |
| Text Uploader | Przesyła fragmenty tekstu |
| Code Uploader | Przesyła kod z podświetlaniem składni |
| Markdown Uploader | Przesyła Markdown z wyrenderowanym podglądem |
| Image & File Uploader | Przesyła obrazy i pliki |

### Import

1. Otwórz **ShareX**
2. Przejdź do **Destinations** > **Custom uploader settings**
3. Kliknij **Import**, a następnie:
   - wybierz pobrany plik `.sxcu`
   - albo wklej bezpośrednio URL do pobrania

## Skonfiguruj klucz API

1. Pobierz klucz API z **[Tools > API Token](https://s.ee/user/developers/)**
2. W ShareX otwórz **Custom uploader settings** i wybierz importer S.EE
3. W sekcji **Headers** ustaw wartość `Authorization` na swój klucz API

<Callout type="warn">
Nie udostępniaj publicznie klucza API i nie zapisuj go w systemie kontroli wersji.
</Callout>

## Użycie

### Wybierz uploadery

W ShareX przejdź do **Destinations** i przypisz uploadery S.EE:

- URL shortener -> S.EE URL Shortener
- Text uploader -> S.EE Text Uploader, Code Uploader albo Markdown Uploader
- Image uploader -> S.EE File Uploader
- File uploader -> S.EE File Uploader

### Przesyłanie

Użyj skrótów klawiaturowych ShareX albo menu kontekstowego, aby skrócić URL, przesłać zrzut ekranu, udostępnić fragment tekstu itd. ShareX prześle dane do S.EE i skopiuje link do schowka.

## Rozwiązywanie problemów

| Problem | Rozwiązanie |
|-------|-----|
| Błąd 401 | Sprawdź, czy klucz API jest poprawnie ustawiony w nagłówku Authorization |
| Błąd 403 | Upewnij się, że klucz API ma odpowiednie uprawnienia |
| Plik odrzucony | Typ pliku może nie być obsługiwany. Zobacz [Udostępnianie plików](/docs/pl/file-sharing) |

Możesz kliknąć **Test** w **Custom uploader settings**, aby sprawdzić, czy wszystko działa poprawnie.
