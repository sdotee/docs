## Обзор

S.EE Text Sharing позволяет делиться обычным текстом, фрагментами кода и документами Markdown по короткой и удобной ссылке. Это удобно для обмена конфигурационными файлами, примерами кода, заметками и документацией.

---

## Создание текстовой публикации

1. Войдите в [https://s.ee/user/dashboard](https://s.ee/user/dashboard)
2. Нажмите **Sharing > Text** в левой боковой панели или сразу откройте [https://s.ee/user/text/](https://s.ee/user/text/)
3. Заполните поля:
   - **Title** (необязательно) - используется как имя файла при скачивании
   - **Short Link** - выберите подходящий домен
   - **Text Type** - выберите **Plain Text**, **Code** или **Markdown**
4. Введите содержимое в поле **Редактировать**
5. При необходимости задайте дополнительные параметры
6. Нажмите **Создать текст**

---

## Типы содержимого

| Тип | Описание |
|------|----------|
| Plain Text | Простой неформатированный текст |
| Code | Фрагменты кода с подсветкой синтаксиса |
| Markdown | Markdown с отрисованным форматированием |

---

## Параметры

| Параметр | Описание |
|--------|----------|
| Password | Защитить текст паролем |
| Tags | Организовать текстовые публикации с помощью меток |
| Expiration | Указать, когда публикация перестанет быть доступной |
| Disable search engine indexing | Запретить поисковым системам индексировать текст |

---

## Просмотр текстовой публикации

После создания вы получите короткую ссылку вида `ba.sh/demo`.

### Режимы просмотра

| URL | Описание |
|-----|----------|
| `ba.sh/demo` | Отображает оформленную страницу текста с подсветкой синтаксиса |
| `ba.sh/demo/raw` | Отображает исходный неформатированный текст напрямую |

Эндпоинт `/raw` полезен для:

- Прямой загрузки скриптов через `curl` или `wget`
- Встраивания в другие приложения
- Просмотра без форматирования

## Поддерживаемые домены

Обмен текстом доступен на следующих доменах для всех тарифов:

| Домен | Лучше всего подходит для |
|--------|----------|
| [ba.sh](https://ba.sh) | Bash- и shell-скрипты |
| [z.sh](https://z.sh) | Shell-скрипты |
| [fs.to](https://fs.to) | Общий обмен файлами |
| [files.to](https://files.to) | Общий обмен файлами |
| [fileshare.to](https://fileshare.to) | Общий обмен файлами |
| [filesharing.to](https://filesharing.to) | Общий обмен файлами |

Для клиентов Premium и Enterprise см. [официальные домены](/docs/ru/official-domains/#premium--enterprise).
