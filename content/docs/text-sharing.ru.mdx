S.EE Text Sharing позволяет делиться обычным текстом, фрагментами кода и документами Markdown по короткой и удобной ссылке. Это удобно для обмена конфигурационными файлами, примерами кода, заметками и документацией.

---

## Создание текстовой публикации

1. Войдите в [https://s.ee/user/dashboard](https://s.ee/user/dashboard)
2. Нажмите **Sharing > Text** в левой боковой панели или сразу откройте [https://s.ee/user/text/](https://s.ee/user/text/)
3. Нажмите **Create Text**
4. Выберите тип содержимого: **Text**, **Code** или **Markdown**
5. Вставьте содержимое
6. При необходимости задайте дополнительные параметры
7. Нажмите **Create**

---

## Типы содержимого

| Тип | Описание |
|------|------|
| Text | Обычный текст |
| Code | Исходный код с подсветкой синтаксиса |
| Markdown | Markdown с отрендеренным предпросмотром |

---

## Параметры

| Параметр | Описание |
|--------|------|
| Domain | Домен для страницы общего доступа |
| Custom Slug | Пользовательский короткий код |
| Title | Заголовок страницы |
| Password | Защита паролем |
| Tags | Организуйте текстовые публикации с помощью меток |
| Expires At | Укажите срок действия публикации |
| Disable Search Engine Indexing | Запрещает поисковым системам индексировать публикацию |

---

## Просмотр текстовой публикации

После создания вы получите короткую ссылку. Страница будет отображаться по-разному в зависимости от типа содержимого:

- **Text** — чистый текст с простым оформлением
- **Code** — код с подсветкой синтаксиса
- **Markdown** — отрендеренный Markdown с форматированием

---

## Поддерживаемые домены

Обмен текстом доступен на следующих доменах для всех тарифов:

| Домен | Лучше всего подходит для |
|--------|----------|
| [sm.ms](https://sm.ms) | Обмен изображениями и коротким контентом |
| [fs.to](https://fs.to) | Общий обмен файлами |
| [files.to](https://files.to) | Общий обмен файлами |
| [fileshare.to](https://fileshare.to) | Общий обмен файлами |
| [filesharing.to](https://filesharing.to) | Общий обмен файлами |
