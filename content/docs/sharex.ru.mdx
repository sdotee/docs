[ShareX](https://getsharex.com/) — это open source инструмент для создания скриншотов и обмена файлами в Windows. S.EE предоставляет настройки пользовательских загрузчиков для ShareX, чтобы вы могли отправлять скриншоты, файлы, текст и код прямо в S.EE и сразу получать ссылку для публикации.

См. также: [Документация по пользовательским загрузчикам ShareX](https://getsharex.com/docs/custom-uploader)

---

## Пользовательские загрузчики

S.EE предоставляет конфигурации для следующих типов загрузки:

| Загрузчик | Назначение |
|----------|-------------|
| Сокращатель URL | Сокращение URL через S.EE |
| Загрузчик текста | Загрузка текстовых фрагментов |
| Загрузчик кода | Загрузка кода с подсветкой синтаксиса |
| Загрузчик Markdown | Загрузка Markdown с отрендеренным предпросмотром |
| Загрузчик изображений и файлов | Загрузка изображений и файлов |

## Установка

Получите файлы конфигурации `.sxcu` ниже. Их можно скачать или импортировать по URL в ShareX.

| Загрузчик | Ссылка для скачивания / импорта |
|----------|------------------------|
| Сокращатель URL | [https://assets.seecdn.com/files/sharex/S.EE_URL_Shortener.sxcu](https://assets.seecdn.com/files/sharex/S.EE_URL_Shortener.sxcu) |
| Загрузчик текста | [https://assets.seecdn.com/files/sharex/S.EE_Text_Uploader.sxcu](https://assets.seecdn.com/files/sharex/S.EE_Text_Uploader.sxcu) |
| Загрузчик кода | [https://assets.seecdn.com/files/sharex/S.EE_Code_Uploader.sxcu](https://assets.seecdn.com/files/sharex/S.EE_Code_Uploader.sxcu) |
| Загрузчик Markdown | [https://assets.seecdn.com/files/sharex/S.EE_Markdown_Uploader.sxcu](https://assets.seecdn.com/files/sharex/S.EE_Markdown_Uploader.sxcu) |
| Загрузчик изображений и файлов | [https://assets.seecdn.com/files/sharex/S.EE_File_Uploader.sxcu](https://assets.seecdn.com/files/sharex/S.EE_File_Uploader.sxcu) |

### Импорт

1. Откройте **ShareX**
2. Перейдите в **Назначения** > **Настройки пользовательского загрузчика**
3. Нажмите **Импорт**, затем:
   - Выберите скачанный файл `.sxcu`
   - Или вставьте URL для скачивания напрямую

## Настройка API-ключа

1. Получите API-ключ в **[Инструменты > API Token](https://s.ee/user/developers/)**
2. В ShareX откройте **Настройки пользовательского загрузчика** и выберите импортированный загрузчик S.EE
3. В разделе **Заголовки** задайте значение `Authorization` равным вашему API-ключу

<Callout type="warn">
Не публикуйте API-ключ и не отправляйте его в систему контроля версий.
</Callout>

## Использование

### Выбор загрузчиков

В ShareX перейдите в **Назначения** и назначьте загрузчики S.EE:

- Сокращатель URL → S.EE URL Shortener
- Загрузчик текста → S.EE Text Uploader, Code Uploader или Markdown Uploader
- Загрузчик изображений → S.EE File Uploader
- Загрузчик файлов → S.EE File Uploader

### Загрузка

Используйте горячие клавиши ShareX или контекстное меню, чтобы сократить URL, загрузить скриншот, поделиться текстовым фрагментом и т. д. ShareX загрузит данные в S.EE и скопирует ссылку в буфер обмена.

## Устранение неполадок

| Проблема | Решение |
|-------|-----|
| Ошибка 401 | Проверьте, правильно ли указан API-ключ в заголовке Authorization |
| Ошибка 403 | Убедитесь, что у API-ключа есть нужные разрешения |
| Файл отклонен | Возможно, тип файла не поддерживается. См. [Обмен файлами](/docs/ru/file-sharing), чтобы узнать допустимые типы |

Вы можете нажать **Проверить** в **Настройки пользовательского загрузчика**, чтобы убедиться, что все работает корректно.
