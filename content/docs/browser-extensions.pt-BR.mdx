A extensão de navegador da S.EE leva encurtamento de URL, compartilhamento de texto e envio de arquivos diretamente para o seu navegador. Disponível para Chrome, Firefox e Edge.

Código aberto no [GitHub](https://github.com/sdotee/browser-extensions).

## Recursos

| Recurso | Descrição |
|---------|-------------|
| Encurtamento de URL | Encurte qualquer URL com um clique, slugs personalizados e várias opções de domínio |
| Compartilhamento de texto | Compartilhe trechos de código e texto com suporte a realce de sintaxe |
| Compartilhamento de arquivos | Envie arquivos em lote com acompanhamento de progresso |
| Formatos de exportação | Copie resultados em URL simples, Markdown, HTML ou BBCode |
| Geração de código QR | Gere códigos QR para qualquer URL encurtada |
| Exportação de código QR | Baixe códigos QR em PNG, SVG ou PDF (512x512) |
| Menu de contexto | Encurte página / link, mostre QR, compartilhe texto selecionado, envie imagens |
| Gerenciamento de histórico | Histórico por aba para URLs, textos e arquivos com exclusão em lote |
| Persistência de rascunhos | Os campos de entrada são salvos automaticamente entre sessões |
| Cópia automática | Copie automaticamente os resultados para a área de transferência |
| Modo escuro / claro | Detecção automática do tema com alternância manual |
| Notificações no desktop | Receba notificações quando ações forem concluídas pelo menu de contexto |
| Multinavegador | Funciona em Chrome (MV3), Firefox (MV3) e Edge |
