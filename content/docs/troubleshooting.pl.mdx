## Problemy z kontem i logowaniem

| Komunikat błędu | Przyczyna | Rozwiązanie |
|---------------|-------|----------|
| "Invalid email or password. Please check your credentials and try again." | Nieprawidłowy e-mail lub hasło | Sprawdź dokładnie adres e-mail i hasło. Wielkość liter w haśle ma znaczenie. |
| "We couldn't find an account with this email." | E-mail nie jest zarejestrowany | Zweryfikuj adres e-mail albo utwórz nowe konto. |
| "Your account has been suspended. Please contact support for assistance." | Konto zostało zawieszone | Skontaktuj się z zespołem pomocy pod adresem [hi@s.ee](mailto:hi@s.ee). |
| "This login link has expired or is invalid. Please request a new one." | Link logowania wygasł | Poproś o nowy link logowania na stronie logowania. |
| "This verification code has expired or is invalid. Please request a new one." | Kod weryfikacyjny wygasł lub jest nieprawidłowy | Poproś o nowy kod weryfikacyjny. |
| "This email is already registered." | E-mail jest już używany | Użyj innego adresu e-mail albo spróbuj zalogować się tym adresem. |
| "The current password you entered is incorrect." | Błędne obecne hasło przy zmianie hasła | Zweryfikuj obecne hasło i spróbuj ponownie. |

**Uwaga:** do logowania działa wyłącznie główny adres e-mail konta. Inne zweryfikowane adresy e-mail nie służą do uwierzytelniania. Wiadomości do resetowania hasła również trafiają tylko na główny adres e-mail.

## Problemy z nazwą użytkownika

| Komunikat błędu | Przyczyna | Rozwiązanie |
|---------------|-------|----------|
| "This username is already taken. Please choose a different one." | Nazwa użytkownika jest zajęta | Wybierz inną nazwę użytkownika. |

## Problemy z krótkimi linkami

| Komunikat błędu | Przyczyna | Rozwiązanie |
|---------------|-------|----------|
| "This slug is already taken. Please choose a different one." | Własny slug jest już zajęty | Spróbuj użyć innego slugu dla krótkiego linku. |
| "This slug cannot be used. Please choose a different one." | Slug jest zarezerwowany przez system | Ten slug jest zarezerwowany. Wybierz inny. |
| "This short link does not exist or is no longer available." | Krótki link nie istnieje albo został usunięty | Sprawdź, czy URL linku jest poprawny. Link mógł zostać usunięty. |
| "The domain linked to this short URL is unavailable or no longer exists." | Domena jest niedostępna | Skontaktuj się z pomocą albo spróbuj użyć innej domeny. |
| "Please enter a valid destination URL starting with http:// or https://" | Nieprawidłowy format URL-a | Upewnij się, że docelowy URL zaczyna się od `http://` albo `https://`. |
| "This destination URL is not allowed. Please use a different link." | Docelowy URL jest zablokowany | Ten URL znajduje się na naszej liście blokad. Użyj innego adresu docelowego. |
| "Incorrect password. Please try again." | Błędne hasło do chronionego linku | Wpisz poprawne hasło, aby uzyskać dostęp do linku. |
| "URL has already been shortened." | URL został już skrócony | Ten URL ma już powiązany istniejący krótki link. |

## Problemy z subskrypcją i planem

| Komunikat błędu | Przyczyna | Rozwiązanie |
|---------------|-------|----------|
| "You already have an active subscription plan." | Subskrypcja jest już aktywna | Jeśli chcesz zmienić plan, najpierw anuluj obecną subskrypcję. |
| "You don't have an active subscription to cancel." | Brak aktywnej subskrypcji | Obecnie nie masz aktywnej subskrypcji do anulowania. |
| "This feature requires a premium subscription. Please upgrade your plan to access it." | Funkcja wymaga wyższego planu | Zmień plan na wyższy, aby uzyskać dostęp do tej funkcji. |
| "Unable to find your subscription plan. Please contact support if this issue persists." | Nie znaleziono planu | Skontaktuj się z zespołem pomocy pod adresem [hi@s.ee](mailto:hi@s.ee). |
| "The selected pricing option is no longer available." | Opcja cenowa wygasła | Odśwież stronę i wybierz aktualną opcję cenową. |

## Problemy z przesyłaniem plików

| Komunikat błędu | Przyczyna | Rozwiązanie |
|---------------|-------|----------|
| "File size exceeds limit" | Plik jest za duży | Skompresuj plik albo przejdź na wyższy plan, aby przesyłać większe pliki. |
| "Storage limit exceeded" | Wykorzystano cały limit miejsca | Usuń część plików, aby zwolnić miejsce, albo przejdź na wyższy plan. |
| "Unsupported file type" / "File type not allowed" | Format pliku nie jest obsługiwany | Użyj obsługiwanego formatu pliku. |
| "Maximum concurrent uploads exceeded" | Zbyt wiele równoległych przesyłań | Poczekaj, aż bieżące przesyłania się zakończą, zanim rozpoczniesz kolejne. |
| "This file format is not supported. Allowed formats: JPG, JPEG, PNG, GIF, WEBP, AVIF." | Nieobsługiwany format awatara | Wgraj awatar w jednym z obsługiwanych formatów: JPG, JPEG, PNG, GIF, WEBP lub AVIF. |

## Problemy zespołowe

| Komunikat błędu | Przyczyna | Rozwiązanie |
|---------------|-------|----------|
| "This feature is not available on your current plan." | Funkcja zespołowa nie jest dostępna w tym planie | Przejdź na plan, który obejmuje funkcje zespołowe. |
| "You have already created a team." | Zespół już istnieje | Każdy użytkownik może utworzyć tylko jeden zespół. Obecnym zespołem zarządzisz z poziomu panelu. |
| "You don't have permission to manage this team." | Brak uprawnień | Skontaktuj się z właścicielem zespołu. |
| "The invitation has already been processed." | Zaproszenie zostało już obsłużone | To zaproszenie zostało już zaakceptowane albo odrzucone. |
| "Please wait at least 24 hours before resending." | Zbyt szybkie ponowne wysłanie zaproszenia | Odczekaj 24 godziny przed ponownym wysłaniem zaproszenia. |

## Problemy z zarządzaniem e-mailem

| Komunikat błędu | Przyczyna | Rozwiązanie |
|---------------|-------|----------|
| "Email already exists." | E-mail jest już używany | Użyj innego adresu e-mail. |
