[Страница загрузки изображений](https://s.ee/user/images/) поддерживает пакетную загрузку. Перетаскивайте файлы или папки, вставляйте из буфера обмена или нажмите, чтобы выбрать файлы. Вставленные изображения автоматически преобразуются в WebP (или PNG, если ваш браузер не поддерживает WebP).

Загруженные файлы сортируются по имени, поэтому порядок в пакетах сохраняется.

---

## Коды для публикации

Для каждого изображения создается ссылка для публикации. Вы можете копировать ссылки по одной или получить код для публикации сразу для всего набора.

Форматы: прямая URL, Markdown, HTML, BBCode.

---

## Управление файлами

Страница изображений предназначена только для загрузки. Чтобы удалить изображения или управлять файлами, откройте [s.ee/user/files/](https://s.ee/user/files/).
