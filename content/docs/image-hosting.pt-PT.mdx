A [página de carregamento de imagens](https://s.ee/user/images/) permite carregamentos em lote. Arraste e largue ficheiros ou pastas, cole a partir da área de transferência ou clique para escolher ficheiros. As imagens coladas são convertidas automaticamente para WebP (ou PNG, se o seu navegador não suportar WebP).

Os ficheiros carregados são ordenados por nome, para que os lotes se mantenham na ordem correta.

---

## Códigos de partilha

Cada imagem recebe um link de partilha. Pode copiar os links um a um ou obter um código de partilha para o lote completo.

Formatos: URL direto, Markdown, HTML, BBCode.

---

## Gerir ficheiros

A página de imagens serve apenas para carregamento. Para eliminar imagens ou gerir ficheiros, aceda a [s.ee/user/files/](https://s.ee/user/files/).
