S.EE 브라우저 확장 프로그램을 사용하면 URL 단축, 텍스트 공유, 파일 업로드를 브라우저에서 바로 처리할 수 있습니다. Chrome, Firefox, Edge 를 지원합니다.

오픈소스 저장소는 [GitHub](https://github.com/sdotee/browser-extensions)에서 확인할 수 있습니다.

## 기능

| Feature | Description |
|---------|-------------|
| URL Shortening | 한 번의 클릭으로 URL 을 단축하고, 사용자 정의 슬러그와 여러 도메인 옵션을 지원합니다 |
| Text Sharing | 구문 강조를 지원하는 코드 스니펫과 텍스트를 공유합니다 |
| File Sharing | 진행 상태 추적과 함께 여러 파일을 업로드합니다 |
| Export Formats | 결과를 Plain URL, Markdown, HTML, BBCode 형식으로 복사합니다 |
| QR Code Generation | 단축 URL 에 대한 QR 코드를 생성합니다 |
| QR Code Export | QR 코드를 PNG, SVG, PDF 형식 (512x512) 으로 다운로드합니다 |
| Right-Click Menu | 페이지/링크 단축, QR 표시, 선택한 텍스트 공유, 이미지 업로드를 지원합니다 |
| History Management | URL, 텍스트, 파일에 대해 탭별 기록과 일괄 삭제를 지원합니다 |
| 초안 유지 | 입력 필드가 세션 간 자동 저장됩니다 |
| 자동 복사 | 결과를 자동으로 클립보드에 복사합니다 |
| 다크/라이트 모드 | 자동 테마 감지와 수동 전환을 지원합니다 |
| 데스크톱 알림 | 컨텍스트 메뉴 작업 완료 시 알림을 표시합니다 |
| Cross-Browser | Chrome (MV3), Firefox (MV3), Edge 에서 동작합니다 |
