[S.EE WordPress 插件](https://wordpress.org/plugins/sdotee/) 让您无需离开 WordPress，就能创建短网址、分享文本和上传文件。发布文章后，系统会自动创建短网址，无需额外步骤。

- [WordPress 插件目录](https://wordpress.org/plugins/sdotee/)
- [GitHub 源代码](https://github.com/sdotee/see-wordpress)
- [GitHub Releases](https://github.com/sdotee/see-wordpress/releases)

---

## 功能

| 功能 | 说明 |
|---------|-------------|
| 短网址 | 发布或更新文章、页面时自动生成短网址 |
| 文本分享 | 通过链接分享纯文本、Markdown 和代码片段 |
| 文件托管 | 从媒体库上传文件，并将链接复制为 URL、HTML、Markdown 或 BBCode |
| 自动化 | 自动上传新媒体，并在发布时自动生成短网址，配置一次即可长期使用 |
| 后台集成 | 直接在文章 / 页面列表中查看和管理短网址 |
| 控制台面板 | 专用的 S.EE 面板，用于文本和文件分享，并提供完整使用历史 |
| 13 种语言 | 英语、中文（简体 / 繁体）、日语、韩语、印尼语、越南语、德语、法语、西班牙语、葡萄牙语（巴西 / 葡萄牙）、俄语 |

## 要求

- PHP 8.2+
- WordPress 6.0+
- 带有 [API Token](https://s.ee/user/developers/) 的 S.EE 账号

---

## 安装

### 从 WordPress 插件目录安装

1. 在 WordPress 后台前往 **插件** > **安装插件**
2. 搜索 **S.EE**
3. 点击 **现在安装**，然后点击 **启用**

### 从 GitHub 安装

1. 从 [GitHub Releases](https://github.com/sdotee/see-wordpress/releases) 下载最新 ZIP
2. 前往 **插件** > **安装插件** > **上传插件**
3. 上传 ZIP，点击 **现在安装**，然后点击 **启用**

---

## 设置

### 1. 获取您的 API Token

1. 登录您的 S.EE 账号
2. 前往 **[Tools > API Token](https://s.ee/user/developers/)**
3. 生成新的 API Token

<Callout type="warn">
不要公开分享您的 API Token，也不要将其提交到版本控制系统中。
</Callout>

### 2. 配置插件

1. 前往 **设置** > **S.EE**
2. 粘贴您的 API Token 并点击 **验证**
3. 为短网址、文本和文件选择默认域名
4. 如果您希望完全自动化，可以启用自动生成和自动上传

<Callout type="info">
您也可以在 `wp-config.php` 中设置 API Token，以避免将其保存在数据库中。
</Callout>

## 使用方法

### 短网址

启用自动生成功能后，每篇文章和页面在发布时都会立即获得一个短网址。您也可以在后台文章 / 页面列表列中创建或查看短网址。

### 文本分享

打开 **S.EE** 控制台面板并粘贴内容。支持的格式包括：

- 纯文本
- Markdown
- 代码片段

### 文件托管

通过 **S.EE** 控制台面板或媒体库上传文件。上传完成后，您可以按需要获取 URL、HTML、Markdown 或 BBCode 格式的链接。
