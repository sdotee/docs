## ¿Qué es S.EE?

S.EE es una plataforma de gestión de enlaces para acortar URL, generar códigos QR y compartir contenido, con analíticas integradas y compatibilidad con dominios personalizados. La plataforma incluye:

- **[S.EE Links](https://s.ee/user/links/)** - Cree enlaces cortos con dominios personalizados, protección por contraseña y segmentación inteligente
- **[S.EE QR Codes](https://s.ee/user/qr/)** - Genere códigos QR personalizables con analíticas
- **[S.EE Text Sharing](https://s.ee/user/text/)** - Comparta texto sin formato, fragmentos de código y Markdown con enlaces cortos
- **[S.EE File Sharing](https://s.ee/user/files/)** - Suba y comparta archivos con enlaces cortos
- **[S.EE Image Hosting](https://s.ee/user/images/)** - Suba y aloje imágenes con enlaces cortos

## Funciones principales

### Gestión de enlaces

Cree enlaces cortos con:
- Slugs y dominios personalizados
- Protección por contraseña
- Fechas de vencimiento
- Segmentación por ubicación, dispositivo o idioma
- Parámetros UTM
- Reenvío de parámetros

### Códigos QR

Genere códigos QR con:
- Colores y logotipos personalizados
- Descarga en PNG, SVG o PDF
- Analíticas integradas

### Compartir texto

Comparta contenido de texto con:
- Compatibilidad con texto sin formato, fragmentos de código y Markdown
- Resaltado de sintaxis para código
- Protección por contraseña y vencimiento
- Endpoint de texto sin formato para descargas sencillas

### Compartir archivos

Suba y comparta archivos con:
- Arrastrar y soltar, pegar o subir carpetas
- Dominios y slugs personalizados
- Protección por contraseña para archivos privados y vencimiento
- Enlaces de descarga directa

### Alojamiento de imágenes

Aloje imágenes con:
- Arrastrar y soltar, pegar o subir carpetas
- Optimización automática de imágenes al pegar desde el portapapeles
- Enlaces directos de imagen para incrustar

### Analíticas

Supervise el rendimiento con:
- Número de clics
- Datos geográficos
- Información del dispositivo y del navegador
- Fuentes de referencia
