## 建立QR Code

1. 前往 [https://s.ee/user/qr/](https://s.ee/user/qr/)
2. 點擊 **New QR Code**
3. 輸入目標 URL
4. 自訂外觀（可選）
5. 點擊 **Create**

---

## 自訂選項

| 選項 | 說明 |
|--------|-------------|
| 顏色 | 設定前景色和背景色 |
| Logo | 在QR Code中央新增您的 Logo |
| 下載格式 | 匯出為 PNG、SVG 或 PDF |

<Callout type="info">
**提示：** 對於在此頁面建立的 QR Code，您可以在短網址後追加 `/qr.png`、`/qr.svg` 或 `/qr.pdf`，快速存取 QR Code 圖片。例如，如果您的 QR Code 短網址為 `https://s.ee/sZvL`，則可以存取：
- `https://s.ee/sZvL/qr.png` - PNG 圖片
- `https://s.ee/sZvL/qr.svg` - SVG 向量圖
- `https://s.ee/sZvL/qr.pdf` - PDF 檔案

**注意：** 此功能僅適用於在 [QR Code](https://s.ee/user/qr/) 頁面建立的動態QR Code，不適用於普通短網址。
</Callout>

## 資料分析

QR Code提供與短網址相同的資料分析：

- 掃碼量
- 地理位置資料
- 裝置資訊
- 來源

---

## 最佳實踐

- 使用高對比度顏色，以獲得更好的掃描效果
- 保持 Logo 較小，以確保QR Code易於掃描
- 列印前先測試您的QR Code
