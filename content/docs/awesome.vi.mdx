Công cụ, plugin và tích hợp dành cho [S.EE](https://s.ee).

---

## Công cụ CLI

| Name | Description | Link |
|------|-------------|------|
| **see-cli** | Sử dụng S.EE từ command line. Rút gọn URL, chia sẻ văn bản và tải tệp lên. | [sdotee/cli](https://github.com/sdotee/cli) |

---

## Plugin

| Name | Description | Link |
|------|-------------|------|
| **S.EE for WordPress** | Rút gọn URL, chia sẻ văn bản và lưu trữ tệp cho WordPress. Tự động tạo URL ngắn khi publish, tải media lên S.EE và theo dõi mọi thứ trong bảng điều khiển. | [WordPress.org](https://wordpress.org/plugins/sdotee/) · [GitHub](https://github.com/sdotee/see-wordpress) · [Tài liệu](/docs/vi/wordpress) |
| **S.EE for Raycast** | Rút gọn URL, chia sẻ văn bản và tải tệp lên ngay từ Raycast. Tự động phát hiện nội dung clipboard, hỗ trợ slug tùy chỉnh, mật khẩu, ngày hết hạn và lịch sử có thể duyệt. | [Raycast Store](https://www.raycast.com/missuo/sdotee) · [GitHub](https://github.com/missuo/raycast-sdotee) |
| **Typecho-Plugin-PicUp** | Plugin cho Typecho dùng để tải ảnh và tệp đính kèm với nhiều backend. Plugin cũng hỗ trợ tải ảnh lên S.EE, giúp bạn quản lý media trên trang Typecho dễ dàng hơn. | [GitHub](https://github.com/lhl77/Typecho-Plugin-PicUp) · [Homepage](https://blog.lhl.one/artical/1026.html) |
