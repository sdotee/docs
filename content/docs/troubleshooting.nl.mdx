## Problemen met account en inloggen

| Error Message | Oorzaak | Oplossing |
|---------------|-------|----------|
| "Invalid email or password. Please check your credentials and try again." | Onjuist e-mailadres of wachtwoord | Controleer uw e-mailadres en wachtwoord nogmaals. Wachtwoorden zijn hoofdlettergevoelig. |
| "We couldn't find an account with this email." | E-mailadres niet geregistreerd | Controleer uw e-mailadres of maak een nieuw account aan. |
| "Your account has been suspended. Please contact support for assistance." | Account geschorst | Neem voor hulp contact op met ons supportteam via [hi@s.ee](mailto:hi@s.ee). |
| "This login link has expired or is invalid. Please request a new one." | Inloglink verlopen | Vraag een nieuwe inloglink aan via de inlogpagina. |
| "This verification code has expired or is invalid. Please request a new one." | Verificatiecode verlopen of ongeldig | Vraag een nieuwe verificatiecode aan. |
| "This email is already registered." | E-mailadres al in gebruik | Gebruik een ander e-mailadres of probeer met dit e-mailadres in te loggen. |
| "The current password you entered is incorrect." | Huidig wachtwoord fout bij wijzigen | Controleer uw huidige wachtwoord en probeer het opnieuw. |

**Opmerking:** alleen uw primaire e-mailadres werkt voor inloggen. Andere geverifieerde adressen op uw account werken niet voor authenticatie. E-mails voor wachtwoordherstel worden ook alleen naar uw primaire e-mailadres gestuurd.

## Problemen met korte links

| Error Message | Oorzaak | Oplossing |

| "This slug is already taken. Please choose a different one." | Aangepaste slug al in gebruik | Probeer een andere aangepaste slug voor uw korte link. |

| "The domain linked to this short URL is unavailable or no longer exists." | Domein is niet beschikbaar | Neem contact op met support of probeer een ander domein. |

| "This destination URL is not allowed. Please use a different link." | Doel-URL is geblokkeerd | Deze URL staat op onze blokkeerlijst. Gebruik een andere doel-URL. |
