## Ikhtisar Dasbor

Akses [Dasbor](https://s.ee/user/dashboard/) Anda setelah login. Dasbor S.EE memiliki dua area navigasi utama:

### Sidebar Kiri

Akses fitur utama Anda:

| Feature | Description |
|---------|-------------|
| [Links](https://s.ee/user/links/) | Kelola tautan pendek Anda |
| [QR Codes](https://s.ee/user/qr/) | Buat dan kelola kode QR |
| [Text Sharing](https://s.ee/user/text/) | Bagikan teks, cuplikan kode, dan Markdown |
| [File Sharing](https://s.ee/user/files/) | Unggah dan bagikan file |
| [Domains](https://s.ee/user/domains/) | Atur domain kustom |
| [Analytics](https://s.ee/user/analytics/) | Lihat statistik klik dan pemindaian |
| [Team](https://s.ee/user/team/) | Kelola anggota tim |
| [UTM Builder](https://s.ee/user/library/utm/) | Buat dan kelola parameter UTM |
| [Usage](https://s.ee/user/usage/) | Pantau penggunaan paket Anda |
| [Settings](https://s.ee/user/settings/) | Pengaturan akun dan penagihan |

Pada perangkat seluler, sidebar akan otomatis dilipat.

### Menu Profil

Terletak di sidebar kiri. Gunakan untuk:

- Mengakses dokumentasi
- [Mengelola API key](https://s.ee/user/developers/)
- [Upgrade paket](https://s.ee/user/pricing/)
- Berpindah akun
- Keluar
