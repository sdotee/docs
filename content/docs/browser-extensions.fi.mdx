S.EE:n selainlaajennus tuo URL-lyhennyksen, tekstin jakamisen ja tiedostojen lataamisen suoraan selaimeesi.

Avoin lähdekoodi on saatavilla [GitHubissa](https://github.com/sdotee/browser-extensions).

## Ominaisuudet

| Ominaisuus | Kuvaus |
|---------|-------------|
| URL-lyhennys | Lyhennä mikä tahansa URL yhdellä napsautuksella, käytä mukautettuja slugeja ja useita verkkotunnuksia |
| Tekstin jakaminen | Jaa tekstikatkelmia ja koodia syntaksikorostuksella |
| Tiedostojen jakaminen | Lähetä tiedostoja erissä ja seuraa etenemistä |
| Vientimuodot | Kopioi tulokset URL-, Markdown-, HTML- tai BBCode-muodossa |
| QR-koodit | Luo QR-koodit mille tahansa lyhennetylle URL:lle |
| QR-vienti | Lataa QR-koodit PNG-, SVG- tai PDF-muodossa |
| Oikean painikkeen valikko | Lyhennä sivu tai linkki, näytä QR-koodi, jaa valittu teksti tai lataa kuva |
| Historian hallinta | Säilytä URL-, teksti- ja tiedostohistoria välilehtikohtaisesti ja poista eriä kerralla |
| Automaattinen kopiointi | Kopioi tulokset automaattisesti leikepöydälle |
