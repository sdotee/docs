## Utilizzo

Ci sono diversi modi per caricare:

- Trascina i file nell'area di upload
- Fai clic sull'area di upload per scegliere file dal computer
- Fai clic sulla mini finestra flottante per caricare direttamente
- Fai clic con il tasto destro sulla mini finestra flottante e seleziona **Carica dagli appunti**

PicGo copia il link risultante negli appunti. Puoi scegliere il formato: Markdown, HTML, URL grezzo e altro.
