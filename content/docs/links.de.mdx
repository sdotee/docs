## Einen Link erstellen

1. Gehen Sie zu [https://s.ee/user/links/](https://s.ee/user/links/)
2. Klicken Sie auf **Create Link**
3. Geben Sie Ihre Ziel-URL ein
4. Konfigurieren Sie die Optionen (optional)
5. Klicken Sie auf **Create**

---

## Link-Optionen

Beim Erstellen eines Links können Sie Folgendes konfigurieren:

| Option | Description |
|--------|-------------|
| Custom slug | Legen Sie einen benutzerdefinierten Pfad für Ihren Link fest (z. B. `s.ee/link`) |
| Domain | Wählen Sie aus offiziellen Domains oder Ihren benutzerdefinierten Domains |
| Password | Fordern Sie ein Passwort für den Zugriff auf das Ziel an |
| Expiration | Legen Sie ein Datum fest, an dem der Link nicht mehr funktioniert |
| Targeting | Leiten Sie Benutzer basierend auf Standort, Gerät oder Sprache weiter |
| Tags | Organisieren Sie Links mit Tags |
| UTM parameters | Fügen Sie Tracking-Parameter für Kampagnen hinzu |
| Parameter passing | Leiten Sie URL-Parameter an das Ziel weiter |

---

## Analyse

Klicken Sie auf einen beliebigen Link, um dessen Analysen anzuzeigen:

- Gesamte Klickzahl
- Klicks nach Standort
- Klicks nach Gerät und Browser
- Referrer-Quellen

---

## Links verwalten

Im Links-Dashboard können Sie:

- **Edit** - Ziel-URL oder Einstellungen aktualisieren
- **Delete** - Den Link dauerhaft entfernen
- **Copy** - Den Kurzlink in die Zwischenablage kopieren
