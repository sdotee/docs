## Einen Link erstellen

1. Gehen Sie zu [https://s.ee/user/links/](https://s.ee/user/links/)
2. Klicken Sie auf **Link erstellen**
3. Geben Sie Ihre Ziel-URL ein
4. Konfigurieren Sie die Optionen (optional)
5. Klicken Sie auf **Erstellen**

---

## Link-Optionen

Beim Erstellen eines Links können Sie Folgendes konfigurieren:

| Option | Beschreibung |
|--------|-------------|
| Benutzerdefinierter Slug | Legen Sie einen benutzerdefinierten Pfad für Ihren Link fest (z. B. `s.ee/link`) |
| Domain | Wählen Sie aus offiziellen Domains oder Ihren benutzerdefinierten Domains |
| Passwort | Fordern Sie ein Passwort für den Zugriff auf das Ziel an |
| Ablaufdatum | Legen Sie ein Datum fest, an dem der Link nicht mehr funktioniert |
| Targeting | Leiten Sie Benutzer basierend auf Standort, Gerät oder Sprache weiter |
| Tags | Organisieren Sie Links mit Tags |
| UTM-Parameter | Fügen Sie Tracking-Parameter für Kampagnen hinzu |
| Parameterweitergabe | Leiten Sie URL-Parameter an das Ziel weiter |

---

## Analyse

Klicken Sie auf einen beliebigen Link, um dessen Analysen anzuzeigen:

- Gesamte Klickzahl
- Klicks nach Standort
- Klicks nach Gerät und Browser
- Referrer-Quellen

---

## Links verwalten

Im Links-Dashboard können Sie:

- **Bearbeiten** - Ziel-URL oder Einstellungen aktualisieren
- **Löschen** - Den Link dauerhaft entfernen
- **Kopieren** - Den Kurzlink in die Zwischenablage kopieren
