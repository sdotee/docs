[ShareX](https://getsharex.com/) là công cụ chụp màn hình và chia sẻ tệp mã nguồn mở cho Windows. S.EE cung cấp cấu hình trình tải lên tùy chỉnh cho ShareX, để bạn có thể tải screenshot, tệp, văn bản và mã trực tiếp lên S.EE và nhận lại liên kết chia sẻ.

Xem thêm: [Tài liệu trình tải lên tùy chỉnh của ShareX](https://getsharex.com/docs/custom-uploader)

---

## Trình tải lên tùy chỉnh

S.EE có cấu hình cho các loại upload sau:

| Trình tải lên | Chức năng |
|----------|-------------|
| Trình rút gọn URL | Rút gọn URL qua S.EE |
| Trình tải văn bản lên | Tải lên đoạn văn bản |
| Trình tải mã lên | Tải lên mã có tô sáng cú pháp |
| Trình tải Markdown lên | Tải lên Markdown với bản xem trước đã render |
| Trình tải hình ảnh và tệp lên | Tải lên hình ảnh và tệp |

## Cài đặt

Lấy các tệp cấu hình `.sxcu` dưới đây. Bạn có thể tải chúng xuống hoặc nhập bằng URL trong ShareX.

### Nhập

1. Mở **ShareX**
2. Truy cập **Đích đến** > **Cài đặt trình tải lên tùy chỉnh**
3. Nhấp vào **Nhập**, sau đó chọn một trong hai cách:
   - Chọn tệp `.sxcu` đã tải xuống
   - Dán trực tiếp URL tải xuống

## Thiết lập API key

1. Lấy API key của bạn từ **[Công cụ > API Token](https://s.ee/user/developers/)**
2. Trong ShareX, mở **Cài đặt trình tải lên tùy chỉnh** và chọn trình tải lên S.EE mà bạn đã import
3. Trong phần **Tiêu đề**, đặt giá trị `Authorization` thành API key của bạn

<Callout type="warn">
Không chia sẻ công khai API key hoặc commit nó vào version control.
</Callout>

## Cách sử dụng

### Chọn trình tải lên

Trong ShareX, vào **Đích đến** và gán các trình tải lên của S.EE:

- Trình rút gọn URL → S.EE URL Shortener
- Trình tải văn bản lên → S.EE Text Uploader, Code Uploader hoặc Markdown Uploader
- Trình tải hình ảnh lên → S.EE File Uploader
- Trình tải tệp lên → S.EE File Uploader

### Upload

Dùng phím tắt hoặc menu chuột phải của ShareX để rút gọn URL, tải screenshot lên, chia sẻ đoạn văn bản, v.v. ShareX sẽ tải lên S.EE và sao chép liên kết vào clipboard.

## Khắc phục sự cố

| Sự cố | Cách khắc phục |
|-------|-----|
| Lỗi 401 | Kiểm tra xem API key đã được đặt đúng trong header Authorization chưa |
| 403 error | Đảm bảo API key có đủ quyền cần thiết |
| File rejected | Có thể loại tệp này không được hỗ trợ. Hãy xem [Chia sẻ tệp](/docs/vi/file-sharing) để biết các loại tệp được phép |

Bạn có thể nhấp **Kiểm tra** trong **Cài đặt trình tải lên tùy chỉnh** để xác minh mọi thứ hoạt động chính xác.
