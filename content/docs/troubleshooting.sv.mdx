## Konto- och inloggningsproblem

| Felmeddelande | Orsak | Lösning |
|---------------|-------|----------|
| "Invalid email or password. Please check your credentials and try again." | Fel e-postadress eller lösenord | Kontrollera e-postadress och lösenord noggrant. Observera att lösenord är skiftlägeskänsliga. |
| "We couldn't find an account with this email." | E-postadressen är inte registrerad | Kontrollera e-postadressen eller skapa ett nytt konto. |
| "Your account has been suspended. Please contact support for assistance." | Kontot är avstängt | Kontakta supporten på [hi@s.ee](mailto:hi@s.ee) för hjälp. |
| "This login link has expired or is invalid. Please request a new one." | Inloggningslänken har löpt ut | Begär en ny inloggningslänk från inloggningssidan. |
| "This verification code has expired or is invalid. Please request a new one." | Verifieringskoden har löpt ut eller är ogiltig | Begär en ny verifieringskod. |
| "This email is already registered." | E-postadressen används redan | Använd en annan e-postadress eller försök logga in med denna adress. |
| "The current password you entered is incorrect." | Fel nuvarande lösenord vid lösenordsbyte | Kontrollera ditt nuvarande lösenord och försök igen. |

**Obs:** Endast din primära e-postadress fungerar för inloggning. Andra verifierade e-postadresser på kontot fungerar inte för autentisering. E-post för lösenordsåterställning skickas också bara till den primära e-postadressen.

| Felmeddelande | Orsak | Lösning |

## Problem med korta länkar

| Felmeddelande | Orsak | Lösning |
|---------------|-------|----------|
| "This slug is already taken. Please choose a different one." | Den anpassade sluggen används redan | Prova en annan slug för den korta länken. |
| "This slug cannot be used. Please choose a different one." | Sluggen är reserverad av systemet | Den här sluggen är reserverad. Välj en annan. |
| "This short link does not exist or is no longer available." | Den korta länken finns inte eller har tagits bort | Kontrollera att länkens URL är korrekt. Länken kan ha tagits bort. |
| "The domain linked to this short URL is unavailable or no longer exists." | Domänen är inte tillgänglig | Kontakta supporten eller prova en annan domän. |
| "Please enter a valid destination URL starting with http:// or https://" | Ogiltigt URL-format | Kontrollera att mål-URL:en börjar med `http://` eller `https://`. |
| "This destination URL is not allowed. Please use a different link." | Måladressen är blockerad | Den här URL:en finns på vår blocklista. Använd en annan måladress. |
| "Incorrect password. Please try again." | Fel lösenord för lösenordsskyddad länk | Ange rätt lösenord för att öppna länken. |
| "URL has already been shortened." | URL:en har redan förkortats | Den här URL:en har redan en befintlig kort länk. |

## Problem med abonnemang och plan

| Felmeddelande | Orsak | Lösning |
|---------------|-------|----------|
| "You already have an active subscription plan." | Du har redan ett aktivt abonnemang | Om du vill byta abonnemang måste du först avsluta det nuvarande. |
| "You don't have an active subscription to cancel." | Inget aktivt abonnemang | Du har för närvarande inget aktivt abonnemang att avsluta. |
| "This feature requires a premium subscription. Please upgrade your plan to access it." | Funktionen kräver högre nivå | Uppgradera abonnemanget för att få tillgång till funktionen. |
| "Unable to find your subscription plan. Please contact support if this issue persists." | Abonnemanget hittas inte | Kontakta supporten på [hi@s.ee](mailto:hi@s.ee) om problemet kvarstår. |
| "The selected pricing option is no longer available." | Prisalternativet har löpt ut | Uppdatera sidan och välj ett aktuellt prisalternativ. |

## Problem med filuppladdning

| Felmeddelande | Orsak | Lösning |
|---------------|-------|----------|
| "File size exceeds limit" | Filen är för stor | Komprimera filen eller uppgradera abonnemanget för större uppladdningar. |
| "Storage limit exceeded" | Lagringskvoten är nådd | Ta bort några filer för att frigöra utrymme eller uppgradera abonnemanget för mer lagring. |
| "Unsupported file type" / "File type not allowed" | Filformatet stöds inte | Använd ett filformat som stöds. |
| "Maximum concurrent uploads exceeded" | För många samtidiga uppladdningar | Vänta tills pågående uppladdningar är klara innan du startar nya. |
| "This file format is not supported. Allowed formats: JPG, JPEG, PNG, GIF, WEBP, AVIF." | Avatarformat stöds inte | Ladda upp avataren i JPG, JPEG, PNG, GIF, WEBP eller AVIF. |

## Problem med team

| Felmeddelande | Orsak | Lösning |
|---------------|-------|----------|
| "This feature is not available on your current plan." | Teamfunktionen ingår inte i din plan | Uppgradera till en plan som innehåller teamfunktioner. |
| "You have already created a team." | Team finns redan | Varje användare kan bara skapa ett team. Hantera det befintliga teamet från instrumentpanelen. |
| "You don't have permission to manage this team." | Otillräckliga behörigheter | Kontakta teamägaren för hjälp. |
| "The invitation has already been processed." | Inbjudan har redan hanterats | Inbjudan har redan accepterats eller avvisats. |
| "Please wait at least 24 hours before resending." | Inbjudan skickas om för tidigt | Vänta 24 timmar innan du skickar en ny inbjudan. |
