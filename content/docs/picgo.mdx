[PicGo](https://picgo.app/) is an open-source image and file upload tool for Windows, macOS, and Linux. Recent versions ship with S.EE support out of the box, so you just need an API Token to get started.

- [Download PicGo](https://github.com/Molunerfinn/PicGo/releases)
- [Source Code](https://github.com/Molunerfinn/PicGo)

---

## Setup

### 1. Get your API Token

1. Log in to your S.EE account
2. Go to **[Tools > API Token](https://s.ee/user/developers/)**
3. Generate a new API Token

<Callout type="warn">
Don't share your API Token publicly or commit it to version control.
</Callout>

### 2. Configure PicGo

1. Download and install [PicGo](https://github.com/Molunerfinn/PicGo/releases)
2. Open PicGo
3. Go to **Picbeds Settings** > **SM.MS (S.EE)**
4. Click **+** to add a new configuration
5. Enter a name in **Configuration Name** (e.g. `S.EE`)
6. Paste your API Token in **Set Token**
7. Click **Confirm**

![PicGo S.EE Configuration](https://i.see.you/2026/02/10/k6Nr/pasted-image-1770748236942.webp)

## Usage

There are a few ways to upload:

- Drag and drop files onto the Upload Area
- Click the Upload Area to pick files from your computer
- Click the floating mini window to upload directly
- Right-click the floating mini window and select **Upload by Clipboard**

PicGo copies the resulting link to your clipboard. You can choose the format: Markdown, HTML, raw URL, etc.
