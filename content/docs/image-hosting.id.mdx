[Halaman unggah gambar](https://s.ee/user/images/) menangani unggahan gambar massal. Tarik dan lepas file atau folder, paste dari clipboard, atau klik untuk memilih file. Gambar yang dipaste akan otomatis dikonversi ke WebP (atau PNG jika browser Anda tidak mendukung WebP).

File yang diunggah diurutkan berdasarkan nama file sehingga unggahan batch tetap berurutan.

---

## Kode berbagi

Setiap gambar mendapatkan tautan berbagi. Anda dapat menyalin tautan satu per satu atau mengambil kode berbagi untuk seluruh batch.

Format yang didukung: Direct URL, Markdown, HTML, BBCode.

---

## Mengelola file

Halaman gambar hanya untuk unggahan. Untuk menghapus gambar atau mengelola file, buka [s.ee/user/files/](https://s.ee/user/files/).
