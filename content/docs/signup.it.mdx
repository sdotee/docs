Per usare S.EE, devi prima creare un account.

1. Apri [https://s.ee/signup/](https://s.ee/signup/)
2. Inserisci il tuo indirizzo email
3. Fai clic su **Continue with Email**
4. Controlla la tua casella di posta e apri il link di conferma

---

## Opzioni di accesso

Puoi registrarti o accedere anche con:

- Google
- GitHub
- Apple

---

## Passaggi successivi

Dopo la creazione dell'account puoi:

- creare il tuo primo link breve
- caricare file e immagini
- condividere testo o Markdown
- generare una API key per integrazioni e strumenti
