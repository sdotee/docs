## Problemi di account e accesso

| Messaggio di errore | Causa | Soluzione |
|---------------|-------|----------|
| "Invalid email or password. Please check your credentials and try again." | Email o password non validi | Controlla con attenzione il tuo indirizzo email e la password. Ricorda che le password sono sensibili alle maiuscole. |
| "We couldn't find an account with this email." | Email non registrata | Verifica l'indirizzo email oppure crea un nuovo account. |
| "Your account has been suspended. Please contact support for assistance." | Account sospeso | Contatta il nostro team di supporto all'indirizzo [hi@s.ee](mailto:hi@s.ee) per ricevere assistenza. |
| "This login link has expired or is invalid. Please request a new one." | Link di accesso scaduto | Richiedi un nuovo link di accesso dalla pagina di login. |
| "This verification code has expired or is invalid. Please request a new one." | Codice di verifica scaduto o non valido | Richiedi un nuovo codice di verifica. |
| "This email is already registered." | Email già in uso | Usa un indirizzo email diverso oppure prova ad accedere con questa email. |
| "The current password you entered is incorrect." | Password attuale errata durante la modifica della password | Verifica la password attuale e riprova. |

**Nota:** solo il tuo indirizzo email principale funziona per l'accesso. Le altre email verificate sul tuo account non possono essere usate per l'autenticazione. Anche le email di reimpostazione della password vengono inviate solo al tuo indirizzo email principale.

|---------------|-------|----------|
| "This username is already taken. Please choose a different one." | Nome utente già in uso | Scegli un nome utente diverso. |

## Problemi con i link brevi

| Messaggio di errore | Causa | Soluzione |
|---------------|-------|----------|
| "This slug is already taken. Please choose a different one." | Slug personalizzato già in uso | Prova con uno slug diverso per il tuo link breve. |
| "This slug cannot be used. Please choose a different one." | Slug riservato dal sistema | Questo slug è riservato. Scegline uno diverso. |
| "This short link does not exist or is no longer available." | Il link breve non esiste o è stato eliminato | Verifica che l'URL del link sia corretto. Potrebbe essere stato eliminato. |
| "The domain linked to this short URL is unavailable or no longer exists." | Dominio non disponibile | Contatta il supporto oppure prova a usare un dominio diverso. |
| "Please enter a valid destination URL starting with http:// or https://" | Formato URL non valido | Assicurati che l'URL di destinazione inizi con `http://` o `https://`. |
| "This destination URL is not allowed. Please use a different link." | URL di destinazione bloccato | Questo URL è nella nostra blocklist. Usa un URL di destinazione diverso. |
| "Incorrect password. Please try again." | Password errata per un link protetto | Inserisci la password corretta per accedere a questo link. |
| "URL has already been shortened." | URL già accorciato | Questo URL ha già un link breve associato. |

## Problemi di abbonamento e piano

| Messaggio di errore | Causa | Soluzione |
|---------------|-------|----------|
| "You already have an active subscription plan." | Abbonamento già attivo | Se vuoi cambiare piano, annulla prima l'abbonamento corrente. |
| "You don't have an active subscription to cancel." | Nessun abbonamento attivo | Al momento non hai un abbonamento attivo da annullare. |
| "This feature requires a premium subscription. Please upgrade your plan to access it." | Funzionalità disponibile solo su un piano superiore | Aggiorna il tuo piano per accedere a questa funzione. |
| "Unable to find your subscription plan. Please contact support if this issue persists." | Piano non trovato | Contatta il nostro team di supporto all'indirizzo [hi@s.ee](mailto:hi@s.ee) per ricevere assistenza. |
| "The selected pricing option is no longer available." | Opzione di prezzo scaduta | Ricarica la pagina e seleziona un'opzione di prezzo disponibile. |

## Problemi di caricamento file

| Messaggio di errore | Causa | Soluzione |
|---------------|-------|----------|
| "File size exceeds limit" | File troppo grande | Comprimi il file oppure aggiorna il tuo piano per caricare file più grandi. |
| "Storage limit exceeded" | Quota di archiviazione raggiunta | Elimina alcuni file per liberare spazio oppure aggiorna il piano per ottenere più storage. |
| "Unsupported file type" / "File type not allowed" | Formato file non supportato | Usa un formato file supportato. |
| "Maximum concurrent uploads exceeded" | Troppi caricamenti simultanei | Attendi che i caricamenti attuali finiscano prima di avviarne di nuovi. |
| "This file format is not supported. Allowed formats: JPG, JPEG, PNG, GIF, WEBP, AVIF." | Formato avatar non supportato | Carica l'avatar in uno dei formati supportati: JPG, JPEG, PNG, GIF, WEBP o AVIF. |

## Problemi di team

| Messaggio di errore | Causa | Soluzione |
|---------------|-------|----------|
| "This feature is not available on your current plan." | Funzionalità team non inclusa nel piano | Passa a un piano che includa le funzioni team. |
| "You have already created a team." | Team già esistente | Ogni utente può creare un solo team. Puoi gestire il team esistente dalla dashboard. |
| "You don't have permission to manage this team." | Permessi insufficienti | Contatta il proprietario del team per assistenza. |
| "The invitation has already been processed." | Invito già gestito | Questo invito è già stato accettato o rifiutato. |
| "Please wait at least 24 hours before resending." | Invito reinviato troppo presto | Attendi 24 ore prima di inviare un altro invito a questo utente. |
