## Utwórz link

1. Przejdź do [https://s.ee/user/links/](https://s.ee/user/links/)
2. Kliknij **Create Link**
3. Wprowadź docelowy adres URL
4. Skonfiguruj opcje (opcjonalnie)
5. Kliknij **Create**

---

## Opcje linku

Podczas tworzenia linku możesz ustawić:

| Opcja | Opis |
|-------|------|
| Własny slug | Ustaw własną końcówkę linku, np. `s.ee/link` |
| Domena | Wybierz jedna z oficjalnych domen albo jedna ze swoich domen |
| Hasło | Wymagaj hasła, aby otworzyć adres docelowy |
| Wygaśnięcie | Ustaw datę, po której link przestanie działać |
| Targetowanie | Przekierowuj użytkowników zależnie od lokalizacji, urządzenia lub języka |
| Tagi | Organizuj linki za pomocą tagów |
| Parametry UTM | Dodaj parametry do śledzenia kampanii |
| Przekazywanie parametrów | Przekazuj parametry URL do adresu docelowego |

---

## Analityka

Kliknij dowolny link, aby zobaczyć jego analitykę:

- Liczbę kliknięć
- Kliknięcia według lokalizacji
- Kliknięcia według urządzenia i przeglądarki
- Źródła odwiedzin

---

## Zarządzanie linkami

W panelu linków możesz:

- **Edytować** - zmienić adres docelowy lub ustawienia
- **Usunąć** - trwale usunąć link
- **Skopiować** - skopiować krótki link do schowka
