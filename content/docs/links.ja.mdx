## 短縮URLを作成する

1. [https://s.ee/user/links/](https://s.ee/user/links/) にアクセスします
2. **Create Link** をクリックします
3. 転送先 URL を入力します
4. 必要に応じてオプションを設定します
5. **Create** をクリックします

---

## 短縮URLのオプション

短縮URLの作成時に、次の設定を行えます。

| オプション | 説明 |
|--------|-------------|
| カスタムスラッグ | リンクの末尾を任意の文字列に設定します（例: `s.ee/link`） |
| ドメイン | 公式ドメインまたはご自身のカスタムドメインから選択できます |
| パスワード | 転送先へアクセスする際にパスワード入力を必須にします |
| 有効期限 | 短縮URLが無効になる日時を設定します |
| ターゲティング | 地域、端末、言語に応じてリダイレクトします |
| タグ | タグで短縮URLを整理できます |
| UTM パラメータ | キャンペーン計測用のパラメータを追加します |
| パラメータ引き継ぎ | URL パラメータを転送先へ引き継ぎます |

---

## アナリティクス

任意の短縮URLをクリックすると、次の分析データを確認できます。

- 総クリック数
- 地域別クリック数
- 端末別・ブラウザ別クリック数
- 参照元

---

## 短縮URLを管理する

短縮URLのダッシュボードでは、次の操作を行えます。

- **Edit** - 転送先 URL や設定を更新します
- **Delete** - 短縮URLを完全に削除します
- **Copy** - 短縮URLをクリップボードにコピーします
