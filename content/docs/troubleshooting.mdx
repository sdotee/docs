## Account & Login Issues

| Error Message | Cause | Solution |
|---------------|-------|----------|
| "Invalid email or password. Please check your credentials and try again." | Incorrect email or password | Please double-check your email and password. Note that passwords are case-sensitive. |
| "We couldn't find an account with this email." | Email not registered | Please verify your email address or create a new account. |
| "Your account has been suspended. Please contact support for assistance." | Account suspended | Please contact our support team at [hi@s.ee](mailto:hi@s.ee) for assistance. |
| "This login link has expired or is invalid. Please request a new one." | Login link expired | Please request a new login link from the login page. |
| "This verification code has expired or is invalid. Please request a new one." | Verification code expired or invalid | Please request a new verification code. |
| "This email is already registered." | Email already in use | Please use a different email address or try logging in with this email. |
| "The current password you entered is incorrect." | Wrong current password when changing password | Please verify your current password and try again. |

**Note:** Only your primary email address works for login. Other verified emails on your account won't work for authentication. Password reset emails are also sent only to your primary email address.

## Username Issues

| Error Message | Cause | Solution |
|---------------|-------|----------|
| "This username is already taken. Please choose a different one." | Username already in use | Please choose a different username. |
| "Username can only be changed once." | Username change limit reached | Unfortunately, usernames can only be changed once per account. This change is permanent. |

## Short Link Issues

| Error Message | Cause | Solution |
|---------------|-------|----------|
| "This slug is already taken. Please choose a different one." | Custom slug already in use | Please try a different custom slug for your short link. |
| "This slug cannot be used. Please choose a different one." | Slug is reserved by the system | This slug is reserved. Please choose a different one. |
| "This short link does not exist or is no longer available." | Short link doesn't exist or has been deleted | Please verify that the link URL is correct. The link may have been deleted. |
| "The domain linked to this short URL is unavailable or no longer exists." | Domain is unavailable | Please contact support or try using a different domain. |
| "Please enter a valid destination URL starting with http:// or https://" | Invalid URL format | Please ensure your destination URL starts with `http://` or `https://`. |
| "This destination URL is not allowed. Please use a different link." | Destination URL is blocked | This URL is on our blocklist. Please use a different destination URL. |
| "Incorrect password. Please try again." | Wrong password for password-protected link | Please enter the correct password to access this link. |
| "URL has already been shortened." | URL was already shortened | This URL already has an existing short link associated with it. |

## Subscription & Plan Issues

| Error Message | Cause | Solution |
|---------------|-------|----------|
| "You already have an active subscription plan." | Already subscribed | If you'd like to change plans, please cancel your current subscription first. |
| "You don't have an active subscription to cancel." | No active subscription | You currently don't have an active subscription to cancel. |
| "This feature requires a premium subscription. Please upgrade your plan to access it." | Feature requires higher tier | Please upgrade your subscription plan to access this feature. |
| "Unable to find your subscription plan. Please contact support if this issue persists." | Plan not found | Please contact our support team at [hi@s.ee](mailto:hi@s.ee) for assistance. |
| "The selected pricing option is no longer available." | Pricing option expired | Please refresh the page and select a current pricing option. |

## File Upload Issues

| Error Message | Cause | Solution |
|---------------|-------|----------|
| "File size exceeds limit" | File is too large | Please compress your file or upgrade your plan for larger file uploads. |
| "Storage limit exceeded" | Storage quota reached | Please delete some files to free up space, or upgrade your plan for more storage. |
| "Unsupported file type" / "File type not allowed" | File format not supported | Please use a supported file format. |
| "Maximum concurrent uploads exceeded" | Too many simultaneous uploads | Please wait for your current uploads to complete before starting new ones. |
| "This file format is not supported. Allowed formats: JPG, JPEG, PNG, GIF, WEBP, AVIF." | Avatar format not supported | Please upload your avatar in one of the supported formats: JPG, JPEG, PNG, GIF, WEBP, or AVIF. |

## Team Issues

| Error Message | Cause | Solution |
|---------------|-------|----------|
| "This feature is not available on your current plan." | Team feature not included in plan | Please upgrade to a plan that includes team features. |
| "You have already created a team." | Team already exists | Each user can only create one team. You can manage your existing team from the dashboard. |
| "You don't have permission to manage this team." | Insufficient permissions | Please contact your team owner for assistance. |
| "The invitation has already been processed." | Invitation already handled | This invitation has already been accepted or declined. |
| "Please wait at least 24 hours before resending." | Resending invitation too soon | Please wait 24 hours before sending another invitation to this user. |
