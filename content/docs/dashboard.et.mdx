## Juhtpaneeli ülevaade

Pärast sisselogimist ava oma [juhtpaneel](https://s.ee/user/dashboard/). S.EE juhtpaneelil on kaks peamist navigeerimisala.

### Vasak külgriba

Siit pääsed põhiliste funktsioonide juurde:

| Funktsioon | Kirjeldus |
|---------|-------------|
| [Links](https://s.ee/user/links/) | Halda oma lühilinke |
| [QR Codes](https://s.ee/user/qr/) | Loo ja halda QR-koode |
| [Text Sharing](https://s.ee/user/text/) | Jaga teksti, koodilõike ja Markdowni |
| [File Sharing](https://s.ee/user/files/) | Laadi faile üles ja jaga neid |
| [Domains](https://s.ee/user/domains/) | Seadista kohandatud domeene |
| [Analüütika](https://s.ee/user/analytics/) | Vaata klikkide ja skannimiste statistikat |
| [Team](https://s.ee/user/team/) | Halda tiimiliikmeid |
| [UTM Builder](https://s.ee/user/library/utm/) | Loo ja halda UTM-parameetreid |
| [Usage](https://s.ee/user/usage/) | Jälgi paketi kasutust |
| [Seaded](https://s.ee/user/settings/) | Konto- ja arveldusseaded |

Mobiilseadmetes tõmbub külgriba automaatselt kokku.

### Profiilimenüü

See asub vasakul külgribal. Selle kaudu saad:

- Avada dokumentatsiooni
- [Hallata API võtmeid](https://s.ee/user/developers/)
- [Muuta oma paketti](https://s.ee/user/pricing/)
- Vahetada kontode vahel
- Logida välja
