S.EE 瀏覽器擴充套件將短網址、文字分享和檔案上傳直接帶到您的瀏覽器中。支援 Chrome、Firefox 和 Edge。

開源地址：[GitHub](https://github.com/sdotee/browser-extensions)。

## 功能

| 功能 | 說明 |
|---------|-------------|
| URL 縮短 | 一鍵縮短任意 URL，支援自訂短碼和多個網域選項 |
| 文字分享 | 支援帶語法高亮的程式碼片段和文字分享 |
| 檔案分享 | 支援批次上傳檔案並追蹤進度 |
| 匯出格式 | 可將結果複製為 Plain URL、Markdown、HTML 或 BBCode |
| QR Code 生成 | 為任意短網址生成 QR Code |
| QR Code匯出 | 將QR Code下載為 PNG、SVG 或 PDF 格式（512x512） |
| 右鍵選單 | 縮短頁面 / 連結、顯示QR Code、分享選中文字、上傳圖片 |
| 歷史記錄管理 | 按標籤頁區分 URL、文字和檔案歷史，並支援批次刪除 |
| 草稿持久化 | 輸入欄位會跨會話自動儲存 |
| 自動複製 | 自動將結果複製到剪貼簿 |
| 深色 / 淺色模式 | 自動檢測主題，並支援手動切換 |
| 桌面通知 | 透過右鍵選單在操作完成時接收通知 |
| 跨瀏覽器 | 支援 Chrome（MV3）、Firefox（MV3）和 Edge |
