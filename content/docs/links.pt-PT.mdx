## Criar um link

1. Aceda a [https://s.ee/user/links/](https://s.ee/user/links/)
2. Clique em **Create Link**
3. Introduza o URL de destino
4. Configure as opções (opcional)
5. Clique em **Create**

---

## Opções do link

Ao criar um link, pode configurar:

| Opção | Descrição |
|--------|-------------|
| Slug personalizado | Defina a parte final personalizada do seu link (por exemplo, `s.ee/link`) |
| Domínio | Escolha entre domínios oficiais ou os seus domínios personalizados |
| Palavra-passe | Exija uma palavra-passe para aceder ao destino |
| Expiração | Defina uma data em que o link deixará de funcionar |
| Segmentação | Redirecione utilizadores com base em localização, dispositivo ou idioma |
| Etiquetas | Organize links com etiquetas |
| Parâmetros UTM | Adicione parâmetros de rastreio de campanha |
| Encaminhamento de parâmetros | Encaminhe parâmetros do URL para o destino |

---

## Analítica

Clique em qualquer link para ver a respetiva analítica:
- Total de cliques
- Cliques por localização
- Cliques por dispositivo e navegador
- Fontes de referência

---

## Gerir links

No painel de links, pode:
- **Edit** - Atualizar o URL de destino ou as definições
- **Delete** - Remover o link permanentemente
- **Copy** - Copiar o link curto para a área de transferência
