## Einen QR-Code erstellen

1. Gehen Sie zu [https://s.ee/user/qr/](https://s.ee/user/qr/)
2. Klicken Sie auf **New QR Code**
3. Geben Sie die Ziel-URL ein
4. Passen Sie das Erscheinungsbild an (optional)
5. Klicken Sie auf **Erstellen**

---

## Anpassungsoptionen

| Option | Description |
|--------|-------------|
| Colors | Legen Sie Vorder- und Hintergrundfarben fest |
| Logo | Fügen Sie Ihr Logo in der Mitte des QR-Codes hinzu |
| Download-Format | Exportieren Sie als PNG, SVG oder PDF |

<Callout type="info">
**Tipp:** Für auf dieser Seite erstellte QR-Codes können Sie schnell auf das QR-Code-Bild zugreifen, indem Sie `/qr.png`, `/qr.svg` oder `/qr.pdf` an die Kurz-URL anhängen. Wenn Ihre QR-Code-Kurz-URL beispielsweise `https://s.ee/sZvL` ist, können Sie Folgendes aufrufen:
- `https://s.ee/sZvL/qr.png` - PNG-Bild
- `https://s.ee/sZvL/qr.svg` - SVG-Vektor
- `https://s.ee/sZvL/qr.pdf` - PDF-Dokument

**Hinweis:** Diese Funktion funktioniert nur für dynamische QR-Codes, die im Bereich [QR Codes](https://s.ee/user/qr/) erstellt wurden, nicht für normale Kurzlinks.
</Callout>

## Analyse

QR-Codes enthalten die gleichen Analysen wie Links:

- Anzahl der Scans
- Standortdaten
- Geräteinformationen
- Referrer-Quellen

---

## Best Practices

- Verwenden Sie kontrastreiche Farben für bessere Scanbarkeit
- Halten Sie Logos klein, damit der Code gut scannbar bleibt
- Testen Sie Ihren QR-Code vor dem Drucken
