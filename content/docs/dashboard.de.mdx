## Dashboard-Überblick

Greifen Sie nach dem Anmelden auf Ihr [Dashboard](https://s.ee/user/dashboard/) zu. Das S.EE-Dashboard hat zwei Hauptbereiche zur Navigation:

### Linke Seitenleiste

Zugriff auf Ihre Hauptfunktionen:

| Feature | Description |
|---------|-------------|
| [Links](https://s.ee/user/links/) | Verwalten Sie Ihre Kurzlinks |
| [QR Codes](https://s.ee/user/qr/) | Erstellen und verwalten Sie QR-Codes |
| [Text Sharing](https://s.ee/user/text/) | Teilen Sie Text, Code-Snippets und Markdown |
| [File Sharing](https://s.ee/user/files/) | Laden Sie Dateien hoch und teilen Sie sie |
| [Domains](https://s.ee/user/domains/) | Konfigurieren Sie benutzerdefinierte Domains |
| [Analytics](https://s.ee/user/analytics/) | Anzeigen von Klick- und Scan-Statistiken |
| [Team](https://s.ee/user/team/) | Teammitglieder verwalten |
| [UTM Builder](https://s.ee/user/library/utm/) | UTM-Parameter erstellen und verwalten |
| [Usage](https://s.ee/user/usage/) | Nutzung Ihres Tarifs überwachen |
| [Settings](https://s.ee/user/settings/) | Konto- und Abrechnungseinstellungen |

Auf mobilen Geräten wird die Seitenleiste automatisch eingeklappt.

### Profilmenü

Es befindet sich in der linken Seitenleiste. Sie können es verwenden, um:

- Die Dokumentation aufzurufen
- [API-Schlüssel zu verwalten](https://s.ee/user/developers/)
- [Ihren Tarif zu aktualisieren](https://s.ee/user/pricing/)
- Zwischen Konten zu wechseln
- Sich abzumelden
