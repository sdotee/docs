## 링크 만들기

1. [https://s.ee/user/links/](https://s.ee/user/links/)로 이동합니다
2. **링크 만들기** 를 클릭합니다
3. 대상 URL 을 입력합니다
4. 옵션을 설정합니다 (선택 사항)
5. **만들기** 를 클릭합니다

---

## 링크 옵션

링크를 만들 때 다음 옵션을 설정할 수 있습니다.

| Option | Description |
|--------|-------------|
| Custom slug | 링크의 뒤쪽 경로를 직접 설정합니다 (예: `s.ee/link`) |
| Domain | 공식 도메인 또는 사용자 정의 도메인 중에서 선택합니다 |
| Password | 대상 링크에 접근할 때 비밀번호를 요구합니다 |
| Expiration | 링크가 작동을 멈출 날짜를 설정합니다 |
| Targeting | 위치, 기기, 언어에 따라 사용자를 리디렉션합니다 |
| Tags | 태그로 링크를 정리합니다 |
| UTM parameters | 캠페인 추적용 파라미터를 추가합니다 |
| Parameter passing | URL 파라미터를 대상 URL 로 전달합니다 |

---

## 분석

링크를 클릭하면 다음 분석 정보를 볼 수 있습니다.

- 전체 클릭 수
- 위치별 클릭 수
- 기기 및 브라우저별 클릭 수
- 리퍼러 소스

---

## 링크 관리

링크 대시보드에서 다음 작업을 할 수 있습니다.

- **편집** - 대상 URL 또는 설정 수정
- **삭제** - 링크를 영구적으로 삭제
- **복사** - 단축 링크를 클립보드에 복사
