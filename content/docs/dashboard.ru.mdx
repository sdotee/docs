## Обзор панели управления

После входа откройте [панель управления](https://s.ee/user/dashboard/). В S.EE есть две основные зоны навигации:

### Левая боковая панель

Здесь находятся основные функции:

| Раздел | Описание |
|--------|----------|
| [Links](https://s.ee/user/links/) | Управление короткими ссылками |
| [QR Codes](https://s.ee/user/qr/) | Создание и управление QR-кодами |
| [Text Sharing](https://s.ee/user/text/) | Публикация текста, фрагментов кода и Markdown |
| [File Sharing](https://s.ee/user/files/) | Загрузка и обмен файлами |
| [Domains](https://s.ee/user/domains/) | Настройка пользовательских доменов |
| [Analytics](https://s.ee/user/analytics/) | Просмотр статистики кликов и сканирований |
| [Team](https://s.ee/user/team/) | Управление участниками команды |
| [UTM Builder](https://s.ee/user/library/utm/) | Создание и управление UTM-параметрами |
| [Usage](https://s.ee/user/usage/) | Контроль использования тарифа |
| [Settings](https://s.ee/user/settings/) | Настройки аккаунта и биллинга |

На мобильных устройствах боковая панель автоматически сворачивается.

### Меню профиля

Оно находится в левой боковой панели. Через него можно:

- открыть документацию
- [управлять API-ключами](https://s.ee/user/developers/)
- [обновить тариф](https://s.ee/user/pricing/)
- переключаться между аккаунтами
- выйти из системы
