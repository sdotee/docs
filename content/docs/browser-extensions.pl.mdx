Rozszerzenie przeglądarki S.EE przenosi skracanie URL-i, udostępnianie tekstu i przesyłanie plików bezpośrednio do przeglądarki. Jest dostępne dla Chrome, Firefox i Edge.

Kod źródłowy jest otwarty i dostępny na [GitHub](https://github.com/sdotee/browser-extensions).

## Funkcje

| Funkcja | Opis |
|---------|-------------|
| Skracanie URL-i | Skracaj dowolny URL jednym kliknięciem, używaj własnych slugów i różnych domen |
| Udostępnianie tekstu | Udostępniaj tekst i fragmenty kodu z podświetlaniem składni |
| Udostępnianie plików | Przesyłaj pliki wsadowo ze śledzeniem postępu |
| Formaty eksportu | Kopiuj wynik jako zwykły URL, Markdown, HTML lub BBCode |
| Generowanie kodów QR | Twórz kody QR dla każdego skróconego linku |
| Eksport kodów QR | Pobieraj kody QR w PNG, SVG lub PDF (512x512) |
| Menu kontekstowe | Skracaj stronę / link, pokazuj QR, udostępniaj zaznaczony tekst, przesyłaj obrazy |
| Historia | Historia per karta dla URL-i, tekstów i plików z usuwaniem zbiorczym |
| Zachowanie szkiców | Pola wejściowe są automatycznie zapisywane między sesjami |
| Automatyczne kopiowanie | Wyniki są automatycznie kopiowane do schowka |
| Tryb jasny / ciemny | Automatyczne wykrywanie motywu z ręcznym przełączaniem |
| Powiadomienia na pulpicie | Otrzymuj powiadomienia po zakończeniu działań z menu kontekstowego |
| Wiele przeglądarek | Działa w Chrome (MV3), Firefox (MV3) i Edge |
