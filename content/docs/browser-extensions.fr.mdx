L’extension de navigateur S.EE apporte le raccourcissement d’URL, le partage de texte et le téléversement de fichiers directement dans votre navigateur. Disponible pour Chrome, Firefox et Edge.

Open source sur [GitHub](https://github.com/sdotee/browser-extensions).

## Fonctionnalités

| Fonctionnalité | Description |
|---------|-------------|
| Raccourcissement d’URL | Raccourcissez n’importe quelle URL en un clic, avec slugs personnalisés et plusieurs options de domaine |
| Partage de texte | Partagez des extraits de code et du texte avec prise en charge de la coloration syntaxique |
| Partage de fichiers | Téléversez des fichiers par lot avec suivi de progression |
| Formats d’export | Copiez les résultats en URL simple, Markdown, HTML ou BBCode |
| Génération de code QR | Générez des codes QR pour n’importe quelle URL raccourcie |
| Export de code QR | Téléchargez les codes QR au format PNG, SVG ou PDF (512x512) |
| Menu contextuel | Raccourcissez une page/un lien, affichez un QR, partagez le texte sélectionné, téléversez des images |
| Gestion de l’historique | Historique par onglet pour les URL, textes et fichiers avec suppression par lot |
| Persistance des brouillons | Les champs de saisie sont enregistrés automatiquement entre les sessions |
| Copie automatique | Copiez automatiquement les résultats dans le presse-papiers |
| Mode sombre/clair | Détection automatique du thème avec bascule manuelle |
| Notifications du bureau | Recevez une notification lorsque les actions sont terminées via le menu contextuel |
| Multi-navigateurs | Fonctionne sur Chrome (MV3), Firefox (MV3) et Edge |
