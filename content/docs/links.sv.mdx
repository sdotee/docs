## Skapa en länk

1. Gå till [https://s.ee/user/links/](https://s.ee/user/links/)
2. Klicka på **Skapa länk**
3. Ange mål-URL:en
4. Konfigurera alternativ vid behov
5. Klicka på **Skapa**

---

## Länkalternativ

När du skapar en länk kan du konfigurera:

| Alternativ | Beskrivning |
|--------|-------------|
| Anpassad slug | Ange en egen slutdel av länken, till exempel `s.ee/link` |
| Domän | Välj mellan officiella domäner eller dina egna domäner |
| Lösenord | Kräver lösenord för att öppna målet |
| Utgångstid | Ange ett datum då länken ska sluta fungera |
| Målstyrning | Omdirigera efter plats, enhet eller språk |
| Taggar | Organisera länkar med taggar |
| UTM-parametrar | Lägg till parametrar för kampanjspårning |
| Vidarebefordran av parametrar | Skicka URL-parametrar vidare till måladressen |

---

## Analys

Klicka på en länk för att visa analys:

- Totalt antal klick
- Klick per plats
- Klick per enhet och webbläsare
- Hänvisningskällor

---

## Hantera länkar

Från länköversikten kan du:

- **Redigera** - uppdatera mål-URL eller inställningar
- **Ta bort** - ta bort länken permanent
- **Kopiera** - kopiera den korta länken till urklipp
