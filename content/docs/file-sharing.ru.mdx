## Обзор

Функция S.EE File Sharing позволяет загружать и публиковать файлы по короткой и удобной ссылке. Это удобно для изображений, документов, аудио, видео и других файлов.

---

## Загрузка файлов

### Быстрая загрузка (перетаскивание)

1. Войдите в [https://s.ee/user/dashboard](https://s.ee/user/dashboard)
2. Нажмите **Sharing > Files** в левой боковой панели или откройте [https://s.ee/user/files/](https://s.ee/user/files/)
3. Перетащите файлы или папки в окно **Drag & Drop**
   - Вы также можете нажать на окно, чтобы выбрать файлы вручную
   - При загрузке папки будут отправлены все файлы внутри нее
4. Файлы будут опубликованы с доменом по умолчанию и случайным коротким кодом

### Расширенная загрузка (Create File)

Чтобы получить больше контроля над публикацией файлов:

1. Нажмите **Create File**
2. Настройте параметры:
   - **Domain** - Выберите домен для страницы публикации
   - **Custom Slug** - Укажите пользовательский короткий код URL
   - **Other options** - Метки, срок действия и т. д.
3. Выберите файл и загрузите его

## Ссылки на файл

Для каждого загруженного файла доступны два типа ссылок:

| Тип ссылки | Формат | Описание |
|-----------|--------|-------------|
| Страница публикации | `fs.to/[your_slug]` | Настраиваемые домен и короткий код |
| Прямая ссылка | `i.see.you/[yyyy]/[mm]/[dd]/[random_slug]/[filename].[ext]` | Прямой доступ к файлу с исходным именем |

**Примечание:** прямые ссылки пока не поддерживают пользовательские домены.

</Callout>

<Callout type="info">
Для следующих расширений файлов мы можем принудительно установить `Content-Type` ответа в `text/plain`, чтобы снизить риск злоупотреблений со скриптами: `.js`, `.mjs`, `.html`, `.htm`, `.xhtml`, `.shtml`, `.svg`, `.xml`, `.json`, `.map`.
</Callout>

<Callout type="info">

</Callout>

## Лимиты хранения

Лимиты зависят от вашего тарифа:

| Тариф | Максимальный размер файла | Общее хранилище |
|------|---------------|---------------|
| Lite | 100 MB | 100 GB |
| Pro | 1 GB | 500 GB |
| Premium | 5 GB | 5 TB |
| Enterprise | Индивидуально | Индивидуально |

Подробности см. на [s.ee/pricing](https://s.ee/pricing).

---

## Хранение файлов

- Файлы **хранятся постоянно**, пока ваша подписка активна
- Если подписка истекает, файлы хранятся **7 дней**, после чего удаляются

## FAQ

### Кто может загружать файлы?

Только платные подписчики. Требуется действующая подписка.

### Есть ли ограничения на скачивание?

Нет, ограничений по количеству скачиваний и пропускной способности для опубликованных файлов нет.

### Можно ли настроить URL страницы публикации?

Да. Вы можете указать пользовательский домен и короткий код для страницы публикации при использовании метода **Create File**.

### Можно ли настроить прямую ссылку?

Пока прямые ссылки используют фиксированный формат и не поддерживают пользовательские домены.
