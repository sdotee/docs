## 概要

S.EE のテキスト共有機能では、プレーンテキスト、コードスニペット、Markdown ドキュメントを、共有しやすい短い URL で公開できます。設定ファイル、コード例、メモ、ドキュメントの共有に適しています。

---

## テキスト共有を作成する

1. [https://s.ee/user/dashboard](https://s.ee/user/dashboard) にログインします
2. 左サイドバーの **Sharing > Text** をクリックするか、[https://s.ee/user/text/](https://s.ee/user/text/) に直接アクセスします
3. 必要事項を入力します
   - **Title**（任意）- ダウンロード時のファイル名として使用されます
   - **Short Link** - 使用するドメインを選択します
   - **Text Type** - Plain Text、Code、Markdown から選択します
4. **編集** ボックスに内容を入力します
5. 必要に応じてオプション設定を行います
6. **テキストを作成** をクリックします

---

## テキストタイプ

| 種類 | 説明 |
|------|-------------|
| プレーンテキスト | 書式なしのシンプルなテキストです |
| コード | シンタックスハイライト付きのコードスニペットです |
| Markdown | 書式付きで表示される Markdown です |

---

## 任意の設定

| オプション | 説明 |
|--------|-------------|
| パスワード | テキストをパスワードで保護します |
| タグ | タグで共有テキストを整理します |
| 有効期限 | テキスト共有の期限を設定します |
| 検索エンジンのインデックスを無効化 | 検索エンジンによる収集を防ぎます |

---

## 共有したテキストの表示

作成すると、`ba.sh/demo` のような短い URL が発行されます。

### 表示モード

| URL | 説明 |
|-----|-------------|
| `ba.sh/demo` | シンタックスハイライト付きの整形ページを表示します |
| `ba.sh/demo/raw` | 生のプレーンテキストを直接表示します |

`/raw` エンドポイントは次の用途に便利です。

- `curl` や `wget` でスクリプトを直接ダウンロードする
- 他のアプリケーションに埋め込む
- 書式なしで閲覧する

---

## 利用可能なドメイン

すべてのプランで、次のドメインをテキスト共有に利用できます。

| ドメイン | 用途 |
|--------|----------|
| [ba.sh](https://ba.sh) | Bash / Shell スクリプト |
| [z.sh](https://z.sh) | Shell スクリプト |
| [fs.to](https://fs.to) | 一般的なファイル共有 |
| [files.to](https://files.to) | 一般的なファイル共有 |
| [fileshare.to](https://fileshare.to) | 一般的なファイル共有 |
| [filesharing.to](https://filesharing.to) | 一般的なファイル共有 |

Premium または Enterprise をご利用の場合は、[公式ドメイン](https://s.ee/docs/official-domains/#premium--enterprise) もご参照ください。
