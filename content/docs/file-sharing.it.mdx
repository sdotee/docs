## Panoramica

La condivisione file di S.EE ti permette di caricare e condividere file tramite un link breve e facile da inviare. È adatta a immagini, documenti, audio, video e a molti altri tipi di file.

---

## Caricare file

### Upload rapido (trascina e rilascia)

1. Accedi su [https://s.ee/user/dashboard](https://s.ee/user/dashboard)
2. Nel menu laterale seleziona **Sharing > Files** oppure vai direttamente su [https://s.ee/user/files/](https://s.ee/user/files/)
3. Trascina file o cartelle nell'area **Drag & Drop**
   - Puoi anche fare clic sull'area e scegliere i file dal browser
   - Se carichi una cartella, verranno caricati tutti i file al suo interno
4. I file verranno condivisi con il dominio predefinito e uno slug casuale

### Upload avanzato (Create File)

Se vuoi più controllo sulla condivisione:

1. Fai clic su **Create File**
2. Configura le impostazioni:
   - **Domain** - scegli il dominio della pagina di condivisione
   - **Custom Slug** - imposta uno slug URL personalizzato
   - **Other options** - tag, scadenza e altre opzioni
3. Seleziona il file e caricalo

---

## Link dei file

Ogni file caricato ha due tipi di link:

| Tipo di link | Formato | Descrizione |
|-----------|--------|-------------|
| Share Page | `fs.to/[your_slug]` | Dominio e slug personalizzabili |
| Direct Link | `i.see.you/[yyyy]/[mm]/[dd]/[random_slug]/[filename].[ext]` | Accesso diretto al file con il nome originale |

**Nota:** i link diretti al momento non supportano domini personalizzati.

</Callout>

<Callout type="info">
Per tipi di file sconosciuti o ambigui, il `Content-Type` della risposta verrà impostato su `application/octet-stream`.
</Callout>

## Limiti di storage

I limiti variano in base al piano:

| Piano | Dimensione massima file | Storage totale |
|------|---------------|---------------|
| Lite | 100 MB | 100 GB |
| Pro | 1 GB | 500 GB |
| Premium | 5 GB | 5 TB |
| Enterprise | Personalizzato | Personalizzato |

Consulta [s.ee/pricing](https://s.ee/pricing) per i dettagli completi sui piani.

## Conservazione dei file

- I file vengono conservati in modo permanente finché il tuo abbonamento è attivo
- Se l'abbonamento scade, i file vengono conservati per **7 giorni** prima dell'eliminazione

## FAQ

### Chi può caricare file?

Solo gli utenti con un abbonamento a pagamento possono caricare file. È necessario avere un piano attivo.

### Ci sono limiti di download o di banda?

No. I file condivisi non hanno limiti di numero di download né di larghezza di banda.

### Posso personalizzare l'URL della pagina di condivisione?

Sì. Con il metodo **Create File** puoi impostare un dominio personalizzato e uno slug personalizzato per la pagina.

### Posso personalizzare l'URL del link diretto?

Al momento i link diretti seguono un formato fisso e non supportano domini personalizzati.
