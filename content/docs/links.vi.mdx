## Tạo liên kết

1. Truy cập [https://s.ee/user/links/](https://s.ee/user/links/)
2. Nhấp vào **Create Link**
3. Nhập URL đích
4. Cấu hình các tùy chọn (không bắt buộc)
5. Nhấp vào **Create**

---

## Tùy chọn liên kết

Khi tạo liên kết, bạn có thể cấu hình:

| Option | Description |
|--------|-------------|
| Custom slug | Đặt phần đuôi tùy chỉnh cho liên kết của bạn (ví dụ: `s.ee/link`) |
| Domain | Chọn từ các tên miền chính thức hoặc tên miền tùy chỉnh của bạn |
| Password | Yêu cầu mật khẩu để truy cập đích |
| Expiration | Đặt ngày liên kết ngừng hoạt động |
| Targeting | Chuyển hướng người dùng dựa trên vị trí, thiết bị hoặc ngôn ngữ |
| Tags | Sắp xếp liên kết bằng tag |
| UTM parameters | Thêm tham số theo dõi chiến dịch |
| Parameter passing | Chuyển tiếp tham số URL đến đích |

---

## Phân tích

Nhấp vào bất kỳ liên kết nào để xem phân tích:

- Tổng số lượt nhấp
- Lượt nhấp theo vị trí
- Lượt nhấp theo thiết bị và trình duyệt
- Nguồn giới thiệu

---

## Quản lý liên kết

Từ bảng điều khiển liên kết, bạn có thể:

- **Edit** - Cập nhật URL đích hoặc cài đặt
- **Delete** - Xóa liên kết vĩnh viễn
- **Copy** - Sao chép liên kết ngắn vào clipboard
